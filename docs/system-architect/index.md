# 系统架构设计师

> 计算机技术与软件专业技术资格（软考·高级）

## 科目介绍

系统架构设计师是软考高级资格之一，侧重考查信息系统架构设计、软件工程、信息安全等综合能力。考试分为三个科目：

- **综合知识**：75题，150分钟
- **案例分析**：4题，90分钟
- **论文写作**：1题，120分钟

三个科目均需达到45分方可通过考试。

## 考试科目与分值占比

| 科目 | 题型 | 题量 | 考试时间 | 及格线 |
|------|------|------|----------|--------|
| 综合知识 | 单项选择题 | 75题 | 150分钟 | 45分 |
| 案例分析 | 综合分析题 | 4题 | 90分钟 | 45分 |
| 论文写作 | 论文题 | 1题 | 120分钟 | 45分 |

## 备考建议

1. **综合知识**：上午题注重基础知识广度，需熟练掌握计算机系统、软件工程、信息安全等概念
2. **案例分析**：下午题侧重综合分析与设计能力，重点掌握架构模式、系统设计方法
3. **论文写作**：需提前准备架构设计、项目管理等常见主题，注意格式规范与论点展开
4. **时间管理**：综合知识75题时间较紧，建议每题不超过2分钟；论文需预留充足时间构思
5. **重点章节**：第四章（软件工程）、第六章（系统架构设计）分值最高，务必重点掌握
