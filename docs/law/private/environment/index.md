# 环境资源法

## 科目概述

环境资源法是调整因保护和改善环境、合理利用自然资源、防治污染和其他公害而产生的社会关系的法律规范的总称。本学科包括环境保护法基本制度、环境影响评价法、污染防治法和自然资源法等内容。
