# 第三章 工作时间与休息休假

## 本章概要

工作时间与休息休假制度是劳动法的重要组成部分，保障劳动者的休息权。本章重点包括标准工时制度、特殊工时制度、休息休假制度、加班限制与加班工资。

## 高频考点

1. 标准工时制度
2. 加班限制
3. 加班工资计算
4. 法定节假日
5. 年休假制度

## 重点难点

- 综合计算工时与不定时工作制
- 加班工资的计算
- 年休假的享受条件

## 知识点详解

### 一、工作时间

#### 1. 标准工时制度

- 每日工作时间不超过8小时
- 平均每周工作时间不超过44小时
- 每周至少休息1日

#### 2. 特殊工时制度

**综合计算工时**：
- 因工作性质特殊需要连续作业的
- 受季节和自然条件限制的
- 以周、月、季、年等为周期综合计算

**不定时工作**：
- 企业中的高级管理人员、外勤人员、推销人员、部分值班人员
- 企业中的长途运输人员、出租汽车司机和铁路、港口、仓库的部分装卸人员

### 二、休息休假

#### 1. 法定节假日

| 节日 | 放假天数 |
|------|----------|
| 元旦 | 1天 |
| 春节 | 3天 |
| 清明节 | 1天 |
| 劳动节 | 1天 |
| 端午节 | 1天 |
| 中秋节 | 1天 |
| 国庆节 | 3天 |

#### 2. 年休假

| 累计工作年限 | 年休假天数 |
|-------------|-----------|
| 已满1年不满10年 | 5天 |
| 已满10年不满20年 | 10天 |
| 已满20年 | 15天 |

**不享受年休假的情形**：
- 职工依法享受寒暑假，且天数多于年休假天数的
- 请事假累计20天以上且单位按照规定不扣工资的
- 累计工作满1年不满10年的职工，请病假累计2个月以上的
- 累计工作满10年不满20年的职工，请病假累计3个月以上的
- 累计工作满20年以上的职工，请病假累计4个月以上的

### 三、加班

#### 1. 加班限制

- 一般每日不得超过1小时
- 因特殊原因需要延长工作时间的，在保障劳动者身体健康的条件下每日不得超过3小时
- 每月不得超过36小时

#### 2. 加班工资

| 情形 | 工资标准 |
|------|----------|
| 工作日延长工作时间 | 不低于工资的150% |
| 休息日安排工作又不能安排补休 | 不低于工资的200% |
| 法定休假日安排工作 | 不低于工资的300% |

## 例题与解析

### 例题1

法定休假日安排劳动者工作的，支付不低于工资的多少比例的工资报酬？

A. 150%  
B. 200%  
C. 250%  
D. 300%

**答案：D**

**解析**：法定休假日安排劳动者工作的，支付不低于工资的300%的工资报酬。

### 例题2

职工累计工作已满10年不满20年的，年休假为：

A. 5天  
B. 10天  
C. 15天  
D. 20天

**答案：B**

**解析**：职工累计工作已满10年不满20年的，年休假10天。
