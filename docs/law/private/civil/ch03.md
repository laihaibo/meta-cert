# 第三章 所有权

## 本章概要

本章介绍所有权的基本制度，包括所有权的取得、共有和建筑物区分所有权。重点掌握善意取得制度和共有制度。

## 高频考点

> 🔥 高频考点

### 考点1: 善意取得
- 受让人受让该不动产或者动产时是善意的
- 以合理的价格转让
- 转让的不动产或者动产依照法律规定应当登记的已经登记，不需要登记的已经交付给受让人

### 考点2: 共有
- 按份共有与共同共有
- 共有物的处分
- 共有物的分割

### 考点3: 建筑物区分所有权
- 专有部分的所有权
- 共有部分的共有权
- 共同管理权

## 重点难点

> ⚠️ 难点

## 知识点详解

### 3.1 所有权概述

所有权是所有人对自己的不动产或者动产，依法享有占有、使用、收益和处分的权利。

**所有权的权能**：

| 权能 | 内容 |
|------|------|
| 占有 | 对物的实际控制 |
| 使用 | 按照物的性能和用途对物加以利用 |
| 收益 | 收取物所产生的利益 |
| 处分 | 对物进行事实上或法律上的处置 |

### 3.2 所有权的取得

#### 3.2.1 原始取得

| 方式 | 说明 |
|------|------|
| 生产 | 通过劳动创造新的财产 |
| 先占 | 以所有的意思占有无主物 |
| 拾得遗失物 | 拾得遗失物应当返还权利人 |
| 发现埋藏物 | 发现埋藏物参照拾得遗失物的规定 |
| 善意取得 | 无权处分人将不动产或者动产转让给受让人 |

#### 3.2.2 善意取得

**善意取得的条件**：

| 条件 | 内容 |
|------|------|
| 无权处分 | 处分人没有处分权 |
| 善意 | 受让人受让该不动产或者动产时是善意的 |
| 合理价格 | 以合理的价格转让 |
| 登记或交付 | 依照法律规定应当登记的已经登记，不需要登记的已经交付给受让人 |

**善意的认定**：
- 受让人不知道转让人无处分权
- 受让人无重大过失

**不适用善意取得的情形**：
- 赃物
- 遗失物（但有例外）

### 3.3 共有

#### 3.3.1 按份共有与共同共有

| 类型 | 特征 | 份额 |
|------|------|------|
| 按份共有 | 共有人按照各自的份额享有所有权 | 有明确份额 |
| 共同共有 | 共有人不分份额地享有所有权 | 无明确份额 |

**共同共有的类型**：
- 夫妻共有
- 家庭共有
- 遗产分割前的共有

#### 3.3.2 共有物的处分

**按份共有**：
- 处分共有物或者对共有物作重大修缮、变更性质或者用途的，应当经占份额2/3以上的按份共有人同意

**共同共有**：
- 处分共有物应当经全体共同共有人同意

#### 3.3.3 共有物的分割

| 类型 | 分割条件 |
|------|---------|
| 按份共有 | 共有人可以随时请求分割 |
| 共同共有 | 共有的基础丧失或者有重大理由需要分割时可以请求分割 |

### 3.4 建筑物区分所有权

#### 3.4.1 建筑物区分所有权的构成

| 权利 | 内容 |
|------|------|
| 专有部分的所有权 | 业主对其建筑物专有部分享有占有、使用、收益和处分的权利 |
| 共有部分的共有权 | 业主对建筑物专有部分以外的共有部分，享有权利，承担义务 |
| 共同管理权 | 业主对建筑物专有部分以外的共有部分，享有共同管理的权利 |

**共有部分的范围**：
- 建筑区划内的道路（属于业主共有，但属于城镇公共道路的除外）
- 建筑区划内的绿地（属于业主共有，但属于城镇公共绿地或者明示属于个人的除外）
- 建筑区划内的其他公共场所、公用设施和物业服务用房

## 例题与解析

### 例题1

**题目**: 甲将乙的手机以合理价格卖给不知情的丙，并已交付。下列说法正确的是：

**选项**:
A. 丙不能取得手机所有权，因为甲无权处分
B. 丙可以善意取得手机所有权
C. 丙必须经过乙追认才能取得所有权
D. 合同无效，丙不能取得所有权

**答案**: B

**解析**: 甲无权处分乙的手机，但丙善意、以合理价格受让且已交付，符合善意取得的条件，丙可以取得手机所有权。

### 例题2

**题目**: 甲乙按份共有一套房屋，甲占60%份额，乙占40%份额。甲欲将房屋出售，下列说法正确的是：

**选项**:
A. 必须经乙同意才能出售
B. 甲可以自行出售自己的份额
C. 经甲同意即可出售整套房屋
D. 必须经乙同意才能出售整套房屋，但甲可以出售自己的份额

**答案**: D

**解析**: 按份共有人可以转让自己的份额；但处分共有物应当经占份额2/3以上的按份共有人同意。甲占60%份额，不足2/3，不能单独出售整套房屋。
