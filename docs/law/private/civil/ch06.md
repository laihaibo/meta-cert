# 第六章 占有

## 本章概要

本章介绍占有的基本概念、种类和保护。

## 高频考点

> 🔥 高频考点

### 考点1: 占有的种类
- 有权占有与无权占有
- 善意占有与恶意占有
- 自主占有与他主占有

### 考点2: 占有的保护
- 占有物返还请求权
- 占有妨害排除请求权

## 知识点详解

### 6.1 占有的概念

占有是对不动产或者动产的事实管领力。

### 6.2 占有的种类

| 分类标准 | 类型 | 说明 |
|---------|------|------|
| 是否有法律根据 | 有权占有 / 无权占有 | 有权占有有法律根据，无权占有无法律根据 |
| 是否知道无权占有 | 善意占有 / 恶意占有 | 善意占有人不知道自己无权占有 |
| 是否以所有的意思占有 | 自主占有 / 他主占有 | 自主占有人以所有的意思占有 |

### 6.3 占有的保护

- 占有的不动产或者动产被侵占的，占有人有权请求返还原物
- 因侵占或者妨害造成损害的，占有人有权请求损害赔偿

## 例题与解析

### 例题1

**题目**: 甲借用乙的自行车，丙将该自行车盗走。下列说法正确的是：

**选项**:
A. 只有乙可以请求丙返还自行车
B. 甲可以请求丙返还自行车
C. 甲乙均可以请求丙返还自行车
D. 甲无权请求丙返还自行车

**答案**: C

**解析**: 占有人有权请求返还原物，甲作为借用人是占有人，乙作为所有权人也是权利人，均可以请求丙返还。
