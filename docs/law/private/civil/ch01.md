# 第一章 民法总论

## 本章概要

本章是民法的基础章节，介绍民法的基本原则、民事主体制度、民事法律行为制度、代理制度和诉讼时效制度。重点掌握民事法律行为的效力、代理的种类和诉讼时效。

## 高频考点

> 🔥 高频考点

### 考点1: 民事法律行为的效力
- 有效民事法律行为的要件
- 无效民事法律行为
- 可撤销的民事法律行为
- 效力待定的民事法律行为

### 考点2: 代理
- 委托代理与法定代理
- 无权代理与表见代理
- 代理权的滥用

### 考点3: 诉讼时效
- 普通诉讼时效：3年
- 特殊诉讼时效
- 诉讼时效的中止、中断和延长

## 重点难点

> ⚠️ 难点

### 难点1: 无权代理与表见代理的区分
- 无权代理：行为人没有代理权而以他人名义实施代理行为
- 表见代理：行为人没有代理权，但相对人有理由相信其有代理权

### 难点2: 可撤销民事法律行为的撤销权
- 重大误解、欺诈、胁迫、显失公平
- 撤销权的行使期限

## 知识点详解

### 1.1 民法的基本原则

| 原则 | 内容 |
|------|------|
| 平等原则 | 民事主体在民事活动中的法律地位一律平等 |
| 自愿原则 | 民事主体按照自己的意思设立、变更、终止民事法律关系 |
| 公平原则 | 民事主体从事民事活动应当遵循公平原则 |
| 诚信原则 | 民事主体从事民事活动应当遵循诚实信用原则 |
| 公序良俗原则 | 民事主体从事民事活动不得违反公序良俗 |
| 绿色原则 | 民事主体从事民事活动应当有利于节约资源、保护生态环境 |

### 1.2 民事主体

#### 1.2.1 自然人

**民事权利能力**：从出生时起到死亡时止

**民事行为能力**：

| 类型 | 年龄/精神状态 | 范围 |
|------|-------------|------|
| 完全民事行为能力人 | 18周岁以上；16周岁以上以自己劳动收入为主要生活来源 | 可以独立实施民事法律行为 |
| 限制民事行为能力人 | 8周岁以上不满18周岁；不能完全辨认自己行为的成年人 | 可以实施纯获利益的民事法律行为或与其年龄、智力相适应的民事法律行为 |
| 无民事行为能力人 | 不满8周岁；不能辨认自己行为的成年人 | 由法定代理人代理实施民事法律行为 |

#### 1.2.2 法人

**法人的分类**：

| 类型 | 说明 |
|------|------|
| 营利法人 | 以取得利润并分配给股东等出资人为目的成立的法人 |
| 非营利法人 | 为公益目的或者其他非营利目的成立，不向出资人、设立人或者会员分配所取得利润的法人 |
| 特别法人 | 机关法人、农村集体经济组织法人、城镇农村的合作经济组织法人、基层群众性自治组织法人 |

### 1.3 民事法律行为

#### 1.3.1 民事法律行为的效力

| 类型 | 情形 | 法律后果 |
|------|------|---------|
| 有效 | 行为人具有相应的民事行为能力；意思表示真实；不违反法律、行政法规的强制性规定，不违背公序良俗 | 自始有效 |
| 无效 | 无民事行为能力人实施的；虚假意思表示；违反法律、行政法规的强制性规定；违背公序良俗；恶意串通损害他人合法权益 | 自始无效 |
| 可撤销 | 重大误解；欺诈；胁迫；显失公平 | 撤销后自始无效 |
| 效力待定 | 限制民事行为能力人实施的超出其年龄、智力范围的行为；无权代理 | 经追认后有效 |

#### 1.3.2 可撤销民事法律行为的撤销权

| 情形 | 撤销权人 | 行使期限 |
|------|---------|---------|
| 重大误解 | 行为人 | 自知道或应当知道撤销事由之日起90日内 |
| 欺诈 | 受欺诈方 | 自知道或应当知道撤销事由之日起1年内 |
| 胁迫 | 受胁迫方 | 自胁迫行为终止之日起1年内 |
| 显失公平 | 受损害方 | 自知道或应当知道撤销事由之日起1年内 |

### 1.4 代理

#### 1.4.1 代理的种类

| 类型 | 说明 |
|------|------|
| 委托代理 | 基于被代理人的委托而产生的代理 |
| 法定代理 | 基于法律的直接规定而产生的代理 |

#### 1.4.2 无权代理与表见代理

| 类型 | 要件 | 法律后果 |
|------|------|---------|
| 无权代理 | 行为人没有代理权 | 效力待定，被代理人追认后有效 |
| 表见代理 | 行为人没有代理权，但相对人有理由相信其有代理权 | 代理行为有效 |

### 1.5 诉讼时效

#### 1.5.1 诉讼时效的种类

| 类型 | 期间 | 适用情形 |
|------|------|---------|
| 普通诉讼时效 | 3年 | 一般民事权利 |
| 特殊诉讼时效 | 法律另有规定的除外 | 特定类型的民事权利 |
| 最长诉讼时效 | 20年 | 从权利被侵害之日起计算 |

#### 1.5.2 诉讼时效的中止、中断

| 事由 | 效果 |
|------|------|
| 中止 | 在诉讼时效期间的最后6个月内，因不可抗力等障碍不能行使请求权的，诉讼时效中止，自中止时效的原因消除之日起满6个月，诉讼时效期间届满 |
| 中断 | 诉讼时效因提起诉讼、当事人一方提出要求或者同意履行义务而中断，从中断时起，诉讼时效期间重新计算 |

## 例题与解析

### 例题1

**题目**: 甲（15周岁）将自己的手机以500元的价格卖给乙。该买卖合同的效力如何？

**选项**:
A. 有效
B. 无效
C. 效力待定
D. 可撤销

**答案**: C

**解析**: 甲为限制民事行为能力人，出卖手机的行为超出其年龄、智力范围，属于效力待定的民事法律行为，需要经甲的法定代理人追认后有效。

### 例题2

**题目**: 甲没有代理权，但以乙的名义与丙签订合同。丙有理由相信甲有代理权。该合同的效力如何？

**选项**:
A. 无效
B. 效力待定
C. 有效
D. 可撤销

**答案**: C

**解析**: 甲的行为构成表见代理，代理行为有效，合同对乙发生效力。
