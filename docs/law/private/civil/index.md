# 民法

## 学科概述

民法是法考中分值最高的学科之一，属于**私法**范畴，主要调整平等主体之间的人身关系和财产关系。民法学科内容广泛、体系庞大，是法考客观题和主观题的重中之重。

## 分值权重

| 考试题型 | 分值占比 | 重要程度 |
|---------|---------|---------|
| 客观题 | 40-45分 | ⭐⭐⭐⭐⭐ |
| 主观题 | 30-35分 | ⭐⭐⭐⭐⭐ |

## 备考建议

1. **总则部分**：重点掌握民事法律行为、代理、诉讼时效
2. **物权部分**：重点掌握物权变动、担保物权
3. **合同部分**：重点掌握合同的效力、违约责任
4. **学习方法**：构建民法体系，理解各项制度之间的逻辑关系
