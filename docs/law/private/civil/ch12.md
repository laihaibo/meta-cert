# 第十二章 继承

## 本章概要

本章介绍继承的基本制度，包括法定继承、遗嘱继承、遗赠和遗赠扶养协议。重点掌握法定继承的顺序和遗嘱的形式。

## 重点难点

> ⚠️ 难点

### 难点1: 代位继承与转继承的区别
- 代位继承：被继承人的子女先于被继承人死亡，由被继承人子女的晚辈直系血亲代位继承
- 转继承：继承人在被继承人死亡后、遗产分割前死亡，其应继承的遗产份额转由其继承人继承

## 知识点详解

### 12.1 继承概述

**继承的开始**：继承从被继承人死亡时开始

**继承权的丧失**：
- 故意杀害被继承人
- 为争夺遗产而杀害其他继承人
- 遗弃被继承人，或者虐待被继承人情节严重
- 伪造、篡改、隐匿或者销毁遗嘱，情节严重
- 以欺诈、胁迫手段迫使或者妨碍被继承人设立、变更或者撤回遗嘱，情节严重

### 12.2 法定继承

#### 12.2.1 法定继承人的顺序

| 顺序 | 继承人 |
|------|--------|
| 第一顺序 | 配偶、子女、父母 |
| 第二顺序 | 兄弟姐妹、祖父母、外祖父母 |

- 继承开始后，由第一顺序继承人继承，第二顺序继承人不继承
- 没有第一顺序继承人继承的，由第二顺序继承人继承

**丧偶儿媳和丧偶女婿的继承权**：
- 丧偶儿媳对公婆，丧偶女婿对岳父母，尽了主要赡养义务的，作为第一顺序继承人

#### 12.2.2 代位继承

**代位继承的条件**：
- 被继承人的子女先于被继承人死亡
- 由被继承人子女的晚辈直系血亲代位继承

**代位继承的特点**：
- 代位继承人一般只能继承被代位继承人有权继承的遗产份额
- 被继承人的孙子女、外孙子女、曾孙子女、外曾孙子女都可以代位继承

#### 12.2.3 转继承

**转继承的条件**：
- 继承人在被继承人死亡后、遗产分割前死亡
- 继承人没有放弃继承

**转继承与代位继承的区别**：

| 比较项 | 代位继承 | 转继承 |
|--------|---------|--------|
| 继承人死亡时间 | 先于被继承人死亡 | 后于被继承人死亡 |
| 继承人范围 | 被继承人子女的晚辈直系血亲 | 继承人的所有法定继承人 |
| 适用范围 | 只适用于法定继承 | 既适用于法定继承，也适用于遗嘱继承 |

### 12.3 遗嘱继承

#### 12.3.1 遗嘱的形式

| 形式 | 要件 |
|------|------|
| 自书遗嘱 | 由遗嘱人亲笔书写，签名，注明年、月、日 |
| 代书遗嘱 | 应当有两个以上见证人在场见证，由其中一人代书，注明年、月、日，并由代书人、其他见证人和遗嘱人签名 |
| 打印遗嘱 | 应当有两个以上见证人在场见证，遗嘱人和见证人应当在遗嘱每一页签名，注明年、月、日 |
| 录音录像遗嘱 | 应当有两个以上见证人在场见证，遗嘱人和见证人应当记录其姓名或者肖像，以及年、月、日 |
| 口头遗嘱 | 遗嘱人在危急情况下，可以立口头遗嘱，应当有两个以上见证人在场见证 |
| 公证遗嘱 | 经公证机构办理 |

#### 12.3.2 遗嘱的效力

- 遗嘱应当为缺乏劳动能力又没有生活来源的继承人保留必要的遗产份额
- 遗嘱人可以撤回、变更自己所立的遗嘱
- 立有数份遗嘱，内容相抵触的，以最后的遗嘱为准

#### 12.3.3 遗嘱见证人的限制

**不能作为遗嘱见证人的人**：
- 无民事行为能力人、限制民事行为能力人以及其他不具有见证能力的人
- 继承人、受遗赠人
- 与继承人、受遗赠人有利害关系的人

### 12.4 遗赠

**遗赠与遗嘱继承的区别**：

| 比较项 | 遗嘱继承 | 遗赠 |
|--------|---------|------|
| 主体 | 法定继承人 | 法定继承人以外的人 |
| 接受方式 | 明示放弃视为接受 | 明示接受视为放弃 |
| 债务清偿 | 继承遗产应当清偿被继承人依法应当缴纳的税款和债务 | 受遗赠人不承担债务 |

### 12.5 遗赠扶养协议

**遗赠扶养协议的特征**：
- 遗赠扶养协议是双方民事法律行为
- 扶养人承担该公民生养死葬的义务
- 享有受遗赠的权利

**遗赠扶养协议的效力**：
- 遗赠扶养协议的效力优先于遗嘱和法定继承

## 例题与解析

### 例题1

**题目**: 甲有配偶乙，儿子丙，女儿丁。丙先于甲死亡，丙有儿子戊。甲死亡后，下列说法正确的是：

**选项**:
A. 由乙、丁继承，戊不能继承
B. 由乙、丁、戊共同继承
C. 由乙、丁继承，戊可以代位继承
D. 由乙、丁继承，戊可以转继承

**答案**: C

**解析**: 丙先于甲死亡，丙的儿子戊可以代位继承丙有权继承的遗产份额。甲的第一顺序继承人为乙、丁、戊。

### 例题2

**题目**: 甲立有数份遗嘱，内容相抵触。下列说法正确的是：

**选项**:
A. 以公证遗嘱为准
B. 以最后的遗嘱为准
C. 以第一份遗嘱为准
D. 所有遗嘱均无效

**答案**: B

**解析**: 立有数份遗嘱，内容相抵触的，以最后的遗嘱为准。
