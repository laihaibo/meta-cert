# 第十三章 侵权责任总论

## 本章概要

本章介绍侵权责任的基本制度，包括侵权责任的归责原则、构成要件和免责事由。重点掌握过错责任原则和无过错责任原则。

## 知识点详解

### 13.1 侵权责任概述

**侵权责任的保护范围**：
- 侵害民事权益，应当承担侵权责任
- 民事权益包括生命权、健康权、姓名权、名誉权、荣誉权、肖像权、隐私权、婚姻自主权、监护权、所有权、用益物权、担保物权、著作权、专利权、商标专用权、发现权、股权、继承权等人身、财产权益

### 13.2 归责原则

#### 13.2.1 过错责任原则

- 行为人因过错侵害他人民事权益造成损害的，应当承担侵权责任
- 依照法律规定推定行为人有过错，其不能证明自己没有过错的，应当承担侵权责任（过错推定）

#### 13.2.2 无过错责任原则

- 行为人造成他人民事权益损害，不论行为人有无过错，法律规定应当承担侵权责任的，依照其规定

**适用无过错责任的情形**：
- 产品责任
- 机动车交通事故责任
- 环境污染和生态破坏责任
- 高度危险责任
- 饲养动物损害责任
- 建筑物和物件损害责任

### 13.3 侵权责任的构成要件

| 要件 | 内容 |
|------|------|
| 行为 | 作为或不作为 |
| 损害 | 财产损害或人身损害 |
| 因果关系 | 行为与损害之间有因果关系 |
| 过错 | 故意或过失（过错责任） |

### 13.4 免责事由

| 事由 | 效果 |
|------|------|
| 不可抗力 | 因不可抗力不能履行民事义务的，不承担民事责任 |
| 正当防卫 | 因正当防卫造成损害的，不承担民事责任 |
| 紧急避险 | 因紧急避险造成损害的，由引起险情发生的人承担民事责任 |
| 受害人故意 | 损害是因受害人故意造成的，行为人不承担责任 |
| 第三人原因 | 损害是因第三人造成的，第三人应当承担侵权责任 |

### 13.5 共同侵权

#### 13.5.1 共同侵权行为

- 二人以上共同实施侵权行为，造成他人损害的，应当承担连带责任

#### 13.5.2 共同危险行为

- 二人以上实施危及他人人身、财产安全的行为，其中一人或者数人的行为造成他人损害，不能确定具体侵权人的，行为人承担连带责任

#### 13.5.3 教唆、帮助行为

- 教唆、帮助他人实施侵权行为的，应当与行为人承担连带责任
- 教唆、帮助无民事行为能力人、限制民事行为能力人实施侵权行为的，应当承担侵权责任

### 13.6 损害赔偿

**损害赔偿的范围**：
- 财产损失：按照损失发生时的市场价格或者其他合理方式计算
- 人身损害：医疗费、护理费、交通费、营养费、住院伙食补助费等合理费用，以及因误工减少的收入
- 精神损害：侵害自然人人身权益造成严重精神损害的，被侵权人有权请求精神损害赔偿

## 例题与解析

### 例题1

**题目**: 甲饲养的狗将乙咬伤。下列说法正确的是：

**选项**:
A. 甲如果能证明自己没有过错，可以不承担责任
B. 甲无论是否有过错，都应当承担责任
C. 只有甲有过错时才承担责任
D. 甲不承担责任，因为乙可能有挑逗行为

**答案**: B

**解析**: 饲养的动物造成他人损害的，动物饲养人或者管理人应当承担侵权责任，适用无过错责任原则。
