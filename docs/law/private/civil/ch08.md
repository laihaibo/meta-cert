# 第八章 合同法总论

## 本章概要

本章介绍合同法的基本制度，包括合同的订立、效力、履行、变更、转让和终止。重点掌握合同的效力、违约责任和合同的解除。

- 有效合同
- 无效合同

## 重点难点

> ⚠️ 难点

### 难点1: 同时履行抗辩权、先履行抗辩权、不安抗辩权的区分
- 同时履行抗辩权：双方债务同时到期，一方未履行的，另一方可以拒绝履行
- 先履行抗辩权：先履行一方未履行的，后履行一方可以拒绝履行
- 不安抗辩权：先履行一方有证据证明后履行一方不能履行的，可以中止履行

### 难点2: 违约责任与侵权责任的竞合
- 因当事人一方的违约行为，损害对方人身权益、财产权益的，受损害方有权选择请求其承担违约责任或者侵权责任

## 知识点详解

### 8.1 合同的订立

#### 8.1.1 要约

**要约的条件**：
- 内容具体确定
- 表明经受要约人承诺，要约人即受该意思表示约束

**要约的撤回与撤销**：

| 类型 | 条件 |
|------|------|
| 撤回 | 撤回要约的通知应当在要约到达受要约人之前或者与要约同时到达受要约人 |
| 撤销 | 撤销要约的通知应当在受要约人发出承诺通知之前到达受要约人 |

**不得撤销的要约**：
- 要约人确定了承诺期限或者以其他形式明示要约不可撤销
- 受要约人有理由认为要约是不可撤销的，并已经为履行合同作了准备工作

#### 8.1.2 承诺

- 承诺应当在要约确定的期限内到达要约人
- 承诺的内容应当与要约的内容一致
- 承诺生效时合同成立

### 8.2 合同的效力

| 类型 | 情形 | 法律后果 |
|------|------|---------|
| 有效 | 合同成立且有效要件齐备 | 自始有效 |
| 无效 | 无民事行为能力人签订；虚假意思表示；违反强制性规定；违背公序良俗；恶意串通 | 自始无效 |
| 可撤销 | 重大误解；欺诈；胁迫；显失公平 | 撤销后自始无效 |
| 效力待定 | 限制行为能力人签订的超出范围的合同；无权代理 | 经追认后有效 |

### 8.3 合同的履行

#### 8.3.1 合同履行的原则

- 全面履行原则
- 诚实信用原则
- 绿色原则

#### 8.3.2 双务合同履行中的抗辩权

| 抗辩权 | 适用条件 | 效力 |
|--------|---------|------|
| 同时履行抗辩权 | 双方债务同时到期，一方未履行 | 拒绝履行 |
| 先履行抗辩权 | 先履行一方未履行 | 后履行一方拒绝履行 |
| 不安抗辩权 | 先履行一方有证据证明后履行一方不能履行 | 中止履行 |

### 8.4 合同的变更和转让

#### 8.4.1 合同的变更

- 当事人协商一致，可以变更合同
- 变更内容约定不明确的，推定为未变更

#### 8.4.2 合同的转让

- 债权转让：应当通知债务人
- 债务承担：应当经债权人同意

### 8.5 合同的终止

#### 8.5.1 合同终止的原因

| 原因 | 说明 |
|------|------|
| 清偿 | 债务已经履行 |
| 合同解除 | 当事人协商解除或法定解除 |
| 抵销 | 互负债务相互充抵 |
| 提存 | 债务人将标的物提存 |
| 免除 | 债权人免除债务 |
| 混同 | 债权债务同归于一人 |

#### 8.5.2 合同的解除

**法定解除的情形**：
- 因不可抗力致使不能实现合同目的
- 在履行期限届满前，当事人一方明确表示或者以自己的行为表明不履行主要债务
- 当事人一方迟延履行主要债务，经催告后在合理期限内仍未履行
- 当事人一方迟延履行债务或者有其他违约行为致使不能实现合同目的
- 法律规定的其他情形

### 8.6 违约责任

#### 8.6.1 违约责任的承担方式

| 方式 | 说明 |
|------|------|
| 继续履行 | 要求违约方按照合同约定继续履行 |
| 采取补救措施 | 修理、更换、重作、退货、减少价款或者报酬 |
| 赔偿损失 | 赔偿因违约造成的损失 |
| 违约金 | 按照合同约定支付违约金 |
| 定金 | 适用定金罚则 |

#### 8.6.2 违约金与定金

- 违约金与定金不能同时适用
- 当事人既约定违约金，又约定定金的，一方违约时，对方可以选择适用违约金或者定金条款

## 例题与解析

### 例题1

**题目**: 甲向乙发出要约，规定承诺期限为10天。第3天，甲又向乙发出撤销要约的通知。下列说法正确的是：

**选项**:
A. 要约可以撤销
B. 要约不得撤销
C. 乙已经为履行合同作了准备工作的，要约不得撤销
D. 需要乙同意才能撤销

**答案**: B

**解析**: 要约人确定了承诺期限的，要约不得撤销。

### 例题2

**题目**: 甲乙签订买卖合同，约定甲先交货，乙后付款。甲有证据证明乙经营状况严重恶化。甲可以：

**选项**:
A. 解除合同
B. 中止履行
C. 拒绝履行
D. 要求乙提供担保

**答案**: BD

**解析**: 甲享有不安抗辩权，可以中止履行；也可以要求乙提供适当担保。
