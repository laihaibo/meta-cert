# 第四章 用益物权

本章介绍用益物权的基本制度，包括土地承包经营权、建设用地使用权、宅基地使用权和地役权。

## 知识点详解

### 4.1 用益物权概述

用益物权是权利人对他人所有的不动产或者动产，依法享有占有、使用和收益的权利。

**用益物权的种类**：
- 土地承包经营权
- 建设用地使用权
- 宅基地使用权
- 地役权
- 居住权

### 4.2 土地承包经营权

#### 4.2.1 土地承包经营权的设立

- 自土地承包经营权合同生效时设立
- 登记机构应当向土地承包经营权人发放土地承包经营权证

#### 4.2.2 土地承包经营权的期限

| 类型 | 期限 |
|------|------|
| 耕地 | 30年 |
| 草地 | 30年至50年 |
| 林地 | 30年至70年 |

#### 4.2.3 土地承包经营权的流转

- 可以互换、转让
- 流转期限不得超过承包期的剩余期限
- 未经登记，不得对抗善意第三人

### 4.3 建设用地使用权

#### 4.3.1 建设用地使用权的设立方式

| 方式 | 说明 |
|------|------|
| 出让 | 国家将国有土地使用权在一定年限内出让给土地使用者，由土地使用者向国家支付土地使用权出让金 |
| 划拨 | 经县级以上人民政府批准，土地使用者缴纳补偿、安置等费用后取得的土地使用权，或者无偿取得的土地使用权 |

#### 4.3.2 建设用地使用权的流转

- 可以转让、互换、出资、赠与或者抵押
- 应当向登记机构申请变更登记

### 4.4 宅基地使用权

#### 4.4.1 宅基地使用权的特征

- 主体是农村集体经济组织成员
- 用途是建造住宅及其附属设施
- 一户一宅原则

### 4.5 地役权

#### 4.5.1 地役权的概念

地役权是指地役权人有权按照合同约定，利用他人的不动产，以提高自己的不动产的效益。

#### 4.5.2 地役权与相邻关系的区别

| 比较项 | 地役权 | 相邻关系 |
|--------|--------|---------|
| 性质 | 约定权利 | 法定权利 |
| 是否有偿 | 可以有偿 | 一般无偿 |
| 取得方式 | 合同约定 | 法律规定 |
| 登记要求 | 需要登记 | 不需要登记 |

### 4.6 居住权

#### 4.6.1 居住权的设立

- 应当采用书面形式订立居住权合同
- 应当向登记机构申请居住权登记，居住权自登记时设立
- 居住权不得转让、继承

#### 4.6.2 居住权的消灭

- 居住权期限届满或者居住权人死亡的，居住权消灭
- 居住权消灭的，应当及时办理注销登记

## 例题与解析

### 例题1

**题目**: 关于地役权，下列说法正确的是：

**选项**:
A. 地役权是法定权利
B. 地役权必须有偿
C. 地役权自合同生效时设立
D. 地役权未经登记不得对抗善意第三人

**答案**: D

**解析**: 地役权是约定权利而非法定权利；可以有偿也可以无偿；地役权自地役权合同生效时设立，但未经登记不得对抗善意第三人。
