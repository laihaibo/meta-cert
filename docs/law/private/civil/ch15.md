# 第十五章 知识产权

## 本章概要

本章介绍知识产权的基本制度，包括著作权、专利权和商标权。重点掌握著作权的保护期限和合理使用。

## 知识点详解

### 15.1 著作权

#### 15.1.1 著作权的取得

- 著作权自作品创作完成时自动取得
- 不需要登记、注册

#### 15.1.2 著作权的保护期限

| 权利类型 | 保护期限 |
|---------|---------|
| 署名权、修改权、保护作品完整权 | 永久保护 |
| 发表权、财产权利 | 作者终生及其死亡后50年 |
| 法人作品 | 作品发表后50年 |

#### 15.1.3 著作权的合理使用

**合理使用的情形**：
- 为个人学习、研究或者欣赏，使用他人已经发表的作品
- 为介绍、评论某一作品或者说明某一问题，在作品中适当引用他人已经发表的作品
- 为报道新闻，在报纸、期刊、广播电台、电视台等媒体中不可避免地再现或者引用已经发表的作品
- 其他合理使用的情形

### 15.2 专利权

#### 15.2.1 专利权的取得条件

| 类型 | 条件 |
|------|------|
| 发明 | 新颖性、创造性、实用性 |
| 实用新型 | 新颖性、创造性、实用性 |
| 外观设计 | 新颖性、不属于现有设计 |

#### 15.2.2 专利权的保护期限

| 类型 | 保护期限 |
|------|---------|
| 发明 | 20年 |
| 实用新型 | 10年 |
| 外观设计 | 15年 |

### 15.3 商标权

#### 15.3.1 商标权的取得

- 商标权通过注册取得
- 经商标局核准注册的商标为注册商标

#### 15.3.2 商标权的保护期限

- 注册商标的有效期为10年
- 自核准注册之日起计算
- 期满需要继续使用的，应当在期满前12个月内办理续展手续

## 例题与解析

### 例题1

**题目**: 甲创作了一部小说，该小说的著作权何时产生？

**选项**:
A. 甲向版权局登记时
B. 甲创作完成时
C. 甲发表时
D. 甲与出版社签订合同时

**答案**: B

**解析**: 著作权自作品创作完成时自动取得，不需要登记、注册。
