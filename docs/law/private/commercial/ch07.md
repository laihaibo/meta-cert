# 第七章 保险法

## 本章概要

本章介绍保险法的基本制度，包括保险合同和保险理赔。重点掌握保险合同的特征和保险理赔。

## 知识点详解

### 7.1 保险合同的特征

| 特征 | 说明 |
|------|------|
| 射幸合同 | 保险合同的效果在订约时不能确定 |
| 最大诚信合同 | 保险合同的当事人必须以最大的诚信订立合同 |
| 附和合同 | 保险合同的内容由保险人一方预先拟定 |

### 7.2 保险合同的当事人

| 当事人 | 说明 |
|--------|------|
| 保险人 | 与投保人订立保险合同，并按照合同约定承担赔偿或者给付保险金责任的保险公司 |
| 投保人 | 与保险人订立保险合同，并按照合同约定负有支付保险费义务的人 |
| 被保险人 | 其财产或者人身受保险合同保障，享有保险金请求权的人 |
| 受益人 | 人身保险合同中由被保险人或者投保人指定的享有保险金请求权的人

### 7.3 保险理赔

**理赔程序**：
- 投保人、被保险人或者受益人知道保险事故发生后，应当及时通知保险人
- 保险人收到被保险人或者受益人的赔偿或者给付保险金的请求后，应当及时作出核定

**代位求偿权**：
- 因第三者对保险标的的损害而造成保险事故的，保险人自向被保险人赔偿保险金之日起，在赔偿金额范围内代位行使被保险人对第三者请求赔偿的权利

## 例题与解析

### 例题1

**题目**: 下列关于保险合同的说法，正确的是：

**options**:
A. 保险合同是射幸合同
B. 保险合同是诺成合同
C. 保险合同是有偿合同
D. A和C

**答案**: D

**解析**: 保险合同是射幸合同和有偿合同。
