# 第六章 票据法

## 本章概要

本章介绍票据法的基本制度，包括汇票、本票和支票。重点掌握票据的特征和票据行为。

## 知识点详解

### 6.1 票据的特征

| 特征 | 说明 |
|------|------|
| 设权证券 | 票据权利的产生必须作成票据 |
| 要式证券 | 票据的作成必须依法定方式 |
| 文义证券 | 票据权利的内容以票据上记载的文义为准 |
| 无因证券 | 票据权利的行使不以原因关系的存在为前提 |

### 6.2 票据行为

| 行为 | 说明 |
|------|------|
| 出票 | 出票人签发票据并将其交付给收款人的票据行为 |
| 背书 | 在票据背面或者粘单上记载有关事项并签章的票据行为 |
| 承兑 | 汇票付款人承诺在汇票到期日支付汇票金额的票据行为 |
| 保证 | 票据债务人以外的人，为担保票据债务的履行，以负担同一内容的票据债务为目的所为的票据行为 |

### 6.3 汇票、本票、支票的区别

| 比较项 | 汇票 | 本票 | 支票 |
|--------|------|------|------|
| 付款人 | 付款人 | 出票人 | 付款人 |
| 付款期限 | 见票即付或定日付款 | 见票即付 | 见票即付 |
| 出票人 | 任何人 | 银行 | 任何人 |

## 例题与解析

### 例题1

**题目**: 下列关于票据的说法，正确的是：

**options**:
A. 票据是有因证券
B. 票据是无因证券
C. 票据是要式证券
D. B和C

**答案**: D

**解析**: 票据是无因证券和要式证券。
