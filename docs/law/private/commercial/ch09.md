# 第九章 海商法

## 本章概要

本章介绍海商法的基本制度，包括船舶和海上货物运输。

- 船舶所有权

### 考点2: 海上货物运输
- 海上货物运输合同

## 知识点详解

### 9.1 船舶物权

| 物权 | 说明 |
|------|------|
| 船舶所有权 | 船舶所有人依法对自己的船舶享有占有、使用、收益和处分的权利 |
| 船舶抵押权 | 抵押权人对于抵押人提供的作为债务担保的船舶，在抵押人不履行债务时，可以依法拍卖，从卖得的价款中优先受偿的权利 |
| 船舶优先权 | 海事请求人依法向船舶所有人、光船承租人、船舶经营人提出海事请求，对产生该海事请求的船舶具有优先受偿的权利 |

### 9.2 海上货物运输

**海上货物运输合同**：
- 承运人或者托运人可以要求书面确认海上货物运输合同的成立

**提单**：
- 提单是承运人收到货物后签发的，证明承运人已收到货物的书面凭证

## 例题与解析

### 例题1

**题目**: 下列关于船舶优先权的说法，正确的是：

**options**:
A. 船舶优先权可以转让
B. 船舶优先权不可以转让
C. 船舶优先权必须登记
D. 船舶优先权没有优先效力

**答案**: B

**解析**: 船舶优先权不可以转让。
