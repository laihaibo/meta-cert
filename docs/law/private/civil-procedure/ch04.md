# 第四章 证据

## 本章概要

本章介绍民事诉讼中的证据制度，包括证据的种类、证明责任和质证。重点掌握证明责任的分配。

## 知识点详解

### 4.1 证据的种类

| 种类 | 说明 |
|------|------|
| 当事人的陈述 | 当事人就案件事实向人民法院所作的陈述 |
| 书证 | 以其记载的内容和表达的思想证明案件事实的书面材料 |
| 物证 | 以其外部特征、物质属性、存在状况等证明案件事实的实物 |
| 视听资料 | 以录音、录像等设备储存的信息证明案件事实的资料 |
| 电子数据 | 以电子计算机等设备储存的信息证明案件事实的资料 |
| 证人证言 | 证人就其所知道的案件情况向人民法院所作的陈述 |
| 鉴定意见 | 鉴定人对案件中的专门性问题进行鉴定后所作的书面意见 |
| 勘验笔录 | 审判人员对与案件有关的场所、物品等进行勘验后所作的记录 |

### 4.2 证明责任

**一般规则**：
- 当事人对自己提出的主张，有责任提供证据

**举证责任倒置**：
- 因新产品制造方法发明专利引起的专利侵权诉讼
- 高度危险作业致人损害的侵权诉讼
- 因环境污染引起的损害赔偿诉讼
- 建筑物或者其他设施以及建筑物上的搁置物、悬挂物发生倒塌、脱落、坠落致人损害的侵权诉讼
- 饲养动物致人损害的侵权诉讼
- 因缺陷产品致人损害的侵权诉讼

### 4.3 质证

**质证的程序**：
- 证据应当在法庭上出示，由当事人互相质证
- 未经当事人质证的证据，不得作为认定案件事实的根据

## 例题与解析

### 例题1

**题目**: 民事诉讼中，证明责任的一般规则是：

**选项**:
A. 被告举证
B. 原告举证
C. 谁主张谁举证
D. 法院举证

**答案**: C

**解析**: 民事诉讼中，证明责任的一般规则是谁主张谁举证。
