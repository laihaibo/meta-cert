# 民事诉讼法

## 学科概述

民事诉讼法是法考的重要科目，主要规范民事诉讼活动。本学科内容涵盖民事诉讼的基本原则、管辖、当事人、证据、保全与先予执行、审判程序、执行程序和仲裁制度等。
