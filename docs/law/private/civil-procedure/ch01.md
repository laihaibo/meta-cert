# 第一章 民事诉讼法概述

## 本章概要

本章介绍民事诉讼法的基本概念和基本原则。

## 高频考点

> 🔥 高频考点

### 考点1: 民事诉讼法的基本原则
- 当事人诉讼权利平等原则
- 辩论原则
- 处分原则
- 调解原则
- 诚实信用原则

### 考点2: 民事诉讼的基本制度
- 合议制度
- 回避制度
- 公开审判制度
- 两审终审制度

## 知识点详解

### 1.1 民事诉讼法的基本原则

| 原则 | 内容 |
|------|------|
| 当事人诉讼权利平等 | 当事人有平等的诉讼权利 |
| 辩论原则 | 当事人有权进行辩论 |
| 处分原则 | 当事人有权在法律规定的范围内处分自己的民事权利和诉讼权利 |
| 调解原则 | 人民法院审理民事案件，应当根据自愿和合法的原则进行调解 |
| 诚实信用原则 | 当事人行使诉讼权利应当遵循诚实信用原则 |

### 1.2 民事诉讼的基本制度

| 制度 | 内容 |
|------|------|
| 合议制度 | 由审判员组成合议庭或者由审判员和人民陪审员组成合议庭审理案件 |
| 回避制度 | 审判人员有法定情形的，应当回避 |
| 公开审判制度 | 人民法院审理民事案件，除涉及国家秘密、个人隐私或者法律另有规定的以外，应当公开进行 |
| 两审终审制度 | 一个民事案件经过两级人民法院审理后即告终结 |

## 例题与解析

### 例题1

**题目**: 下列哪项不属于民事诉讼法的基本原则？

**options**:
A. 当事人诉讼权利平等原则
B. 辩论原则
C. 罪刑法定原则
D. 处分原则

**答案**: C

**解析**: 罪刑法定原则是刑法的基本原则，不是民事诉讼法的基本原则。
