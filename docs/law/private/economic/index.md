# 经济法

## 科目概述

经济法是法考的重要组成部分，涉及市场竞争、消费者保护、产品质量、食品安全、金融监管、税收、劳动关系、土地管理、房地产、环境保护等多个领域。经济法具有较强的行政管理色彩，涉及大量行政法规和部门规章。

## 备考建议

1. 重点掌握竞争法、消费者权益保护法和劳动法
2. 注意各法之间的联系，如消费者权益保护法与产品质量法的竞合
3. 关注最新立法动态，如《反垄断法》2022年修正
4. 结合实际案例理解法律适用
