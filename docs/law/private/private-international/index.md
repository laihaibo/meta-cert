# 国际私法

## 科目概述

国际私法是调整涉外民商事关系的法律部门，主要解决法律冲突问题。本学科包括国际私法的基本理论、冲突规范、适用冲突规范的制度、国际民商事法律适用、国际民商事争议解决等内容。
