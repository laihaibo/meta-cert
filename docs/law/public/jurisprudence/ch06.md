# 第六章 法律责任

## 本章概要

本章阐述法律责任的概念、构成要件、归责原则和免责事由。

## 高频考点

1. 法律责任的种类
2. 归责原则
3. 免责事由

## 重点难点

- **重点**：归责原则、免责事由
- **难点**：法律责任与法律义务的区别

## 知识点详解

### 一、法律责任的概念

#### 1. 定义

法律责任是指行为人由于违法行为、违约行为或者由于法律规定而应承受的某种不利的法律后果。

#### 2. 法律责任与法律义务的区别

| 比较项目 | 法律责任 | 法律义务 |
|----------|----------|----------|
| 产生原因 | 违法、违约或法律规定 | 法律规范的规定 |
| 性质 | 不利后果 | 应为约束 |
| 履行 | 被动承担 | 主动履行 |

### 二、法律责任的种类

| 类型 | 产生原因 | 举例 |
|------|----------|------|
| 刑事责任 | 犯罪行为 | 有期徒刑、罚金 |
| 民事责任 | 违约或侵权 | 赔偿损失、赔礼道歉 |
| 行政责任 | 行政违法 | 行政处罚、行政处分 |
| 违宪责任 | 违反宪法 | 撤销违宪法律 |

### 三、法律责任的构成要件

1. **主体**：责任主体必须具有责任能力
2. **违法行为或违约行为**：包括作为和不作为
3. **损害结果**：实际造成的损害
4. **因果关系**：行为与结果之间的因果联系
5. **主观过错**：故意或过失（部分责任不以过错为要件）

### 四、归责原则

#### 1. 责任法定原则

- 法律责任的追究必须有法律依据
- 法律责任由法律规范预先规定

#### 2. 因果联系原则

- 行为与损害结果之间必须存在因果关系

#### 3. 责任相当原则（比例原则）

- 法律责任的大小与违法行为的程度相适应

#### 4. 责任自负原则

- 谁违法谁负责
- 不允许株连

### 五、免责事由

| 免责类型 | 含义 |
|----------|------|
| 时效免责 | 超过法定追诉时效 |
| 不诉免责 | 告诉才处理的案件，不告诉则免责 |
| 自首立功免责 | 自首或立功可以减轻或免除 |
| 补救免责 | 实施违法行为后主动补救 |
| 协议免责 | 双方当事人协商同意免除（限于私法） |
| 人道主义免责 | 基于人道主义考虑 |
| 不可抗力免责 | 因不可抗力导致的违法 |
| 正当防卫免责 | 正当防卫不负法律责任 |
| 紧急避险免责 | 紧急避险不负或减轻法律责任 |

## 例题与解析

### 例题1

下列不属于归责原则的是：

A. 责任法定原则
B. 因果联系原则
C. 推定过错原则
D. 责任相当原则

**答案**：C

**解析**：归责原则包括责任法定原则、因果联系原则、责任相当原则和责任自负原则。推定过错是侵权责任的归责方式之一，不属于法理学中的归责原则。

### 例题2

下列属于免责事由的是：

A. 时效免责
B. 主观无过错
C. 行为合法
D. 未造成损害

**答案**：A

**解析**：时效免责是指超过法定追诉时效可以免除法律责任。主观无过错、行为合法、未造成损害属于不构成法律责任的情形，不属于免责。
