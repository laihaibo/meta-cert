# 法理学

## 科目概述

法理学是法学的基础理论学科，研究法的一般理论、基本问题和基本规律。在法考中，法理学是公共科目的重要组成部分，为其他法学学科提供理论基础。

## 考情分析

- 分值占比：约15-20分
- 题型：选择题为主
- 难度：中等偏难，需要理解抽象概念
