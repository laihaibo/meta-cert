# 第七章 法律解释

## 本章概要

本章阐述法律解释的概念、方法和体制，是法考中常考的知识点。

## 高频考点

1. 法律解释的分类
2. 法律解释的方法
3. 我国的法律解释体制

## 重点难点

- **重点**：法律解释的方法、我国法律解释体制
- **难点**：各种解释方法的适用

## 知识点详解

### 一、法律解释的概念

#### 1. 定义

法律解释是指一定的人或组织对法律规定含义的说明。

#### 2. 必要性

- 法律规范的抽象性与社会生活的具体性之间的矛盾
- 法律的稳定性与社会的变动性之间的矛盾
- 法律的普遍性与个案的特殊性之间的矛盾

### 二、法律解释的分类

| 分类标准 | 类型 | 含义 |
|----------|------|------|
| 解释主体和效力 | 正式解释（法定解释） | 特定国家机关的解释 |
| | 非正式解释（学理解释） | 学者或其他组织的解释 |
| 解释尺度 | 字面解释 | 严格按照文字字面含义 |
| | 扩大解释（扩充解释） | 含义宽于字面含义 |
| | 限制解释（缩小解释） | 含义窄于字面含义 |

### 三、法律解释的方法

#### 1. 文义解释（语法解释）

- 按照法律条文的语言文字的含义进行解释
- 是最基本、最常用的解释方法

#### 2. 体系解释

- 联系其他条文进行解释
- 保持法律体系的统一性

#### 3. 历史解释（立法解释）

- 参照立法时的历史资料进行解释
- 探求立法者的原意

#### 4. 目的解释

- 根据法律的目的进行解释
- 使法律的适用符合立法目的

#### 5. 合宪性解释

- 按照宪法的精神解释法律
- 使法律的解释不与宪法相抵触

#### 6. 比较解释

- 参照外国立法和判例进行解释

### 四、我国的法律解释体制

#### 1. 正式解释的分类

| 类型 | 主体 | 效力 |
|------|------|------|
| 立法解释 | 全国人大常委会 | 与法律同等效力 |
| 司法解释 | 最高人民法院、最高人民检察院 | 在司法领域有约束力 |
| 行政解释 | 国务院及其主管部门 | 在行政管理领域有约束力 |

#### 2. 法律解释的要求

- 以宪法为指导
- 符合立法目的和原则
- 维护法制统一

#### 3. 《立法法》关于法律解释的规定

- 法律解释权属于全国人民代表大会常务委员会
- 法律的规定需要进一步明确具体含义的，由全国人大常委会解释
- 法律制定后出现新的情况，需要明确适用法律依据的，由全国人大常委会解释

## 例题与解析

### 例题1

将"武器"解释为包括枪支、弹药、爆炸物等，属于：

A. 字面解释
B. 扩大解释
C. 限制解释
D. 类推解释

**答案**：B

**解析**：将"武器"解释为包括枪支、弹药、爆炸物等，是对其含义的扩大，属于扩大解释。

### 例题2

我国法律解释体制中，法律解释权属于：

A. 最高人民法院
B. 最高人民检察院
C. 全国人大常委会
D. 国务院

**答案**：C

**解析**：根据《立法法》规定，法律解释权属于全国人民代表大会常务委员会。
