# 第四章 隋唐法律制度

## 本章概要

本章介绍隋朝和唐朝的法律制度，重点掌握《唐律疏议》的内容和特点。

## 高频考点

1. 《开皇律》
2. 《唐律疏议》
3. 唐律的特点和历史地位

## 重点难点

- **重点**：《唐律疏议》的内容和特点
- **难点**：唐律的刑罚制度

## 知识点详解

### 一、隋朝法律制度

#### 1. 《开皇律》

- 隋文帝时期编纂
- 确立**新五刑**：笞、杖、徒、流、死
- 改"重罪十条"为**十恶**

#### 2. 十恶制度

| 罪名 | 含义 |
|------|------|
| 谋反 | 谋害皇帝、危害国家 |
| 谋大逆 | 毁坏宗庙、陵墓、宫阙 |
| 谋叛 | 叛国投敌 |
| 恶逆 | 殴打、杀害尊亲属 |
| 不道 | 杀一家非死罪三人 |
| 大不敬 | 盗用皇帝器物 |
| 不孝 | 不孝敬父母 |
| 不睦 | 谋害亲属 |
| 不义 | 杀上级长官 |
| 内乱 | 亲属间乱伦 |

### 二、唐朝法律制度

#### 1. 法律形式

| 形式 | 含义 |
|------|------|
| 律 | 刑事法律规范 |
| 令 | 国家制度规范 |
| 格 | 国家机关的办事规则 |
| 式 | 国家机关的公文程式 |

#### 2. 《唐律疏议》

- **地位**：中国历史上保存最完整、最早、最具有社会影响力的封建法典
- **结构**：12篇502条
- **内容**：以《名例律》为总则，其余11篇为分则

#### 3. 唐律的篇目

| 篇名 | 内容 |
|------|------|
| 名例律 | 刑罚制度和适用原则 |
| 卫禁律 | 宫殿警卫、关津防守 |
| 职制律 | 官吏职务犯罪 |
| 户婚律 | 户籍、婚姻、继承 |
| 厩库律 | 牲畜、仓库管理 |
� 擅兴律 | 军事征调、工程营造 |
| 贼盗律 | 谋反、杀人、盗窃 |
| 斗讼律 | 斗殴、诉讼 |
| 诈伪律 | 诈骗、伪造 |
| 杂律 | 不能归入其他篇的犯罪 |
| 捕亡律 | 追捕逃犯 |
| 断狱律 | 审判制度 |

#### 4. 唐律的特点

| 特点 | 内容 |
|------|------|
| 礼法合一 | "一准乎礼" |
| 科条简要 | 条文简约、刑罚适中 |
| 用刑持平 | 刑罚较为宽缓 |
| 语言精练 | 文字简要明确 |

#### 5. 唐律的刑罚制度

**五刑制度**：

| 刑罚 | 等级 |
|------|------|
| 笞刑 | 十、二十、三十、四十、五十 |
| 杖刑 | 六十、七十、八十、九十、一百 |
| 徒刑 | 一年、一年半、二年、二年半、三年 |
| 流刑 | 二千里、二千五百里、三千里 |
| 死刑 | 绞、斩 |

#### 6. 唐律的刑法原则

| 原则 | 含义 |
|------|------|
| 十恶不赦 | 犯十恶者不得赦免 |
| 八议 | 贵族官僚犯罪减免处罚 |
| 上请 | 五品以上官僚犯罪上请皇帝裁决 |
| 老幼废疾减免刑罚 | 老幼残疾犯罪减免处罚 |
| 自首减免刑罚 | 自首者减轻处罚 |
| 同居相为隐 | 亲属之间可以隐瞒犯罪 |

#### 7. 唐律的历史地位

- 对中国古代法律发展有深远影响
- 对亚洲各国法律产生重要影响
- 被称为"中华法系"的代表

## 例题与解析

### 例题1

《唐律疏议》共几篇？

A. 十篇
B. 十一篇
C. 十二篇
D. 十三篇

**答案**：C

**解析**：《唐律疏议》共12篇502条。

### 例题2

唐律的特点不包括：

A. 礼法合一
B. 科条简要
C. 严刑峻法
D. 用刑持平

**答案**：C

**解析**：唐律的特点是礼法合一、科条简要、用刑持平、语言精练。严刑峻法不是唐律的特点。
