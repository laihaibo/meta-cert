# 第三章 魏晋南北朝法律制度

## 本章概要

本章介绍三国两晋南北朝时期的法律制度，重点掌握法典编纂和法律儒家化的进一步发展。

## 高频考点

1. 重要法典（《新律》《泰始律》《北魏律》《北齐律》）
2. 八议制度
3. 准五服以制罪

## 重点难点

- **重点**：各朝代重要法典的特点
- **难点**：法律儒家化的具体表现

## 知识点详解

### 一、三国法律制度

#### 1. 曹魏《新律》

- 由陈群、刘劭等编纂
- 将《具律》改为《刑名》置于律首
- **八议制度**入律

#### 2. 八议制度

| 类型 | 对象 |
|------|------|
| 议亲 | 皇室宗亲 |
| 议故 | 皇帝故旧 |
| 议贤 | 有大德行者 |
| 议能 | 有大才能者 |
| 议功 | 有大功勋者 |
| 议贵 | 贵族官僚 |
| 议勤 | 有大勤劳者 |
| 议宾 | 前朝贵族 |

### 二、两晋法律制度

#### 1. 《泰始律》（晋律）

- 由贾充、杜预等编纂
- 增设《法例》篇
- 张斐、杜预为之作注

#### 2. 注释律学

- 张斐、杜预的律注具有法律效力
- 开创了"注释律学"的传统

### 三、南北朝法律制度

#### 1. 《北魏律》

- 采汉律、魏晋律之长
- **存留养亲制度**入律

#### 2. 《北齐律》

- **地位**：南北朝时期最重要的法典
- 首创《名例律》作为法典总则
- 确立"重罪十条"

#### 3. 重罪十条

| 罪名 | 含义 |
|------|------|
| 反逆 | 谋反大逆 |
| 大逆 | 毁坏宗庙、宫阙 |
| 叛 | 叛国投敌 |
| 降 | 投降敌人 |
| 恶逆 | 殴打、杀害尊亲属 |
| 不道 | 杀一家非死罪三人 |
| 不敬 | 盗用皇帝器物 |
| 不孝 | 不孝敬父母 |
| 不义 | 杀上级长官 |
| 内乱 | 亲属间乱伦 |

### 四、法律儒家化的进一步发展

| 制度 | 朝代 | 内容 |
|------|------|------|
| 八议 | 曹魏 | 贵族官僚犯罪减免处罚 |
| 准五服以制罪 | 晋 | 按照亲属关系远近定罪量刑 |
| 存留养亲 | 北魏 | 独子犯罪可留家养亲 |
| 官当 | 南陈 | 以官品抵罪 |

## 例题与解析

### 例题1

八议制度最早入律于：

A. 《法经》
B. 《新律》
C. 《泰始律》
D. 《北齐律》

**答案**：B

**解析**：八议制度最早入律于曹魏《新律》。

### 例题2

南北朝时期最重要的法典是：

A. 《北魏律》
B. 《北齐律》
C. 《梁律》
D. 《陈律》

**答案**：B

**解析**：《北齐律》是南北朝时期最重要的法典，首创《名例律》，确立"重罪十条"。
