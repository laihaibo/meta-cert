# 第二章 秦汉法律制度

## 本章概要

本章介绍秦朝和汉朝的法律制度，重点掌握秦朝的法律形式和汉朝的法律儒家化。

## 高频考点

1. 秦朝的法律形式
2. 汉朝的法律儒家化
3. 春秋决狱

## 重点难点

- **重点**：秦朝法律形式、汉朝法律儒家化
- **难点**：春秋决狱的含义和意义

## 知识点详解

### 一、秦朝法律制度

#### 1. 法律形式

| 形式 | 含义 |
|------|------|
| 律 | 正式制定的法律规范 |
| 令 | 皇帝的命令 |
| 式 | 关于案件调查、勘验、审讯的文书程式 |
| 法律答问 | 对法律条文的解释 |
| 廷行事 | 判例 |

#### 2. 刑法原则

| 原则 | 含义 |
|------|------|
| 以身高为标准 | 男六尺五寸、女六尺二寸为成年 |
| 区分故意与过失 | 故意称"端"，过失称"不端" |
| 连坐制度 | 一人犯罪，牵连亲属、邻里 |
| 诬告反坐 | 诬告者承担所告罪名的刑罚 |

#### 3. 主要罪名

- 谋反、不道、不敬、诽谤、妖言

### 二、汉朝法律制度

#### 1. 法律形式

| 形式 | 含义 |
|------|------|
| 律 | 基本法律形式 |
| 令 | 皇帝的命令 |
| 科 | 单行法规 |
| 比 | 判例（决事比） |

#### 2. 汉朝法律改革

- **约法三章**：刘邦入关中"杀人者死，伤人及盗抵罪"
- **《九章律》**：萧何在《法经》六篇基础上增加《户律》《兴律》《厩律》

#### 3. 法律儒家化

| 措施 | 内容 |
|------|------|
| 上请制度 | 贵族官僚犯罪须上请皇帝裁决 |
| 恤刑制度 | 老幼妇残犯罪减免刑罚 |
| 亲亲得相首匿 | 直系亲属之间可以隐瞒犯罪 |

#### 4. 春秋决狱

- **含义**：以《春秋》经义作为审判案件的依据
- **创始者**：董仲舒
- **原则**："论心定罪"——以犯罪者的主观动机作为定罪量刑的主要依据
- **意义**：开启了法律儒家化的进程

#### 5. 秋冬行刑

- 除谋反大逆外，死刑一般在秋冬执行
- 体现了"顺天行刑"的思想

### 三、秦汉监察制度

#### 1. 秦朝

- 御史大夫：中央监察长官
- 监御史：地方监察官员

#### 2. 汉朝

- 御史大夫：中央监察长官
- 刺史：地方监察官员（州刺史）
- 司隶校尉：监察京师地区

## 例题与解析

### 例题1

秦朝的法律形式不包括：

A. 律
B. 令
C. 比
D. 法律答问

**答案**：C

**解析**：秦朝的法律形式包括律、令、式、法律答问、廷行事。"比"是汉朝的法律形式。

### 例题2

春秋决狱的原则是：

A. 以事实为根据
B. 论心定罪
C. 罪刑法定
D. 疑罪从轻

**答案**：B

**解析**：春秋决狱的原则是"论心定罪"，以犯罪者的主观动机作为定罪量刑的主要依据。
