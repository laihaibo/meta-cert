# 第六章 明清法律制度

## 本章概要

本章介绍明朝和清朝的法律制度，重点掌握明朝的《大明律》和清朝的《大清律例》。

## 高频考点

1. 《大明律》的特点
2. 明朝的刑罚制度
3. 《大清律例》

## 重点难点

- **重点**：《大明律》的特点
- **难点**：明清刑罚制度的变化

## 知识点详解

### 一、明朝法律制度

#### 1. 《大明律》

- **编纂者**：朱元璋亲自主持
- **体例**：7篇460条（名例律、吏律、户律、礼律、兵律、刑律、工律）
- **特点**：以六部分篇，条文简于唐律

#### 2. 《大诰》

- 朱元璋亲自编纂
- 严刑惩治贪官污吏
- 有独立的法律效力

#### 3. 明朝的刑罚制度

| 刑罚 | 特点 |
|------|------|
| 廷杖 | 在朝廷上杖责大臣 |
| 充军 | 流放到边远地区充军 |
| 刺字 | 在脸上刺字 |
| 凌迟 | 最残酷的死刑 |

#### 4. 明朝的司法制度

- **三法司**：刑部、大理寺、都察院
- **厂卫制度**：东厂、西厂、锦衣卫

### 二、清朝法律制度

#### 1. 《大清律例》

- **地位**：中国历史上最后一部封建成文法典
- **体例**：律文436条，附例1049条
- **特点**：律例合编

#### 2. 清朝的法律特点

| 特点 | 内容 |
|------|------|
| 律例并行 | 律文与条例并行 |
| 满汉异法 | 满族人和汉人适用不同法律 |
| 维护旗地旗产 | 保护满族特权 |

#### 3. 清朝的司法制度

- **三法司**：刑部、大理寺、都察院
- **秋审**：对死刑案件的复核制度
- **朝审**：对京师死刑案件的复核

#### 4. 秋审的处理结果

| 结果 | 含义 |
|------|------|
| 情实 | 罪情属实，执行死刑 |
| 缓决 | 暂缓执行，下次再审 |
| 可矜 | 值得怜悯，减等处罚 |
| 留养承嗣 | 独子可留家养亲 |

## 例题与解析

### 例题1

《大明律》的体例是：

A. 12篇
B. 7篇
C. 6篇
D. 9篇

**答案**：B

**解析**：《大明律》共7篇：名例律、吏律、户律、礼律、兵律、刑律、工律。

### 例题2

秋审的处理结果不包括：

A. 情实
B. 缓决
C. 立决
D. 可矜

**答案**：C

**解析**：秋审的处理结果包括情实、缓决、可矜、留养承嗣。立决不是秋审的结果。
