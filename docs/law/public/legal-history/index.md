# 中国法律史

## 科目概述

中国法律史研究中国历史上法律制度的产生、发展和变化规律。在法考中，中国法律史主要考查各朝代的重要法律制度和法典。

## 考情分析

- 分值占比：约10-15分
- 题型：选择题为主
- 难度：中等，需要记忆大量历史知识
