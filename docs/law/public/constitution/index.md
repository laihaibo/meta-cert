# 宪法

## 科目概述

宪法是国家的根本法，是治国安邦的总章程。宪法学是法考公共科目的重要组成部分，涉及宪法基本理论、国家制度、公民权利和国家机构等内容。

## 考情分析

- 分值占比：约20-25分
- 题型：选择题为主
- 难度：中等，知识点分散需要系统记忆
