# 第一章 宪法基本理论

## 本章概要

本章阐述宪法的基本概念、特征、分类和基本原则，是学习宪法学的基础。

## 高频考点

1. 宪法的特征（根本法特征）
2. 宪法的基本原则
3. 宪法与普通法律的区别

## 重点难点

- **重点**：宪法的特征、基本原则
- **难点**：宪法的最高法律效力

## 知识点详解

### 一、宪法的概念

宪法是集中体现统治阶级意志，规定国家根本制度和根本任务，具有最高法律效力的国家根本法。

### 二、宪法的特征

| 特征 | 内容 |
|------|------|
| 内容最根本 | 规定国家最根本、最重要的问题 |
| 效力最高 | 具有最高法律效力，是一切法律的立法依据 |
| 制定修改程序最严格 | 制定和修改的程序比普通法律更严格 |

### 三、宪法与普通法律的区别

| 比较项目 | 宪法 | 普通法律 |
|----------|------|----------|
| 内容 | 国家根本制度和根本任务 | 某一方面的社会关系 |
| 效力 | 最高法律效力 | 依据宪法制定 |
| 制定主体 | 特定制宪机关 | 立法机关 |
| 修改程序 | 更严格 | 相对简单 |

### 四、宪法的分类

| 分类标准 | 类型 |
|----------|------|
| 是否有统一法典形式 | 成文宪法与不成文宪法 |
| 效力和修改程序 | 刚性宪法与柔性宪法 |
| 制定主体 | 钦定宪法、民定宪法、协定宪法 |

### 五、宪法的基本原则

#### 1. 人民主权原则

- 国家的一切权力属于人民
- 人民行使国家权力的机关是全国人民代表大会和地方各级人民代表大会

#### 2. 基本人权原则

- 国家尊重和保障人权
- 公民享有宪法和法律规定的权利

#### 3. 法治原则

- 依法治国，建设社会主义法治国家
- 任何组织或者个人都不得有超越宪法和法律的特权

#### 4. 权力制约原则

- 国家权力的各部分之间相互监督、彼此牵制
- 保障公民权利

### 六、我国宪法的历次修改

| 修改时间 | 主要内容 |
|----------|----------|
| 1982年 | 现行宪法颁布 |
| 1988年 | 私营经济、土地使用权转让 |
| 1993年 | 社会主义市场经济、多党合作和政治协商制度 |
| 1999年 | 依法治国、邓小平理论 |
| 2004年 | 人权入宪、私有财产保护、国歌 |
| 2018年 | 习近平新时代中国特色社会主义思想、监察委员会、国家主席任期 |

## 例题与解析

### 例题1

宪法的特征不包括：

A. 内容最根本
B. 效力最高
C. 制定修改程序最严格
D. 数量最多

**答案**：D

**解析**：宪法的特征包括内容最根本、效力最高、制定修改程序最严格。数量最多不是宪法的特征。

### 例题2

我国宪法的基本原则不包括：

A. 人民主权原则
B. 三权分立原则
C. 基本人权原则
D. 法治原则

**答案**：B

**解析**：我国宪法的基本原则包括人民主权、基本人权、法治、权力制约。我国不实行三权分立。
