# 第八章 监察委员会

## 本章概要

本章阐述监察委员会的性质、地位、组成、职权和监察范围，是2018年宪法修正案新增的重要内容。

## 高频考点

1. 监察委员会的性质和地位
2. 监察委员会的职权
3. 监察范围

## 重点难点

- **重点**：监察委员会的职权和监察范围
- **难点**：监察委与其他国家机关的关系

## 知识点详解

### 一、监察委员会的性质和地位

- 国家监察机关
- 行使国家监察职能的专责机关
- 国家监察委员会是最高监察机关

### 二、监察委员会的组成

| 级别 | 组成 | 产生方式 |
|------|------|----------|
| 国家监察委 | 主任、副主任、委员 | 主任由全国人大选举，其他由全国人大常委会任免 |
| 省级监察委 | 主任、副主任、委员 | 主任由省级人大选举，其他由省级人大常委会任免 |
| 县级监察委 | 主任、副主任、委员 | 主任由县级人大选举，其他由县级人大常委会任免 |

### 三、监察委员会的职权

1. **监督**：对公职人员开展廉政教育
2. **调查**：对涉嫌贪污贿赂、滥用职权、玩忽职守、权力寻租、利益输送、徇私舞弊以及浪费国家资财等职务违法和职务犯罪进行调查
3. **处置**：对违法的公职人员依法作出政务处分决定

### 四、监察范围

对下列公职人员和有关人员进行监察：
1. 中国共产党机关、人大机关、行政机关、政协机关、监察机关、审判机关、检察机关、民主党派机关和工商联机关的公务员
2. 法律法规授权或者受国家机关依法委托管理公共事务的组织中从事公务的人员
3. 国有企业管理人员
4. 公办的教育、科研、文化、医疗卫生、体育等单位中从事管理的人员
5. 基层群众性自治组织中从事管理的人员
6. 其他依法履行公职的人员

### 五、监察委员会的领导体制

- 国家监察委员会领导地方各级监察委员会的工作
- 上级监察委员会领导下级监察委员会的工作
- 监察委员会对产生它的国家权力机关和上一级监察委员会负责

### 六、监察委与其他国家机关的关系

- 与人大的关系：对人大负责，受人大监督
- 与司法机关的关系：办理职务违法和职务犯罪案件，与审判机关、检察机关互相配合、互相制约
- 与执法机关的关系：与执法部门互相配合

## 例题与解析

### 例题1

监察委员会的性质是：

A. 行政机关
B. 司法机关
C. 国家监察机关
D. 立法机关

**答案**：C

**解析**：监察委员会是国家监察机关，行使国家监察职能。

### 例题2

下列不属于监察范围的是：

A. 公务员
B. 国有企业管理人员
C. 私营企业普通员工
D. 公办学校管理人员

**答案**：C

**解析**：监察范围包括公职人员和有关人员，私营企业普通员工不属于监察范围。
