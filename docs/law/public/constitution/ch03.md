# 第三章 公民基本权利与义务

## 本章概要

本章阐述我国公民享有的基本权利和应履行的基本义务，是宪法学的核心内容之一。

## 高频考点

1. 平等权
2. 政治权利和自由
3. 人身自由
4. 社会经济权利

## 重点难点

- **重点**：各项基本权利的具体内容
- **难点**：权利的限制与保障

## 知识点详解

### 一、公民的基本权利

#### 1. 平等权

- 公民在法律面前一律平等
- 任何公民享有宪法和法律规定的权利，同时必须履行宪法和法律规定的义务

#### 2. 政治权利和自由

| 权利 | 内容 |
|------|------|
| 选举权和被选举权 | 年满18周岁，未被剥夺政治权利的公民 |
| 言论自由 | 通过语言表达思想的自由 |
| 出版自由 | 通过出版物表达思想的自由 |
| 集会自由 | 聚集在一定场所的自由 |
| 结社自由 | 组成一定社会团体的自由 |
| 游行自由 | 列队行进表达意愿的自由 |
| 示威自由 | 表达要求或抗议的自由 |

#### 3. 宗教信仰自由

- 公民有信仰宗教的自由
- 有不信仰宗教的自由
- 国家保护正常的宗教活动

#### 4. 人身自由

| 权利 | 内容 |
|------|------|
| 人身自由不受侵犯 | 禁止非法拘禁和以其他方法非法剥夺或限制人身自由 |
| 人格尊严不受侵犯 | 禁止用任何方法对公民进行侮辱、诽谤和诬告陷害 |
| 住宅不受侵犯 | 禁止非法搜查或非法侵入住宅 |
| 通信自由和通信秘密 | 除因国家安全或追查刑事犯罪的需要，由公安机关或检察机关依法检查 |

#### 5. 社会经济权利

| 权利 | 内容 |
|------|------|
| 财产权 | 公民的合法的私有财产不受侵犯 |
| 劳动权 | 公民有劳动的权利和义务 |
| 休息权 | 劳动者有休息的权利 |
| 物质帮助权 | 公民在年老、疾病或丧失劳动能力的情况下有获得物质帮助的权利 |

#### 6. 文化教育权利

- 受教育权：公民有受教育的权利和义务
- 科学文化活动的自由

#### 7. 监督权

| 权利 | 内容 |
|------|------|
| 批评权 | 对国家机关和国家工作人员提出批评 |
| 建议权 | 对国家机关和国家工作人员提出建议 |
| 申诉权 | 对行政处分或司法判决不服提出申诉 |
| 控告权 | 对违法失职行为提出控告 |
| 检举权 | 对违法犯罪行为进行检举 |

### 二、公民的基本义务

1. 维护国家统一和全国各民族团结
2. 遵守宪法和法律
3. 维护祖国安全、荣誉和利益
4. 保卫祖国、抵抗侵略
5. 依法纳税
6. 其他义务（劳动、受教育等）

### 三、基本权利的限制

#### 1. 限制的理由

- 维护公共利益
- 维护他人的权利和自由

#### 2. 限制的原则

- 法律保留原则：限制权利必须有法律依据
- 比例原则：限制手段与目的之间要保持适当比例

## 例题与解析

### 例题1

下列关于公民基本权利的说法，正确的是：

A. 公民的政治权利只包括选举权和被选举权
B. 人身自由只包括人身自由不受侵犯
C. 公民在法律面前一律平等
D. 年满16周岁的公民有选举权

**答案**：C

**解析**：A错误，政治权利包括选举权和被选举权以及六大政治自由。B错误，人身自由包括四项内容。D错误，年满18周岁才有选举权。

### 例题2

下列属于社会经济权利的是：

A. 选举权
B. 言论自由
C. 劳动权
D. 受教育权

**答案**：C

**解析**：社会经济权利包括财产权、劳动权、休息权、物质帮助权。选举权属于政治权利，言论自由属于政治自由，受教育权属于文化教育权利。
