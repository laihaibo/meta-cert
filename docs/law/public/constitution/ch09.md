# 第九章 人民法院和人民检察院

## 本章概要

本章阐述人民法院和人民检察院的性质、组织体系、职权和工作原则。

## 高频考点

1. 人民法院的组织体系
2. 人民检察院的职权
3. 司法工作的基本原则

## 重点难点

- **重点**：法院和检察院的职权
- **难点**：上下级法院和检察院的关系不同

## 知识点详解

### 一、人民法院

#### 1. 性质

- 国家审判机关
- 依法独立行使审判权

#### 2. 组织体系

| 层级 | 名称 |
|------|------|
| 最高 | 最高人民法院 |
| 高级 | 高级人民法院 |
| 中级 | 中级人民法院 |
| 基层 | 基层人民法院 |
| 专门 | 军事法院、海事法院等 |

#### 3. 职权

- 审判刑事、民事、行政案件
- 监督地方各级人民法院和专门人民法院的审判工作

#### 4. 上下级关系

- 上下级法院之间是**监督关系**（不是领导关系）
- 上级法院监督下级法院的审判工作

#### 5. 工作原则

- 依法独立行使审判权
- 公民在适用法律上一律平等
- 公开审判
- 被告人有权获得辩护
- 各民族公民有权使用本民族语言文字进行诉讼

### 二、人民检察院

#### 1. 性质

- 国家法律监督机关
- 依法独立行使检察权

#### 2. 组织体系

| 层级 | 名称 |
|------|------|
| 最高 | 最高人民检察院 |
| 省级 | 省级人民检察院 |
| 地级 | 地级市人民检察院 |
| 县级 | 县级人民检察院 |
| 专门 | 军事检察院等 |

#### 3. 职权

- 刑事案件的批准逮捕和提起公诉
- 对诉讼活动实行法律监督
- 对判决执行实行监督

#### 4. 上下级关系

- 上下级检察院之间是**领导关系**（不是监督关系）
- 上级检察院领导下级检察院的工作

#### 5. 工作原则

- 依法独立行使检察权
- 公民在适用法律上一律平等
- 使用本民族语言文字进行诉讼

### 三、法院和检察院的区别

| 比较项目 | 人民法院 | 人民检察院 |
|----------|----------|------------|
| 性质 | 审判机关 | 法律监督机关 |
| 职权 | 审判权 | 检察权 |
| 上下级关系 | 监督关系 | 领导关系 |
| 首长任期 | 无限制 | 无限制 |

## 例题与解析

### 例题1

上下级人民法院之间的关系是：

A. 领导关系
B. 监督关系
C. 指导关系
D. 隶属关系

**答案**：B

**解析**：上下级人民法院之间是监督关系，不是领导关系。

### 例题2

人民检察院的性质是：

A. 审判机关
B. 行政机关
C. 法律监督机关
D. 监察机关

**答案**：C

**解析**：人民检察院是国家法律监督机关。
