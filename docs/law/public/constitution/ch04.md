# 第四章 全国人民代表大会

## 本章概要

本章阐述全国人民代表大会的地位、组成、职权、会议制度和工作方式。

## 高频考点

1. 全国人大的地位和性质
2. 全国人大的职权
3. 全国人大常委会的职权
4. 全国人大代表的权利和义务

## 重点难点

- **重点**：全国人大及其常委会的职权
- **难点**：全国人大与全国人大常委会职权的区分

## 知识点详解

### 一、全国人民代表大会

#### 1. 地位和性质

- 最高国家权力机关
- 最高国家立法机关
- 行使国家立法权

#### 2. 组成

- 由省、自治区、直辖市、特别行政区和军队选出的代表组成
- 代表总名额不超过3000人
- 每届任期5年

#### 3. 职权

| 职权类型 | 具体内容 |
|----------|----------|
| 立法权 | 修改宪法、制定和修改基本法律 |
| 人事权 | 选举国家主席、副主席、中央军委主席、国家监察委主任、最高法院院长、最高检察院检察长等 |
| 决定权 | 审批国民经济和社会发展计划、预算等 |
| 监督权 | 监督宪法的实施，监督国务院、中央军委、国家监察委、最高法院、最高检察院的工作 |

#### 4. 会议制度

- 每年举行一次会议
- 由全国人大常委会召集
- 有2/3以上的代表出席始得举行

### 二、全国人民代表大会常务委员会

#### 1. 地位和性质

- 全国人大的常设机关
- 行使国家立法权
- 对全国人大负责并报告工作

#### 2. 组成

- 由委员长、副委员长、秘书长、委员组成
- 每届任期5年
- 委员长、副委员长连续任职不得超过两届

#### 3. 职权

| 职权类型 | 具体内容 |
|----------|----------|
| 立法权 | 解释宪法、制定和修改非基本法律、解释法律 |
| 人事权 | 根据总理提名决定部长人选等 |
| 决定权 | 决定特赦、全国总动员等 |
| 监督权 | 监督国务院、中央军委、国家监察委、最高法院、最高检察院的工作 |

### 三、全国人民代表大会代表

#### 1. 权利

| 权利 | 内容 |
|------|------|
| 出席会议权 | 出席全国人大会议 |
| 提案权 | 提出议案 |
| 质询权 | 对国务院等提出质询 |
| 人身特别保护权 | 非经全国人大主席团或常委会许可不受逮捕或刑事审判 |
| 言论免责权 | 在人大各种会议上的发言和表决不受法律追究 |

#### 2. 义务

- 模范遵守宪法和法律
- 保守国家秘密
- 协助宪法和法律的实施
- 同原选举单位和人民保持密切联系

## 例题与解析

### 例题1

全国人民代表大会的性质是：

A. 最高国家行政机关
B. 最高国家权力机关
C. 最高国家监察机关
D. 最高国家审判机关

**答案**：B

**解析**：全国人民代表大会是最高国家权力机关。国务院是最高国家行政机关，国家监察委是最高国家监察机关，最高法院是最高国家审判机关。

### 例题2

全国人大常委会委员长连续任职不得超过：

A. 一届
B. 两届
C. 三届
D. 没有限制

**答案**：B

**解析**：全国人大常委会委员长、副委员长连续任职不得超过两届。
