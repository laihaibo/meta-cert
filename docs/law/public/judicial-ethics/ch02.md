# 第二章 检察制度

## 本章概要

本章阐述我国检察制度的基本内容，包括检察院的性质、组织体系、职权和检察工作的基本原则。

## 高频考点

1. 检察院的性质和组织体系
2. 检察院的职权
3. 检察工作的基本原则

## 重点难点

- **重点**：检察院的职权
- **难点**：上下级检察院的关系

## 知识点详解

### 一、人民检察院的性质和组织体系

#### 1. 性质

- 国家法律监督机关
- 依法独立行使检察权

#### 2. 组织体系

| 层级 | 名称 |
|------|------|
| 最高 | 最高人民检察院 |
| 省级 | 省级人民检察院 |
| 地级 | 地级市人民检察院 |
| 县级 | 县级人民检察院 |
| 专门 | 军事检察院等 |

### 二、人民检察院的职权

| 职权 | 内容 |
|------|------|
| 刑事检察 | 批准逮捕、提起公诉 |
| 诉讼监督 | 对刑事、民事、行政诉讼活动的监督 |
| 刑罚执行监督 | 对判决执行的监督 |
| 公益诉讼 | 提起公益诉讼 |

### 三、检察工作的基本原则

#### 1. 依法独立行使检察权

- 人民检察院依法独立行使检察权
- 不受行政机关、社会团体和个人的干涉

#### 2. 公民在法律面前一律平等

#### 3. 使用本民族语言文字进行诉讼

### 四、上下级检察院的关系

- 上下级人民检察院之间是**领导关系**
- 上级人民检察院领导下级人民检察院的工作
- 与法院的监督关系不同

### 五、检察官制度

#### 1. 检察官的任职条件

- 具有中华人民共和国国籍
- 年满23周岁
- 拥护中华人民共和国宪法
- 有良好的政治、业务素质和良好的品行
- 身体健康
- 具备相应的学历和法律职业资格

#### 2. 检察官的等级

- 首席大检察官
- 大检察官
- 高级检察官
- 检察官

## 例题与解析

### 例题1

上下级人民检察院之间的关系是：

A. 领导关系
B. 监督关系
C. 指导关系
D. 隶属关系

**答案**：A

**解析**：上下级人民检察院之间是领导关系，与法院的监督关系不同。

### 例题2

人民检察院的性质是：

A. 审判机关
B. 行政机关
C. 法律监督机关
D. 监察机关

**答案**：C

**解析**：人民检察院是国家法律监督机关。
