# 第一章 审判制度

## 本章概要

本章阐述我国审判制度的基本内容，包括法院的性质、组织体系、职权和审判工作的基本原则。

## 高频考点

1. 法院的性质和组织体系
2. 法院的职权
3. 审判工作的基本原则
4. 审级制度

## 重点难点

- **重点**：法院的职权、审判工作的基本原则
- **难点**：上下级法院的关系

## 知识点详解

### 一、人民法院的性质和组织体系

#### 1. 性质

- 国家审判机关
- 依法独立行使审判权

#### 2. 组织体系

| 层级 | 名称 |
|------|------|
| 最高 | 最高人民法院 |
| 高级 | 高级人民法院 |
| 中级 | 中级人民法院 |
| 基层 | 基层人民法院 |
| 专门 | 军事法院、海事法院、知识产权法院等 |

### 二、人民法院的职权

| 法院 | 职权 |
|------|------|
| 最高人民法院 | 监督地方各级法院和专门法院的审判工作；审判法律规定由其管辖的案件 |
| 高级人民法院 | 审理法律规定由其管辖的案件 |
| 中级人民法院 | 审理法律规定由其管辖的案件 |
| 基层人民法院 | 审理第一审案件 |

### 三、审判工作的基本原则

#### 1. 依法独立行使审判权

- 人民法院依法独立行使审判权
- 不受行政机关、社会团体和个人的干涉

#### 2. 公民在法律面前一律平等

- 对一切公民不分民族、种族、性别、职业等，在适用法律上一律平等

#### 3. 公开审判

- 除法律规定的特殊情况外，案件一律公开审理
- 不公开审理的情形：涉及国家秘密、个人隐私、未成年人犯罪等

#### 4. 被告人有权获得辩护

- 被告人有权获得辩护
- 人民法院有义务保证被告人获得辩护

#### 5. 使用本民族语言文字进行诉讼

- 各民族公民都有使用本民族语言文字进行诉讼的权利

### 四、审级制度

#### 1. 两审终审制

- 一个案件经过两级人民法院审判即告终结
- 第二审人民法院的判决、裁定是终审的判决、裁定

#### 2. 第一审程序

- 基层人民法院管辖第一审案件
- 中级人民法院管辖重大涉外案件等

#### 3. 第二审程序

- 当事人不服一审判决可以提起上诉
- 上诉期限：判决15日，裁定10日

### 五、审判监督程序

#### 1. 再审的提起

| 主体 | 条件 |
|------|------|
| 各级法院院长 | 提交审判委员会讨论决定 |
| 最高人民法院 | 对各级法院的生效判决 |
| 上级人民法院 | 对下级法院的生效判决 |
| 当事人 | 申请再审 |

#### 2. 检察监督

- 最高人民检察院对各级法院的生效判决提出抗诉
- 上级检察院对下级法院的生效判决提出抗诉

## 例题与解析

### 例题1

上下级人民法院之间的关系是：

A. 领导关系
B. 监督关系
C. 指导关系
D. 隶属关系

**答案**：B

**解析**：上下级人民法院之间是监督关系，不是领导关系。

### 例题2

我国实行的审级制度是：

A. 一审终审
B. 两审终审
C. 三审终审
D. 四审终审

**答案**：B

**解析**：我国实行两审终审制。
