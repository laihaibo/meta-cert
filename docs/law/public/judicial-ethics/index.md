# 司法制度和法律职业道德

## 科目概述

司法制度和法律职业道德是法考公共科目的重要组成部分，主要考查我国的审判制度、检察制度、律师制度、公证制度以及各类法律职业人员的职业道德。

## 考情分析

- 分值占比：约10-15分
- 题型：选择题为主
- 难度：中等，需要记忆具体规定
