# 第三章 律师制度

## 本章概要

本章阐述我国律师制度的基本内容，包括律师执业条件、权利义务、律师事务所和律师协会。

## 高频考点

1. 律师执业条件
2. 律师的权利和义务
3. 律师事务所的组织形式
4. 律师的法律责任

## 重点难点

- **重点**：律师执业条件、律师权利义务
- **难点**：律师执业的限制

## 知识点详解

### 一、律师执业条件

#### 1. 一般条件

- 拥护中华人民共和国宪法
- 通过国家统一法律职业资格考试
- 在律师事务所实习满1年
- 品行良好

#### 2. 不得从事律师职业的人员

- 无民事行为能力或者限制民事行为能力的
- 受过刑事处罚的（过失犯罪除外）
- 被开除公职或者被吊销律师执业证书的

### 二、律师的权利

| 权利 | 内容 |
|------|------|
| 阅卷权 | 查阅、摘抄、复制案卷材料 |
| 会见权 | 会见犯罪嫌疑人、被告人 |
| 调查取证权 | 向有关单位或个人调查取证 |
| 辩护权 | 为犯罪嫌疑人、被告人辩护 |
| 人身保障权 | 律师执业活动中的人身权利不受侵犯 |

### 三、律师的义务

| 义务 | 内容 |
|------|------|
| 遵守法律法规 | 遵守宪法和法律 |
| 恪守职业道德 | 遵守律师职业道德和执业纪律 |
| 保守秘密 | 保守在执业活动中知悉的国家秘密、商业秘密和当事人隐私 |
| 不得双方代理 | 不得在同一案件中为双方当事人代理 |
| 不得私自收费 | 不得私自接受委托、收取费用 |

### 四、律师事务所

#### 1. 组织形式

| 形式 | 特点 |
|------|------|
| 合伙律师事务所 | 3名以上合伙人 |
| 个人律师事务所 | 设立人1人 |
| 国资律师事务所 | 国家出资设立 |

#### 2. 律师事务所的责任

- 合伙律师事务所的合伙人承担无限连带责任
- 个人律师事务所的设立人承担无限责任

### 五、律师协会

#### 1. 性质

- 律师的自律性组织
- 社会团体法人

#### 2. 职责

- 保障律师依法执业
- 维护律师的合法权益
- 制定行业规范和惩戒规则
- 组织律师业务培训

### 六、法律援助

#### 1. 对象

- 经济困难的公民
- 特殊案件的当事人（如可能被判处无期徒刑、死刑的）

#### 2. 范围

- 刑事案件
- 请求国家赔偿
- 请求给予社会保险待遇或最低生活保障待遇
- 请求发给抚恤金、救济金
- 请求给付赡养费、抚养费、扶养费

## 例题与解析

### 例题1

律师执业的实习期限是：

A. 6个月
B. 1年
C. 2年
D. 3年

**答案**：B

**解析**：律师需要在律师事务所实习满1年才能申请执业。

### 例题2

律师事务所的组织形式不包括：

A. 合伙律师事务所
B. 个人律师事务所
C. 国资律师事务所
D. 股份制律师事务所

**答案**：D

**解析**：律师事务所的组织形式包括合伙律师事务所、个人律师事务所和国资律师事务所。
