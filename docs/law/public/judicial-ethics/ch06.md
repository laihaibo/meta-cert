# 第六章 检察官职业道德和律师职业道德

## 本章概要

本章阐述检察官职业道德和律师职业道德的基本要求。

## 高频考点

1. 检察官职业道德的核心内容
2. 律师职业道德的核心内容
3. 律师执业的禁止行为

## 重点难点

- **重点**：检察官和律师的职业道德要求
- **难点**：律师与法官、检察官职业道德的区别

## 知识点详解

### 一、检察官职业道德

#### 1. 核心内容

| 内容 | 要求 |
|------|------|
| 忠诚 | 忠于党、忠于国家、忠于人民、忠于宪法和法律 |
| 公正 | 依法独立行使检察权，坚持以事实为根据、以法律为准绳 |
| 清廉 | 遵守廉洁从检的规定 |
| 文明 | 文明执法，尊重和保障人权 |

#### 2. 禁止行为

- 私自会见当事人
- 收受贿赂
- 泄露案件秘密
- 从事营利性活动

### 二、律师职业道德

#### 1. 核心内容

| 内容 | 要求 |
|------|------|
| 忠于宪法和法律 | 遵守宪法和法律 |
| 诚实守信 | 对当事人诚实守信 |
| 勤勉尽责 | 尽职尽责维护当事人合法权益 |
| 保守秘密 | 保守在执业活动中知悉的秘密 |
| 公平竞争 | 律师之间公平竞争 |

#### 2. 律师与当事人的关系

- 律师应当忠于当事人的合法权益
- 律师不得损害当事人的利益
- 律师应当勤勉尽责

#### 3. 律师与法官的关系

- 律师应当尊重法官
- 律师不得向法官行贿
- 律师不得发表不当言论影响司法公正

#### 4. 律师与对方当事人的关系

- 律师不得与对方当事人恶意串通
- 律师不得损害对方当事人的合法权益

### 三、律师执业的禁止行为

| 禁止行为 | 内容 |
|----------|------|
| 私自接受委托 | 不得私自接受委托、收取费用 |
| 双方代理 | 不得在同一案件中为双方当事人代理 |
| 收受贿赂 | 不得接受对方当事人的财物 |
| 提供虚假证据 | 不得故意提供虚假证据 |
| 煽动闹事 | 不得煽动、教唆当事人采取非法手段 |
| 违反法庭纪律 | 不得扰乱法庭秩序 |

### 四、律师的法律责任

#### 1. 行政责任

- 警告
- 罚款
- 停止执业
- 吊销执业证书

#### 2. 民事责任

- 律师因过错给当事人造成损失的，应当承担赔偿责任

#### 3. 刑事责任

- 律师构成犯罪的，依法追究刑事责任

### 五、三大法律职业的道德要求对比

| 要求 | 法官 | 检察官 | 律师 |
|------|------|--------|------|
| 核心 | 公正 | 公正 | 维护当事人权益 |
| 独立性 | 依法独立审判 | 依法独立检察 | 独立执业 |
| 保密义务 | 审判秘密 | 案件秘密 | 当事人秘密 |
| 廉洁要求 | 不得收受贿赂 | 不得收受贿赂 | 不得收受贿赂 |

## 例题与解析

### 例题1

检察官职业道德的核心内容不包括：

A. 忠诚
B. 公正
C. 效率
D. 清廉

**答案**：C

**解析**：检察官职业道德的核心内容包括忠诚、公正、清廉、文明。效率不是核心内容。

### 例题2

律师执业的禁止行为不包括：

A. 私自接受委托
B. 双方代理
C. 依法辩护
D. 收受贿赂

**答案**：C

**解析**：依法辩护是律师的权利和职责，不是禁止行为。
