# 第四章 公证制度

## 本章概要

本章阐述我国公证制度的基本内容，包括公证机构的性质、业务范围、公证效力和公证程序。

## 高频考点

1. 公证机构的性质
2. 公证业务范围
3. 公证的效力

## 重点难点

- **重点**：公证业务范围、公证效力
- **难点**：公证与鉴证的区别

## 知识点详解

### 一、公证机构的性质

- 公证机构是依法设立的证明机构
- 不以营利为目的
- 依法独立行使公证职能

### 二、公证业务范围

| 业务 | 内容 |
|------|------|
| 证明合同 | 证明各类合同、协议 |
| 证明继承 | 证明继承权、遗嘱 |
| 证明身份 | 证明出生、婚姻、亲属关系 |
| 证明事实 | 证明文书上的签名、印鉴属实 |
| 保全证据 | 对可能灭失的证据进行保全 |
| 提存 | 对债务人的提存进行公证 |

### 三、公证的效力

#### 1. 证据效力

- 经公证的法律行为、法律事实和文书，应当作为认定事实的根据
- 但有相反证据足以推翻公证的除外

#### 2. 强制执行效力

- 对经公证的以给付为内容并载明债务人愿意接受强制执行承诺的债权文书
- 债务人不履行的，债权人可以依法申请人民法院强制执行

#### 3. 法律行为成立要件效力

- 法律、行政法规规定必须公证的事项，未经公证不发生效力

### 四、公证程序

#### 1. 申请

- 自然人、法人或者其他组织可以向住所地、经常居住地、行为地或者事实发生地的公证机构申请

#### 2. 受理

- 公证机构受理申请后进行审查

#### 3. 审查

- 审查当事人的人数、身份、资格
- 审查当事人的意思表示和相应的权利
- 审查需公证的行为、事实或文书的内容

#### 4. 出具公证书

- 经审查符合条件的，出具公证书

### 五、公证员

#### 1. 任职条件

- 具有中华人民共和国国籍
- 年龄25周岁以上65周岁以下
- 通过国家统一法律职业资格考试
- 在公证机构实习2年以上

#### 2. 禁止行为

- 同时在两个以上公证机构执业
- 从事有报酬的其他职业
- 为本人及近亲属办理公证
- 私自出具公证书

## 例题与解析

### 例题1

公证机构的性质是：

A. 行政机关
B. 司法机关
C. 证明机构
D. 营利性企业

**答案**：C

**解析**：公证机构是依法设立的证明机构，不以营利为目的。

### 例题2

公证的效力不包括：

A. 证据效力
B. 强制执行效力
C. 立法效力
D. 法律行为成立要件效力

**答案**：C

**解析**：公证的效力包括证据效力、强制执行效力和法律行为成立要件效力。不包括立法效力。
