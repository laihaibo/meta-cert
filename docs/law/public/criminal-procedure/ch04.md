# 第四章 辩护与代理

## 本章概要

本章介绍刑事诉讼中的辩护与代理制度，包括辩护人的范围、权利和义务。重点掌握辩护人的范围和辩护权。

## 高频考点

> 🔥 高频考点

### 考点1: 辩护人的范围
- 律师
- 人民团体或者犯罪嫌疑人、被告人所在单位推荐的人
- 犯罪嫌疑人、被告人的监护人、亲友

### 考点2: 辩护人的权利
- 会见权
- 阅卷权
- 调查取证权
- 辩论权

### 考点3: 指定辩护
- 法律援助辩护的情形

## 知识点详解

### 4.1 辩护人的范围

**可以担任辩护人的人**：
- 律师
- 人民团体或者犯罪嫌疑人、被告人所在单位推荐的人
- 犯罪嫌疑人、被告人的监护人、亲友

**不得担任辩护人的人**：
- 正在被执行刑罚或者依法被剥夺、限制人身自由的人
- 被开除公职和被吊销律师、公证员执业证书的人
- 无行为能力或者限制行为能力的人

### 4.2 辩护人的权利

| 权利 | 内容 |
|------|------|
| 会见权 | 辩护律师可以同在押的犯罪嫌疑人、被告人会见和通信 |
| 阅卷权 | 辩护律师自人民检察院对案件审查起诉之日起，可以查阅、摘抄、复制本案的案卷材料 |
| 调查取证权 | 辩护律师经证人或者其他有关单位和个人同意，可以向他们收集与本案有关的材料 |
| 辩论权 | 辩护人在法庭上可以对证据和案件情况发表意见、进行辩论 |

### 4.3 指定辩护

**应当指定辩护的情形**：
- 犯罪嫌疑人、被告人是盲、聋、哑人
- 犯罪嫌疑人、被告人是尚未完全丧失辨认或者控制自己行为能力的精神病人
- 可能被判处无期徒刑、死刑
- 审判时不满18周岁的未成年人

## 例题与解析

### 例题1

**题目**: 下列人员中，不得担任辩护人的是：

**选项**:
A. 律师
B. 被告人的亲友
C. 正在被执行刑罚的人
D. 被告人所在单位推荐的人

**答案**: C

**解析**: 正在被执行刑罚或者依法被剥夺、限制人身自由的人，不得担任辩护人。
