# 第六章 证据

## 本章概要

本章介绍刑事诉讼中的证据制度，包括证据的种类、证明标准和非法证据排除。重点掌握证据的种类和非法证据排除规则。

## 高频考点

> 🔥 高频考点

### 考点1: 证据的种类
- 物证
- 书证
- 证人证言
- 被害人陈述
- 犯罪嫌疑人、被告人供述和辩解
- 鉴定意见
- 勘验、检查、辨认、侦查实验等笔录
- 视听资料、电子数据

### 考点2: 证明标准
- 事实清楚，证据确实、充分

### 考点3: 非法证据排除
- 排除非法言词证据
- 排除非法实物证据

## 知识点详解

### 6.1 证据的种类

| 种类 | 说明 |
|------|------|
| 物证 | 以其外部特征、物质属性、存在状况等证明案件事实的实物 |
| 书证 | 以其记载的内容和表达的思想证明案件事实的书面材料 |
| 证人证言 | 证人就其所知道的案件情况向司法机关所作的陈述 |
| 被害人陈述 | 被害人就其所知道的案件情况向司法机关所作的陈述 |
| 犯罪嫌疑人、被告人供述和辩解 | 犯罪嫌疑人、被告人就案件情况向司法机关所作的陈述 |
| 鉴定意见 | 鉴定人对案件中的专门性问题进行鉴定后所作的书面意见 |
| 勘验、检查、辨认、侦查实验等笔录 | 司法人员对与案件有关的场所、物品、人身等进行勘验、检查后所作的记录 |
| 视听资料、电子数据 | 以录音、录像、电子计算机等设备储存的信息证明案件事实的资料 |

### 6.2 证明标准

**证明标准**：
- 事实清楚，证据确实、充分

**证据确实、充分的条件**：
- 定罪量刑的事实都有证据证明
- 据以定案的证据均经法定程序查证属实
- 综合全案证据，对所认定事实已排除合理怀疑

### 6.3 非法证据排除

**排除非法言词证据**：
- 采用刑讯逼供等非法方法收集的犯罪嫌疑人、被告人供述
- 采用暴力、威胁等非法方法收集的证人证言、被害人陈述

**排除非法实物证据**：
- 收集物证、书证不符合法定程序，可能严重影响司法公正的，应当予以补正或者作出合理解释
- 不能补正或者作出合理解释的，对该证据应当予以排除

## 例题与解析

### 例题1

**题目**: 下列属于应当排除的非法证据是：

**选项**:
A. 通过合法搜查取得的物证
B. 通过刑讯逼供取得的口供
C. 通过合法询问取得的证人证言
D. 通过合法鉴定取得的鉴定意见

**答案**: B

**解析**: 采用刑讯逼供等非法方法收集的犯罪嫌疑人、被告人供述，应当予以排除。
