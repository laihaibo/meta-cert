# 第十一章 执行

## 本章概要

本章介绍刑事诉讼中的执行制度，包括各种刑罚的执行。

## 高频考点

> 🔥 高频考点

### 考点1: 各种刑罚的执行
- 死刑的执行
- 无期徒刑、有期徒刑的执行
- 拘役的执行
- 管制的执行
- 缓刑的执行
- 罚金的执行
- 没收财产的执行

### 考点2: 减刑、假释
- 减刑的条件和程序
- 假释的条件和程序

## 知识点详解

### 11.1 各种刑罚的执行

| 刑罚 | 执行机关 |
|------|---------|
| 死刑 | 人民法院 |
| 无期徒刑、有期徒刑 | 监狱 |
| 拘役 | 公安机关 |
| 管制 | 社区矫正机构 |
| 缓刑 | 社区矫正机构 |
| 罚金 | 人民法院 |
| 没收财产 | 人民法院 |

### 11.2 减刑、假释

**减刑的条件**：
- 认真遵守监规
- 接受教育改造
- 确有悔改表现或者立功表现

**假释的条件**：
- 被判处有期徒刑的犯罪分子，执行原判刑期1/2以上
- 被判处无期徒刑的犯罪分子，实际执行13年以上
- 认真遵守监规，接受教育改造，确有悔改表现，没有再犯罪的危险

## 例题与解析

### 例题1

**题目**: 下列关于刑罚执行的说法，正确的是：

**选项**:
A. 死刑由公安机关执行
B. 有期徒刑由人民法院执行
C. 管制由社区矫正机构执行
D. 拘役由监狱执行

**答案**: C

**解析**: 管制由社区矫正机构执行。
