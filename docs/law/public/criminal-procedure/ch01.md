# 第一章 刑事诉讼法概述

## 本章概要

本章介绍刑事诉讼法的基本概念和基本原则。

## 高频考点

> 🔥 高频考点

### 考点1: 刑事诉讼法的基本原则
- 侦查权、检察权、审判权由专门机关行使
- 人民法院、人民检察院依法独立行使职权
- 未经人民法院依法判决不得确定有罪
- 保障诉讼参与人的诉讼权利

### 考点2: 刑事诉讼中的专门机关
- 公安机关
- 人民检察院
- 人民法院

## 知识点详解

### 1.1 刑事诉讼法的基本原则

| 原则 | 内容 |
|------|------|
| 专门机关行使职权 | 侦查权、检察权、审判权由专门机关依法行使 |
| 独立行使职权 | 人民法院、人民检察院依法独立行使审判权、检察权 |
| 未经判决不得定罪 | 未经人民法院依法判决，对任何人都不得确定有罪 |
| 保障诉讼权利 | 人民法院、人民检察院和公安机关应当保障诉讼参与人依法享有的诉讼权利 |

### 1.2 刑事诉讼中的专门机关

| 机关 | 职能 |
|------|------|
| 公安机关 | 侦查、拘留、执行逮捕、预审 |
| 人民检察院 | 检察、批准逮捕、检察机关直接受理案件的侦查、提起公诉 |
| 人民法院 | 审判 |

## 例题与解析

### 例题1

**题目**: 下列哪项不属于刑事诉讼法的基本原则？

**选项**:
A. 侦查权由专门机关行使
B. 未经判决不得定罪
C. 罪刑法定原则
D. 保障诉讼权利

**答案**: C

**解析**: 罪刑法定原则是刑法的基本原则，不是刑事诉讼法的基本原则。
