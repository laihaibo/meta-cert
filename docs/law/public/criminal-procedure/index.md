# 刑事诉讼法

## 学科概述

刑事诉讼法是法考的重要科目，主要规范刑事诉讼活动。本学科内容涵盖刑事诉讼的基本原则、管辖、回避、辩护与代理、强制措施、证据、立案侦查起诉、审判程序等。
