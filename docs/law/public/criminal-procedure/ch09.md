# 第九章 起诉

## 本章概要

本章介绍刑事诉讼中的起诉制度，包括审查起诉和不起诉。重点掌握审查起诉的程序和不起诉的种类。

- 法定不起诉

## 知识点详解

### 9.1 审查起诉

**审查起诉的内容**：
- 犯罪事实、情节是否清楚，证据是否确实、充分
- 犯罪性质和罪名的认定是否正确
- 有无遗漏罪行和其他应当追究刑事责任的人
- 是否属于不应追究刑事责任的
- 有无附带民事诉讼
- 侦查活动是否合法

**审查起诉的期限**：
- 人民检察院对于公安机关移送起诉的案件，应当在1个月以内作出决定
- 重大、复杂的案件，可以延长15日

### 9.2 不起诉

| 类型 | 条件 |
|------|------|
| 法定不起诉 | 犯罪嫌疑人没有犯罪事实，或者有依法不追究刑事责任的情形 |
| 酌定不起诉 | 犯罪情节轻微，依照刑法规定不需要判处刑罚或者免除刑罚 |
| 存疑不起诉 | 对于二次补充侦查的案件，人民检察院仍然认为证据不足，不符合起诉条件 |

## 例题与解析

### 例题1

**题目**: 下列关于不起诉的说法，正确的是：

**选项**:
A. 法定不起诉适用于犯罪情节轻微的情形
B. 酌定不起诉适用于没有犯罪事实的情形
C. 存疑不起诉适用于证据不足的情形
D. 所有不起诉决定都需要经过上级批准

**答案**: C

**解析**: 存疑不起诉适用于二次补充侦查后证据不足的情形。
