# 第十一章 行政诉讼程序

## 本章概要

本章介绍行政诉讼的程序，包括起诉、受理、审理和判决。重点掌握起诉条件和判决类型。

## 高频考点

> 🔥 高频考点

### 考点1: 起诉条件
- 原告是符合规定的公民、法人或者其他组织
- 有明确的被告
- 有具体的诉讼请求和事实根据
- 属于人民法院受案范围和受诉人民法院管辖

### 考点2: 起诉期限
- 直接起诉：自知道或者应当知道作出行政行为之日起6个月内
- 经复议起诉：收到复议决定书之日起15日内

### 考点3: 判决类型
- 驳回诉讼请求判决
- 撤销判决
- 履行判决
- 给付判决
- 确认违法判决
- 确认无效判决
- 变更判决

## 知识点详解

### 11.1 起诉

**起诉条件**：
- 原告是符合规定的公民、法人或者其他组织
- 有明确的被告
- 有具体的诉讼请求和事实根据
- 属于人民法院受案范围和受诉人民法院管辖

**起诉期限**：

| 情形 | 期限 |
|------|------|
| 直接起诉 | 自知道或者应当知道作出行政行为之日起6个月内 |
| 经复议起诉 | 收到复议决定书之日起15日内 |
| 不动产案件 | 自行政行为作出之日起20年内 |
| 其他案件 | 自行政行为作出之日起5年内 |

### 11.2 受理

- 人民法院在接到起诉状时对符合规定的起诉条件的，应当登记立案
- 当场不能判定是否符合规定的起诉条件的，应当接收起诉状，出具注明收到日期的书面凭证，并在7日内决定是否立案

### 11.3 审理

**审理方式**：
- 人民法院公开审理行政案件
- 涉及国家秘密、个人隐私和法律另有规定的除外

**审理期限**：
- 人民法院应当在立案之日起6个月内作出第一审判决
- 有特殊情况需要延长的，由高级人民法院批准

### 11.4 判决

| 判决类型 | 适用情形 |
|---------|---------|
| 驳回诉讼请求 | 行政行为证据确凿，适用法律、法规正确，符合法定程序 |
| 撤销判决 | 主要证据不足；适用法律、法规错误；违反法定程序；超越职权；滥用职权；明显不当 |
| 履行判决 | 被告不履行或者拖延履行法定职责 |
| 给付判决 | 被告依法负有给付义务 |
| 确认违法判决 | 行政行为违法，但不具有可撤销内容 |
| 确认无效判决 | 行政行为实施主体不具有行政主体资格或者没有依据等重大且明显违法情形 |
| 变更判决 | 行政处罚明显不当，或者其他行政行为涉及对款额的确定、认定确有错误 |

## 例题与解析

### 例题1

**题目**: 甲对县公安局作出的行政拘留决定不服，直接向人民法院提起行政诉讼的期限是：

**选项**:
A. 1个月
B. 3个月
C. 6个月
D. 1年

**答案**: C

**解析**: 直接起诉的，自知道或者应当知道作出行政行为之日起6个月内提出。
