# 第四章 行政处罚

## 本章概要

本章介绍行政处罚的基本制度，包括行政处罚的种类、设定、实施程序和听证程序。重点掌握行政处罚的种类、简易程序和听证程序。

## 重点难点

> ⚠️ 难点

### 难点1: 行政处罚与行政强制的区别
- 行政处罚是对违法行为的制裁
- 行政强制是为实现行政目的而采取的强制措施

## 知识点详解

### 4.1 行政处罚的概念

行政处罚是行政机关依法对违反行政管理秩序的公民、法人或者其他组织给予制裁的行为。

### 4.2 行政处罚的种类

| 种类 | 说明 |
|------|------|
| 警告、通报批评 | 声誉罚 |
| 罚款、没收违法所得、没收非法财物 | 财产罚 |
| 暂扣许可证件、降低资质等级、吊销许可证件 | 行为罚 |
| 限制开展生产经营活动、责令停产停业、责令关闭、限制从业 | 行为罚 |
| 行政拘留 | 人身罚 |

### 4.3 行政处罚的设定

| 规范类型 | 设定权限 |
|---------|---------|
| 法律 | 可以设定各种行政处罚 |
| 行政法规 | 可以设定除限制人身自由以外的行政处罚 |
| 地方性法规 | 可以设定除限制人身自由、吊销营业执照以外的行政处罚 |
| 规章 | 可以设定警告、通报批评和一定数额的罚款 |

### 4.4 行政处罚的程序

#### 4.4.1 简易程序

**适用条件**：
- 违法事实确凿并有法定依据
- 对公民处以200元以下、对法人或者其他组织处以3000元以下罚款或者警告

**程序要求**：
- 表明身份
- 告知当事人违法行为的事实、理由和依据
- 听取当事人的陈述和申辩
- 填写行政处罚决定书并当场交付

#### 4.4.2 普通程序

**程序要求**：
- 立案
- 调查取证
- 告知当事人
- 听取当事人陈述和申辩
- 作出处罚决定
- 送达处罚决定书

#### 4.4.3 听证程序

**适用情形**：
- 较大数额罚款
- 没收较大数额违法所得、没收较大价值非法财物
- 降低资质等级、吊销许可证件
- 责令停产停业、责令关闭、限制从业
- 其他较重的行政处罚

**程序要求**：
- 当事人要求听证的，应当在行政机关告知后5日内提出
- 行政机关应当在举行听证的7日前，通知当事人及有关人员听证的时间、地点
- 听证公开举行
- 当事人可以亲自参加听证，也可以委托1至2人代理

### 4.5 行政处罚的执行

**罚款的执行**：
- 作出罚款决定的行政机关应当与收缴罚款的机构分离
- 当事人应当自收到行政处罚决定书之日起15日内，到指定的银行或者通过电子支付系统缴纳罚款

**行政拘留的执行**：
- 由作出决定的公安机关送达拘留所执行

## 例题与解析

### 例题1

**题目**: 下列关于行政处罚设定权的说法，正确的是：

**选项**:
A. 法律可以设定各种行政处罚
B. 行政法规可以设定所有行政处罚
C. 地方性法规可以设定限制人身自由的行政处罚
D. 规章可以设定罚款和没收违法所得

**答案**: A

**解析**: 法律可以设定各种行政处罚，行政法规可以设定除限制人身自由以外的行政处罚，地方性法规可以设定除限制人身自由、吊销营业执照以外的行政处罚。

### 例题2

**题目**: 行政机关对甲作出较大数额罚款的处罚决定前，甲享有的权利不包括：

**选项**:
A. 陈述权
B. 申辩权
C. 要求听证的权利
D. 拒绝处罚的权利

**答案**: D

**解析**: 当事人享有陈述权、申辩权和要求听证的权利，但不享有拒绝处罚的权利。
