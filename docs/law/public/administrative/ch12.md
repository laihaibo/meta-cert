## 知识点详解

### 12.1 国家赔偿概述

**国家赔偿的构成要件**：
- 行为主体是国家机关或者国家机关工作人员
- 行使职权的行为违法
- 造成公民、法人或者其他组织的合法权益损害
- 违法行为与损害之间有因果关系

### 12.2 行政赔偿

**行政赔偿的范围**：

| 侵犯人身权 | 侵犯财产权 |
|-----------|-----------|
| 违法拘留或者违法采取限制公民人身自由的行政强制措施 | 违法实施罚款、吊销许可证和执照、责令停产停业、没收财物等行政处罚 |
| 非法拘禁或者以其他方法非法剥夺公民人身自由 | 违法对财产采取查封、扣押、冻结等行政强制措施 |
| 以殴打、虐待等行为或者唆使、放纵他人以殴打、虐待等行为造成公民身体伤害或者死亡 | 违法征收、征用财产 |
| 造成公民身体伤害或者死亡的其他违法行为 | 造成财产损害的其他违法行为 |

### 12.3 司法赔偿

**刑事赔偿的范围**：

| 侵犯人身权 | 侵犯财产权 |
|-----------|-----------|
| 违反刑事诉讼法的规定对公民采取拘留措施 | 违法对财产采取查封、扣押、冻结、追缴等措施 |
| 依照审判监督程序再审改判无罪，原判刑罚已经执行 | 再审改判无罪，原判罚金、没收财产已经执行 |
| 刑讯逼供或者以殴打、虐待等行为或者唆使、放纵他人以殴打、虐待等行为造成公民身体伤害或者死亡 | |
| 依照审判监督程序再审改判无罪，原判剥夺政治权利已经执行 | |

### 12.4 国家赔偿的方式和标准

**赔偿方式**：
- 以金钱赔偿为主要方式
- 能够返还财产或者恢复原状的，予以返还财产或者恢复原状

**人身自由损害的赔偿标准**：
- 每日赔偿金按照国家上年度职工日平均工资计算

**生命健康损害的赔偿标准**：
- 造成身体伤害的，应当支付医疗费、护理费，以及赔偿因误工减少的收入
- 造成部分或者全部丧失劳动能力的，应当支付医疗费、护理费、残疾生活辅助具费、康复费等因残疾而增加的必要支出和继续治疗所必需的费用，以及残疾赔偿金
- 造成死亡的，应当支付死亡赔偿金、丧葬费，总额为国家上年度职工年平均工资的20倍

## 例题与解析

### 例题1

**题目**: 下列属于国家赔偿范围的是：

**选项**:
A. 行政机关工作人员的个人行为
B. 行政机关合法行使职权造成的损失
C. 行政机关违法行使职权造成的损失
D. 不可抗力造成的损失

**答案**: C

**解析**: 国家赔偿以违法行为为前提，行政机关违法行使职权造成的损失属于国家赔偿范围。
