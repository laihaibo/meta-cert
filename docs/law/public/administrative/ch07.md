# 第七章 行政复议

## 本章概要

本章介绍行政复议的基本制度，包括行政复议的范围、申请、审理和决定。重点掌握行政复议的受案范围和复议机关。

## 高频考点

> 🔥 高频考点

### 考点1: 行政复议的受案范围
- 对行政处罚不服
- 对行政强制措施不服
- 对行政许可不服
- 对行政确认不服

### 考点2: 行政复议的申请
- 申请期限：60日
- 申请方式：书面或口头

### 考点3: 行政复议的决定
- 维持决定
- 撤销决定
- 变更决定
- 确认违法决定
- 责令履行决定

## 知识点详解

### 7.1 行政复议的概念

行政复议是公民、法人或者其他组织认为行政机关的具体行政行为侵犯其合法权益，依法向行政复议机关提出申请，由受理申请的行政机关对具体行政行为进行审查并作出决定的活动。

### 7.2 行政复议的受案范围

**可以申请行政复议的情形**：
- 对行政机关作出的行政处罚决定不服
- 对行政机关作出的行政强制措施决定不服
- 对行政机关作出的行政许可决定不服
- 对行政机关作出的行政确认决定不服
- 认为行政机关侵犯合法的经营自主权
- 认为行政机关变更或者废止农业承包合同，侵犯其合法权益

### 7.3 行政复议的申请

**申请期限**：
- 公民、法人或者其他组织认为具体行政行为侵犯其合法权益的，可以自知道该具体行政行为之日起60日内提出行政复议申请

**申请方式**：
- 书面申请
- 口头申请

### 7.4 行政复议机关

| 被申请人 | 复议机关 |
|---------|---------|
| 县级以上地方人民政府 | 上一级人民政府 |
| 县级以上地方人民政府工作部门 | 本级人民政府或上一级主管部门 |
| 省、自治区人民政府依法设立的派出机关 | 设立该派出机关的人民政府 |

### 7.5 行政复议的审理

**审理方式**：
- 行政复议原则上采取书面审查的办法
- 申请人提出要求或者行政复议机关负责法制工作的机构认为有必要时，可以向有关组织和人员调查情况

**审理期限**：
- 行政复议机关应当自受理申请之日起60日内作出行政复议决定
- 情况复杂，不能在规定期限内作出行政复议决定的，经行政复议机关的负责人批准，可以适当延长，但延长期限最多不超过30日

### 7.6 行政复议的决定

| 决定类型 | 适用情形 |
|---------|---------|
| 维持决定 | 具体行政行为认定事实清楚，证据确凿，适用依据正确，程序合法，内容适当 |
| 撤销决定 | 主要事实不清、证据不足；适用依据错误；违反法定程序；超越或者滥用职权；具体行政行为明显不当 |
| 变更决定 | 认定事实清楚，证据确凿，程序合法，但明显不当或者适用依据错误 |
| 确认违法决定 | 具体行政行为违法，但不具有可撤销内容 |
| 责令履行决定 | 被申请人不履行法定职责 |

## 例题与解析

### 例题1

**题目**: 甲对县公安局作出的行政拘留处罚决定不服，应当向哪个机关申请行政复议？

**选项**:
A. 县公安局
B. 县人民政府
C. 市公安局
D. 县人民政府或市公安局

**答案**: D

**解析**: 对县级以上地方各级人民政府工作部门的具体行政行为不服的，由申请人选择，可以向该部门的本级人民政府申请行政复议，也可以向上一级主管部门申请行政复议。
