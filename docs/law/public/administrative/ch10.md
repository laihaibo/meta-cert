# 第十章 行政诉讼参加人

## 本章概要

本章介绍行政诉讼的参加人，包括原告、被告和第三人。重点掌握原告和被告的确定。

## 高频考点

> 🔥 高频考点

### 考点1: 原告的确定
- 行政行为的相对人
- 与行政行为有利害关系的公民、法人或者其他组织

### 考点2: 被告的确定
- 作出行政行为的行政机关
- 经复议的案件
- 委托的行政机关

### 考点3: 第三人
- 与被诉行政行为有利害关系的公民、法人或者其他组织

## 知识点详解

### 10.1 原告

**原告的确定**：
- 行政行为的相对人以及其他与行政行为有利害关系的公民、法人或者其他组织，有权提起诉讼
- 有权提起诉讼的公民死亡，其近亲属可以提起诉讼
- 有权提起诉讼的法人或者其他组织终止，承受其权利的法人或者其他组织可以提起诉讼

### 10.2 被告

**被告的确定**：

| 情形 | 被告 |
|------|------|
| 直接起诉 | 作出行政行为的行政机关 |
| 经复议维持 | 原行政机关和复议机关为共同被告 |
| 经复议改变 | 复议机关为被告 |
| 委托的行政机关 | 委托的行政机关 |
| 行政机关被撤销 | 继续行使其职权的行政机关 |

### 10.3 第三人

**第三人的条件**：
- 与被诉行政行为有利害关系
- 由人民法院通知参加诉讼或者自己申请参加诉讼

## 例题与解析

### 例题1

**题目**: 甲对县公安局作出的行政拘留决定不服，经市公安局复议后维持原决定。甲提起行政诉讼，被告是：

**选项**:
A. 县公安局
B. 市公安局
C. 县公安局和市公安局为共同被告
D. 由甲选择县公安局或市公安局

**答案**: C

**解析**: 经复议的案件，复议机关决定维持原行政行为的，作出原行政行为的行政机关和复议机关是共同被告。
