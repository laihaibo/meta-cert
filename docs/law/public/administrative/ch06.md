# 第六章 行政强制

## 本章概要

本章介绍行政强制的基本制度，包括行政强制措施和行政强制执行。重点掌握两者的区别和适用条件。

## 高频考点

> 🔥 高频考点

### 考点1: 行政强制措施与行政强制执行的区别
- 行政强制措施：为制止违法行为、防止证据毁损等而采取的临时性措施
- 行政强制执行：为实现行政决定的内容而采取的强制手段

### 考点2: 行政强制措施的种类
- 限制公民人身自由
- 查封场所、设施或者财物
- 扣押财物
- 冻结存款、汇款

### 考点3: 行政强制执行的方式
- 加处罚款或者滞纳金
- 划拨存款、汇款
- 拍卖或者依法处理查封、扣押的场所、设施或者财物
- 排除妨碍、恢复原状
- 代履行

## 知识点详解

### 6.1 行政强制的概念

行政强制包括行政强制措施和行政强制执行。

### 6.2 行政强制措施

**行政强制措施的种类**：

| 种类 | 说明 |
|------|------|
| 限制公民人身自由 | 强制戒毒、强制隔离等 |
| 查封场所、设施或者财物 | 对场所、设施或者财物就地封存 |
| 扣押财物 | 将财物转移至另外场所扣押 |
| 冻结存款、汇款 | 通知金融机构停止支付 |

**行政强制措施的程序**：
- 实施前须向行政机关负责人报告并经批准
- 由两名以上行政执法人员实施
- 出示执法身份证件
- 通知当事人到场
- 当场告知当事人采取行政强制措施的理由、依据以及当事人依法享有的权利、救济途径

### 6.3 行政强制执行

**行政强制执行的方式**：

| 方式 | 说明 |
|------|------|
| 加处罚款或者滞纳金 | 对逾期不履行义务的加处罚款或滞纳金 |
| 划拨存款、汇款 | 将存款、汇款划拨至指定账户 |
| 拍卖或者依法处理 | 拍卖查封、扣押的财物 |
| 排除妨碍、恢复原状 | 强制排除妨碍或恢复原状 |
| 代履行 | 由他人代为履行义务 |

**行政强制执行的程序**：
- 催告当事人履行义务
- 当事人有权陈述和申辩
- 作出强制执行决定
- 送达强制执行决定书

### 6.4 行政强制措施与行政强制执行的区别

| 比较项 | 行政强制措施 | 行政强制执行 |
|--------|-------------|-------------|
| 目的 | 制止违法行为、防止证据毁损 | 实现行政决定的内容 |
| 时间 | 行政决定作出前 | 行政决定作出后 |
| 性质 | 临时性 | 终结性 |

## 例题与解析

### 例题1

**题目**: 下列属于行政强制措施的是：

**选项**:
A. 加处罚款
B. 划拨存款
C. 查封场所
D. 拍卖财物

**答案**: C

**解析**: 查封场所属于行政强制措施，加处罚款、划拨存款、拍卖财物属于行政强制执行。
