# 第三章 行政行为

## 本章概要

本章介绍行政行为的概念、分类和效力。重点掌握具体行政行为与抽象行政行为的区分。

## 高频考点

> 🔥 高频考点

### 考点1: 具体行政行为与抽象行政行为
- 具体行政行为：针对特定对象
- 抽象行政行为：针对不特定对象

### 考点2: 行政行为的效力
- 公定力
- 确定力
- 拘束力
- 执行力

### 考点3: 行政行为的无效、撤销与废止
- 无效：重大且明显违法
- 撤销：一般违法
- 废止：合法但不再需要

## 知识点详解

### 3.1 行政行为的概念

行政行为是行政主体行使行政职权，作出的能够产生行政法律效果的行为。

### 3.2 行政行为的分类

| 分类标准 | 类型 | 说明 |
|---------|------|------|
| 对象是否特定 | 具体行政行为 / 抽象行政行为 | 具体行政行为针对特定对象 |
| 是否需要申请 | 依申请行政行为 / 依职权行政行为 | 依申请需要相对人申请 |
| 是否可以协商 | 羁束行政行为 / 自由裁量行政行为 | 羁束行为没有裁量空间 |

### 3.3 行政行为的效力

| 效力 | 内容 |
|------|------|
| 公定力 | 行政行为一经作出，即被推定为合法有效 |
| 确定力 | 行政行为一经作出，非经法定程序不得随意变更或撤销 |
| 拘束力 | 行政行为对行政主体和行政相对人都具有约束力 |
| 执行力 | 行政行为生效后，行政主体可以依法采取强制手段使其内容得以实现 |

### 3.4 行政行为的无效、撤销与废止

| 情形 | 条件 | 法律后果 |
|------|------|---------|
| 无效 | 重大且明显违法 | 自始无效 |
| 撤销 | 一般违法 | 撤销后失去效力 |
| 废止 | 合法但不再需要 | 废止后失去效力 |

## 例题与解析

### 例题1

**题目**: 下列属于具体行政行为的是：

**选项**:
A. 国务院制定行政法规
B. 某市公安局对甲作出罚款决定
C. 某省政府发布规章
D. 某市人大制定地方性法规

**答案**: B

**解析**: 具体行政行为是针对特定对象作出的行政行为，某市公安局对甲作出的罚款决定是针对特定对象的具体行政行为。
