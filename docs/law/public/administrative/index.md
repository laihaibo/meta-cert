# 行政法与行政诉讼法

## 学科概述

行政法与行政诉讼法是法考的重要科目，主要规范行政权力的行使和监督。本学科内容涵盖行政法的基本原理、行政处罚、行政许可、行政强制、行政复议、行政诉讼和国家赔偿等。
