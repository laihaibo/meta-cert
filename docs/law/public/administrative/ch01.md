## 知识点详解

### 1.1 行政法的概念

行政法是调整行政关系以及在此基础上产生的监督行政关系的法律规范和原则的总称。

### 1.2 行政法的基本原则

| 原则 | 内容 |
|------|------|
| 合法行政 | 行政机关实施行政管理，应当依照法律、法规、规章的规定进行 |
| 合理行政 | 行政机关实施行政管理，应当遵循公平、公正的原则 |
| 程序正当 | 行政机关实施行政管理，应当遵循法定程序 |
| 高效便民 | 行政机关实施行政管理，应当遵守法定时限，积极履行法定职责 |
| 诚实守信 | 行政机关公布的信息应当全面、准确、真实 |
| 权责统一 | 行政机关依法履行管理职责，违法或者不当行使职权，应当承担法律责任 |

## 例题与解析

### 例题1

**题目**: 下列哪项不属于行政法的基本原则？

**选项**:
A. 合法行政原则
B. 合理行政原则
C. 罪刑法定原则
D. 程序正当原则

**答案**: C

**解析**: 罪刑法定原则是刑法的基本原则，不是行政法的基本原则。
