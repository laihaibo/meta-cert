# 第十章 国际争端和平解决

## 本章概要

本章阐述国际争端的和平解决方式，重点掌握各种解决方式的特点。

## 高频考点

1. 政治解决方法
2. 法律解决方法
3. 国际法院的职权

## 重点难点

- **重点**：各种争端解决方式
- **难点**：国际法院的管辖权

## 知识点详解

### 一、国际争端的类型

| 类型 | 含义 |
|------|------|
| 法律争端 | 涉及法律权利义务的争端 |
| 政治争端 | 涉及政治利益的争端 |
| 事实争端 | 涉及事实问题的争端 |
| 混合争端 | 兼有法律和政治性质 |

### 二、政治解决方法

| 方法 | 含义 |
|------|------|
| 谈判与协商 | 争端当事国直接交涉 |
| 斡旋 | 第三方促成当事国谈判 |
| 调停 | 第三方提出解决方案 |
| 调查 | 由委员会查明事实 |
| 和解 | 由委员会提出解决方案 |

### 三、法律解决方法

#### 1. 仲裁

- 当事国自愿将争端提交仲裁
- 仲裁裁决具有约束力

#### 2. 司法解决

- 将争端提交国际法院
- 国际法院的判决具有约束力

### 四、国际法院

#### 1. 地位

- 联合国的主要司法机关
- 设在荷兰海牙

#### 2. 组成

- 15名法官
- 法官任期9年

#### 3. 管辖权

| 类型 | 内容 |
|------|------|
| 诉讼管辖权 | 受理国家之间的争端 |
| 咨询管辖权 | 对联合国大会、安理会等的法律问题发表咨询意见 |

#### 4. 诉讼管辖权的条件

- 双方当事国同意
- 争端必须是法律争端

## 例题与解析

### 例题1

国际法院设在：

A. 纽约
B. 日内瓦
C. 海牙
D. 维也纳

**答案**：C

**解析**：国际法院设在荷兰海牙。

### 例题2

国际法院的法官任期是：

A. 3年
B. 5年
C. 7年
D. 9年

**答案**：D

**解析**：国际法院法官任期9年。
