# 第二章 国际法主体

## 本章概要

本章阐述国际法主体的类型，重点掌握国家的要素、类型和国际组织的法律地位。

## 高频考点

1. 国家的要素
2. 国家的基本权利
3. 国际组织的法律地位

## 重点难点

- **重点**：国家的基本权利
- **难点**：国家承认与继承

## 知识点详解

### 一、国际法主体的类型

| 主体 | 地位 |
|------|------|
| 国家 | 最基本的国际法主体 |
| 国际组织 | 派生的国际法主体 |
| 民族解放组织 | 特定条件下的国际法主体 |

### 二、国家

#### 1. 国家的要素

| 要素 | 含义 |
|------|------|
| 定居的居民 | 国家存在的基本条件 |
| 确定的领土 | 国家行使权力的空间 |
| 政府 | 对内行使权力、对外交往的机构 |
| 主权 | 对内最高、对外独立 |

#### 2. 国家的类型

| 类型 | 含义 |
|------|------|
| 单一国 | 统一的立法、行政、司法系统 |
| 复合国 | 联邦、邦联等 |
| 永久中立国 | 承担永久中立义务 |

#### 3. 国家的基本权利

| 权利 | 含义 |
|------|------|
| 独立权 | 自主处理内外事务 |
| 平等权 | 国际法上的平等地位 |
| 自保权 | 自卫和国防建设的权利 |
| 管辖权 | 对领土内的人和事的管辖 |

#### 4. 国家管辖权

| 类型 | 含义 |
|------|------|
| 属地管辖 | 对领土内一切人和事的管辖 |
| 属人管辖 | 对本国公民的管辖 |
| 保护性管辖 | 对危害本国利益的外国人的管辖 |
| 普遍管辖 | 对国际犯罪的管辖 |

### 三、国家承认

#### 1. 承认的类型

| 类型 | 含义 |
|------|------|
| 对新国家的承认 | 对新国家产生的承认 |
| 对新政府的承认 | 对新政府产生的承认 |

#### 2. 承认的方式

- 明示承认：明文声明
- 默示承认：事实行为

### 四、国家继承

#### 1. 条约的继承

- 政治性条约一般不继承
- 与领土有关的条约一般继承

#### 2. 财产的继承

- 位于继承领土内的国家财产一般继承

#### 3. 债务的继承

- 国家债务一般继承
- 恶债（如战争债务）不继承

### 五、国际组织

#### 1. 联合国

- **成立**：1945年
- **主要机关**：大会、安理会、经社理事会、托管理事会、国际法院、秘书处

#### 2. 安理会

- **组成**：15个理事国（5个常任理事国、10个非常任理事国）
- **表决程序**：实质性问题需要9个同意票，且5个常任理事国无反对票（否决权）

## 例题与解析

### 例题1

国家的要素不包括：

A. 定居的居民
B. 确定的领土
C. 军队
D. 主权

**答案**：C

**解析**：国家的要素包括定居的居民、确定的领土、政府和主权。军队不是国家的必备要素。

### 例题2

安理会的表决程序中，实质性问题需要：

A. 全体一致同意
B. 9个同意票且无常任理事国反对
C. 10个同意票
D. 5个常任理事国一致同意

**答案**：B

**解析**：安理会实质性问题需要9个同意票，且5个常任理事国无反对票。
