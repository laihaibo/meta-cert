# 第九章 条约法

## 本章概要

本章阐述条约的缔结、效力、解释和终止等制度，重点掌握条约的效力和保留制度。

## 高频考点

1. 条约的缔结程序
2. 条约的保留
3. 条约的效力
4. 条约的解释

## 重点难点

- **重点**：条约的保留、效力
- **难点**：条约对第三国的效力

## 知识点详解

### 一、条约的概念

条约是指国家间所缔结的以国际法为准的书面协定。

### 二、条约的缔结程序

| 步骤 | 内容 |
|------|------|
| 谈判 | 各方就条约内容进行协商 |
| 签署 | 表示初步同意 |
| 批准 | 国家权力机关的正式同意 |
| 交换批准书 | 条约生效的最后步骤 |

### 三、条约的保留

#### 1. 概念

保留是指一国在签署、批准、接受、核准或加入条约时所作的单方面声明。

#### 2. 保留的限制

- 条约禁止保留
- 保留不在条约允许的保留范围内
- 保留与条约的目的和宗旨不相容

#### 3. 保留的效力

- 对提出保留的国家：保留范围内适用条约
- 对其他国家：可以接受或反对保留

### 四、条约的效力

#### 1. 对缔约国的效力

- 条约对缔约国有约束力
- 缔约国应善意履行条约义务

#### 2. 对第三国的效力

- 条约非经第三国同意，不为该国创设义务或权利

### 五、条约的解释

#### 1. 解释规则

- 按照条约用语的通常含义解释
- 联系上下文解释
- 参考条约的目的和宗旨

#### 2. 多种文字文本的解释

- 以作准文本为准
- 各作准文本同一作准

### 六、条约的终止

| 原因 | 含义 |
|------|------|
| 到期 | 条约期满自然终止 |
| 解除条约的条件成立 | 条约规定的解除条件出现 |
| 退出 | 缔约国退出条约 |
| 缔约各方同意 | 经各方同意终止 |
| 新的强行法规则出现 | 与新的强行法冲突 |

## 例题与解析

### 例题1

条约的保留是指：

A. 条约的解释
B. 条约的修改
C. 一国在缔约时的单方面声明
D. 条约的终止

**答案**：C

**解析**：保留是指一国在签署、批准、接受、核准或加入条约时所作的单方面声明。

### 例题2

条约对第三国的效力是：

A. 自动对第三国产生约束力
B. 非经第三国同意不为其创设义务或权利
C. 对第三国没有效力
D. 只对第三国创设权利

**答案**：B

**解析**：条约非经第三国同意，不为该国创设义务或权利。
