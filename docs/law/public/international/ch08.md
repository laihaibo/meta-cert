# 第八章 外交领事关系法

## 本章概要

本章阐述外交关系和领事关系的法律制度，重点掌握外交特权与豁免。

## 高频考点

1. 外交机关
2. 外交特权与豁免
3. 领事特权与豁免

## 重点难点

- **重点**：外交特权与豁免
- **难点**：外交特权与领事特权的区别

## 知识点详解

### 一、外交机关

#### 1. 国家外交机关

| 机关 | 职能 |
|------|------|
| 国家元首 | 代表国家进行外交活动 |
| 政府 | 领导外交工作 |
| 外交部门 | 处理日常外交事务 |

#### 2. 驻外外交机关

| 机关 | 职能 |
|------|------|
| 大使馆 | 最高级别的驻外外交机关 |
| 公使馆 | 第二级别的驻外外交机关 |
| 代办处 | 最低级别的驻外外交机关 |

### 二、外交特权与豁免

#### 1. 使馆的特权与豁免

| 特权 | 内容 |
|------|------|
| 使馆馆舍不得侵犯 | 未经馆长同意不得进入 |
| 使馆档案不得侵犯 | 使馆档案不受侵犯 |
| 通讯自由 | 使馆有使用密码通讯的权利 |
| 免纳捐税 | 使馆免纳捐税 |

#### 2. 外交人员的特权与豁免

| 特权 | 内容 |
|------|------|
| 人身不可侵犯 | 不得对外交人员加以逮捕或拘禁 |
| 寓所、文书、信件不可侵犯 | 外交人员的私人寓所不受侵犯 |
| 管辖豁免 | 外交人员免受驻在国的刑事、民事和行政管辖 |
| 免纳捐税 | 外交人员免纳捐税 |
| 免除关税 | 外交人员的私人用品免纳关税 |

#### 3. 享有外交特权的人员

| 人员 | 范围 |
|------|------|
| 外交人员 | 使馆馆长和外交职员 |
| 行政技术人员 | 使馆的行政和技术人员 |
| 服务人员 | 使馆的服务人员 |
| 家属 | 外交人员的家属 |

### 三、领事特权与豁免

#### 1. 领馆的特权

- 领馆馆舍不受侵犯
- 领馆档案不受侵犯
- 通讯自由

#### 2. 领事官员的特权

- 人身自由受一定保护
- 管辖豁免限于执行职务的行为
- 免纳捐税

### 四、外交特权与领事特权的区别

| 比较项目 | 外交特权 | 领事特权 |
|----------|----------|----------|
| 范围 | 更广 | 较窄 |
| 管辖豁免 | 完全豁免 | 限于执行职务 |
| 馆舍保护 | 不可侵犯 | 限制性保护 |

## 例题与解析

### 例题1

外交人员享有的管辖豁免是：

A. 只限于刑事管辖
B. 只限于民事管辖
C. 完全豁免刑事、民事和行政管辖
D. 没有管辖豁免

**答案**：C

**解析**：外交人员免受驻在国的刑事、民事和行政管辖。

### 例题2

使馆馆舍的保护是：

A. 受到一定限制
B. 不可侵犯
C. 需要当地警方保护
D. 没有特殊保护

**答案**：B

**解析**：使馆馆舍不可侵犯，未经馆长同意不得进入。
