# 国际法

## 科目概述

国际法是调整国家之间关系的法律规范的总称。在法考中，国际法主要考查国际法的基本理论和主要制度。

## 考情分析

- 分值占比：约10-15分
- 题型：选择题为主
- 难度：中等，需要理解国际法的基本概念
