# 第一章 国际法导论

## 本章概要

本章阐述国际法的基本概念、渊源和基本原则，是学习国际法的基础。

## 高频考点

1. 国际法的渊源
2. 国际法的基本原则
3. 国际法与国内法的关系

## 重点难点

- **重点**：国际法的渊源、基本原则
- **难点**：国际法与国内法的关系

## 知识点详解

### 一、国际法的概念

国际法是指在国际交往中形成的，主要调整国家之间关系的，具有法律约束力的原则、规则和制度的总体。

### 二、国际法的渊源

| 渊源 | 含义 |
|------|------|
| 国际条约 | 国家间缔结的书面协议 |
| 国际习惯 | 通例之证明而经接受为法律者 |
| 一般法律原则 | 各国法律体系共有的原则 |
| 司法判例 | 辅助资料 |
| 学术著作 | 辅助资料 |

### 三、国际法的基本原则

#### 1. 国家主权平等原则

- 各国主权平等
- 各国不问经济、社会、政治或其他性质有何不同，均有平等权利与责任

#### 2. 不干涉内政原则

- 任何国家或国家集团均无权以任何理由直接或间接干涉任何其他国家之内政或外交事务

#### 3. 不使用武力或武力威胁原则

- 各国有义务在其国际关系上不为侵害任何国家领土完整或政治独立之目的，或以与联合国宗旨不符之任何其他方式使用威胁或武力

#### 4. 和平解决国际争端原则

- 各国应以和平方法解决其国际争端

#### 5. 善意履行国际义务原则

- 各国应一秉善意履行其依联合国宪章所负之义务

#### 6. 民族自决原则

- 被压迫民族有摆脱殖民统治、建立独立国家的权利

### 四、国际法与国内法的关系

#### 1. 理论分歧

| 学说 | 观点 |
|------|------|
| 一元论 | 国际法与国内法属于同一法律体系 |
| 二元论 | 国际法与国内法是两个独立的法律体系 |

#### 2. 我国的实践

- 国际条约是我国法律的渊源之一
- 我国缔结或参加的国际条约与国内法有不同规定的，优先适用国际条约（保留条款除外）

## 例题与解析

### 例题1

国际法的渊源不包括：

A. 国际条约
B. 国际习惯
C. 国内法
D. 一般法律原则

**答案**：C

**解析**：国际法的渊源包括国际条约、国际习惯和一般法律原则。国内法不是国际法的渊源。

### 例题2

下列不属于国际法基本原则的是：

A. 国家主权平等原则
B. 不干涉内政原则
C. 最惠国待遇原则
D. 和平解决国际争端原则

**答案**：C

**解析**：最惠国待遇是国际贸易法中的制度，不属于国际法基本原则。
