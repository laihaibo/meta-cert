# 第五章 国家领土

## 本章概要

本章阐述国家领土的构成和领土主权，重点掌握领土的取得和变更方式。

## 高频考点

1. 领土的构成
2. 领土取得的方式
3. 边境制度

## 重点难点

- **重点**：领土取得的方式
- **难点**：领土主权的限制

## 知识点详解

### 一、领土的构成

| 部分 | 含义 |
|------|------|
| 领陆 | 国家疆界以内的陆地 |
| 领水 | 领陆内的水域和沿岸的领海 |
| 领空 | 领陆和领水上方的空气空间 |
| 底土 | 领陆和领水下的底土 |

### 二、领土取得的方式

| 方式 | 含义 |
|------|------|
| 先占 | 对无主地的有效占领 |
| 时效 | 长期占有他国领土 |
| 添附 | 领土的自然或人工增加 |
| 割让 | 通过条约转让领土 |
| 征服 | 以武力取得领土（现代国际法已禁止） |

### 三、边境制度

- 边界标志的维护
- 界河的利用
- 边境居民的往来

### 四、领土主权的限制

| 限制 | 含义 |
|------|------|
| 共管 | 两个以上国家共同行使主权 |
| 租借 | 一国将领土租给另一国使用 |
| 国际地役 | 根据条约对领土的限制 |

## 例题与解析

### 例题1

现代国际法承认的领土取得方式不包括：

A. 先占
B. 添附
C. 征服
D. 割让

**答案**：C

**解析**：现代国际法已禁止以武力征服的方式取得领土。
