# 第十一章 危害国家安全罪

## 本章概要

本章介绍危害国家安全罪的主要罪名，包括背叛国家罪、分裂国家罪、煽动分裂国家罪、颠覆国家政权罪、间谍罪等。重点掌握间谍罪和叛逃罪的构成要件。

## 高频考点

> 🔥 高频考点

### 考点1: 间谍罪
- 参加间谍组织或者接受间谍组织及其代理人的任务
- 为敌人指示轰击目标
- 为境外的机构、组织、人员窃取、刺探、收买、非法提供国家秘密或者情报

### 考点2: 叛逃罪
- 国家机关工作人员在履行公务期间，擅离岗位，叛逃境外或者在境外叛逃
- 掌握国家秘密的国家工作人员叛逃

### 考点3: 为境外窃取、刺探、收买、非法提供国家秘密、情报罪
- 与间谍罪的区别
- 与故意泄露国家秘密罪的区别

## 重点难点

> ⚠️ 难点

### 难点1: 间谍罪与为境外窃取、刺探、收买、非法提供国家秘密、情报罪的区分
- 间谍罪：参加间谍组织或接受间谍组织任务
- 为境外窃取、刺探、收买、非法提供国家秘密、情报罪：不以参加间谍组织为要件

## 知识点详解

### 11.1 背叛国家罪

#### 11.1.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 国家的主权、领土完整和安全 |
| 客观方面 | 勾结外国，危害国家的主权、领土完整和安全 |
| 主体 | 中国公民（一般主体） |
| 主观方面 | 故意 |

#### 11.1.2 刑事责任

- 处无期徒刑或者10年以上有期徒刑
- 对国家和人民危害特别严重、情节特别恶劣的，可以判处死刑
- 可以并处没收财产

### 11.2 分裂国家罪、煽动分裂国家罪

#### 11.2.1 分裂国家罪

**构成要件**：

| 要件 | 内容 |
|------|------|
| 客体 | 国家统一 |
| 客观方面 | 组织、策划、实施分裂国家、破坏国家统一 |
| 主体 | 一般主体 |
| 主观方面 | 故意 |

#### 11.2.2 煽动分裂国家罪

**构成要件**：
- 煽动分裂国家、破坏国家统一
- 以口头、文字或其他方式进行煽动

### 11.3 颠覆国家政权罪、煽动颠覆国家政权罪

#### 11.3.1 颠覆国家政权罪

**构成要件**：
- 组织、策划、实施颠覆国家政权、推翻社会主义制度
- 首要分子或者罪行重大的

### 11.4 间谍罪

#### 11.4.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 国家安全 |
| 客观方面 | 参加间谍组织或者接受间谍组织及其代理人的任务；为敌人指示轰击目标 |
| 主体 | 一般主体（中国公民、外国人、无国籍人） |
| 主观方面 | 故意 |

#### 11.4.2 刑事责任

- 处10年以上有期徒刑或者无期徒刑
- 情节较轻的，处3年以上10年以下有期徒刑
- 可以并处没收财产

### 11.5 为境外窃取、刺探、收买、非法提供国家秘密、情报罪

#### 11.5.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 国家安全和利益 |
| 客观方面 | 为境外的机构、组织、人员窃取、刺探、收买、非法提供国家秘密或者情报 |
| 主体 | 一般主体 |
| 主观方面 | 故意 |

#### 11.5.2 与相关罪名的区别

| 罪名 | 区别 |
|------|------|
| 间谍罪 | 间谍罪以参加间谍组织或接受任务为要件 |
| 故意泄露国家秘密罪 | 故意泄露国家秘密罪不要求为境外提供 |

### 11.6 叛逃罪

#### 11.6.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 国家安全 |
| 客观方面 | 国家机关工作人员在履行公务期间，擅离岗位，叛逃境外或者在境外叛逃 |
| 主体 | 国家机关工作人员（特殊主体） |
| 主观方面 | 故意 |

#### 11.6.2 加重情节

掌握国家秘密的国家工作人员叛逃的，从重处罚。

## 例题与解析

### 例题1

**题目**: 甲是某国家机关工作人员，在出国考察期间叛逃境外，并向境外机构提供了其掌握的国家秘密。甲的行为构成：

**选项**:
A. 叛逃罪
B. 为境外非法提供国家秘密罪
C. 叛逃罪和为境外非法提供国家秘密罪，数罪并罚
D. 间谍罪

**答案**: C

**解析**: 甲在履行公务期间叛逃境外，构成叛逃罪；向境外机构提供国家秘密，构成境外非法提供国家秘密罪。两罪应当数罪并罚。如果甲是掌握国家秘密的国家工作人员，叛逃罪应从重处罚。
