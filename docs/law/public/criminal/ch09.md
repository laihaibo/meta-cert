# 第九章 刑罚的裁量

## 本章概要

本章介绍刑罚裁量的基本制度，包括量刑情节、累犯、自首、立功和数罪并罚。重点掌握累犯的构成条件、自首的认定和数罪并罚的规则。

## 高频考点

> 🔥 高频考点

### 考点1: 累犯
- 一般累犯：前后罪均为故意犯罪，前罪被判处有期徒刑以上刑罚，后罪发生在前罪刑罚执行完毕或赦免后5年以内
- 特别累犯：危害国家安全犯罪、恐怖活动犯罪、黑社会性质的组织犯罪的犯罪分子，在刑罚执行完毕或者赦免以后，在任何时候再犯上述任一类罪的

### 考点2: 自首
- 一般自首：自动投案，如实供述自己的罪行
- 特别自首：被采取强制措施的犯罪嫌疑人、被告人和正在服刑的罪犯，如实供述司法机关还未掌握的本人其他罪行

### 考点3: 立功
- 一般立功：揭发他人犯罪行为，查证属实；提供重要线索，从而得以侦破其他案件
- 重大立功：揭发他人重大犯罪行为，查证属实；提供重要线索，从而得以侦破其他重大案件

### 考点4: 数罪并罚
- 吸收原则：数罪中有判处死刑或无期徒刑的，执行死刑或无期徒刑
- 限制加重原则：数罪中有判处有期徒刑的，总和刑期以下、数刑中最高刑期以上，酌情决定执行的刑期，但最高不能超过25年

## 重点难点

> ⚠️ 难点

### 难点1: 自首的认定
- "自动投案"的认定
- "如实供述"的认定

### 难点2: 数罪并罚的适用
- 判决宣告前一人犯数罪的并罚

## 知识点详解

### 9.1 量刑概述

#### 9.1.1 量刑的概念

量刑是指人民法院根据行为人所犯罪行及刑事责任的轻重，在定罪的基础上，依法决定对犯罪分子是否判处刑罚、判处何种刑罚、判处何种刑度的刑罚的刑事审判活动。

#### 9.1.2 量刑情节

**法定量刑情节**：

| 类型 | 情节 | 处理 |
|------|------|------|
| 应当从重 | 累犯 | 应当从重处罚 |
| 应当从轻、减轻或免除 | 从犯 | 应当从轻、减轻处罚或者免除处罚 |
| 应当减轻或免除 | 胁从犯 | 应当减轻处罚或者免除处罚 |
| 可以从轻、减轻或免除 | 自首 | 可以从轻或者减轻处罚；犯罪较轻的，可以免除处罚 |
| 可以从轻或减轻 | 立功 | 可以从轻或者减轻处罚 |
| 应当从轻或减轻 | 未成年人犯罪 | 应当从轻或者减轻处罚 |

**酌定量刑情节**：
- 犯罪动机
- 犯罪手段
- 犯罪的时间、地点
- 犯罪对象
- 犯罪后的态度
- 一贯表现

### 9.2 累犯

#### 9.2.1 一般累犯

**一般累犯的构成条件**：

| 条件 | 内容 |
|------|------|
| 主体条件 | 前罪和后罪都必须是故意犯罪 |
| 刑度条件 | 前罪被判处有期徒刑以上刑罚，后罪应当判处有期徒刑以上刑罚 |
| 时间条件 | 后罪发生在前罪刑罚执行完毕或者赦免以后5年以内 |

**累犯的法律后果**：
- 应当从重处罚
- 不得缓刑
- 不得假释

#### 9.2.2 特别累犯

**特别累犯的构成条件**：
- 前罪和后罪均为危害国家安全犯罪、恐怖活动犯罪、黑社会性质的组织犯罪
- 前罪刑罚执行完毕或者赦免以后，在任何时候再犯上述任一类罪

**特别累犯的特点**：
- 不要求前后罪的刑度
- 不要求前后罪的时间间隔
- 要求前后罪必须是特定类型的犯罪

### 9.3 自首

#### 9.3.1 一般自首

**一般自首的构成条件**：

| 条件 | 内容 |
|------|------|
| 自动投案 | 犯罪事实或者犯罪嫌疑人未被司法机关发觉，或者虽被发觉，但犯罪嫌疑人尚未受到讯问、未被采取强制措施时，主动、直接向公安机关、人民检察院或者人民法院投案 |
| 如实供述 | 如实供述自己的罪行 |

**"自动投案"的认定**：
- 主动投案
- 亲友规劝后陪同投案
- 亲友报案后送去投案
- 被通缉、追捕过程中投案
- 形迹可疑被盘问后主动交代

**"如实供述"的认定**：
- 如实交代自己的主要犯罪事实
- 犯有数罪的，如实供述部分犯罪的，对该部分犯罪认定为自首
- 共同犯罪案件中，除如实供述自己的罪行外，还应当供述所知的同案犯

#### 9.3.2 特别自首（准自首）

**特别自首的构成条件**：

| 条件 | 内容 |
|------|------|
| 主体条件 | 被采取强制措施的犯罪嫌疑人、被告人和正在服刑的罪犯 |
| 行为条件 | 如实供述司法机关还未掌握的本人其他罪行 |
| 罪行条件 | 供述的罪行与司法机关已掌握的罪行属于不同种罪行 |

#### 9.3.3 自首的法律后果

- 对于自首的犯罪分子，可以从轻或者减轻处罚
- 犯罪较轻的，可以免除处罚
- 犯罪后自首又有重大立功表现的，应当减轻或者免除处罚

### 9.4 立功

#### 9.4.1 立功的认定

**一般立功**：
- 揭发他人犯罪行为，查证属实
- 提供重要线索，从而得以侦破其他案件
- 阻止他人犯罪活动
- 协助司法机关抓捕其他犯罪嫌疑人

**重大立功**：
- 揭发他人重大犯罪行为，查证属实
- 提供重要线索，从而得以侦破其他重大案件
- 阻止他人重大犯罪活动
- 协助司法机关抓捕其他重大犯罪嫌疑人

#### 9.4.2 立功的法律后果

| 类型 | 法律后果 |
|------|---------|
| 一般立功 | 可以从轻或者减轻处罚 |
| 重大立功 | 可以减轻或者免除处罚 |
| 自首又有重大立功 | 应当减轻或者免除处罚 |

### 9.5 数罪并罚

#### 9.5.1 数罪并罚的原则

| 原则 | 内容 | 适用情形 |
|------|------|---------|
| 吸收原则 | 重刑吸收轻刑 | 死刑或无期徒刑 |
| 限制加重原则 | 总和刑期以下、数刑中最高刑期以上 | 有期徒刑、拘役、管制 |
| 并科原则 | 合并执行 | 附加刑 |

#### 9.5.2 数罪并罚的适用

**判决宣告前一人犯数罪的并罚**：
- 数罪中有判处死刑或无期徒刑的，执行死刑或无期徒刑
- 数罪中有判处有期徒刑的，总和刑期以下、数刑中最高刑期以上，酌情决定执行的刑期，但最高不能超过25年
- 数罪中有判处拘役的，执行拘役
- 数罪中有判处管制的，最高不能超过3年
- 数罪中有判处附加刑的，附加刑仍须执行

**判决宣告后发现漏罪的并罚（先并后减）**：
- 对新发现的罪作出判决
- 把前后两个判决所判处的刑罚进行并罚
- 已执行的刑期，应当计算在新判决决定的刑期以内

**判决宣告后又犯新罪的并罚（先减后并）**：
- 对新犯的罪作出判决
- 把前罪没有执行的刑罚和后罪所判处的刑罚进行并罚

## 例题与解析

### 例题1

**题目**: 甲因盗窃罪被判处有期徒刑3年，执行2年后又犯故意伤害罪。下列说法正确的是：

**选项**:
A. 甲构成累犯
B. 对甲应当先减后并
C. 对甲应当先并后减
D. 对甲应当数罪并罚，但不构成累犯

**答案**: B

**解析**: 甲在刑罚执行期间又犯新罪，应当对新犯的罪作出判决，把前罪没有执行的刑罚（1年）和后罪所判处的刑罚进行并罚，即先减后并。甲在刑罚执行期间犯罪，不构成累犯（累犯要求在刑罚执行完毕或赦免后5年以内再犯罪）。

### 例题2

**题目**: 甲犯罪后自动投案，如实供述了自己的罪行，但一审庭审中翻供。下列说法正确的是：

**选项**:
A. 甲不成立自首
B. 甲成立自首
C. 甲在一审判决前又能如实供述的，成立自首
D. 甲在二审中如实供述的，成立自首

**答案**: C

**解析**: 根据司法解释，犯罪嫌疑自动投案并如实供述自己的罪行后又翻供的，不能认定为自首；但在一审判决前又能如实供述的，应当认定为自首。
