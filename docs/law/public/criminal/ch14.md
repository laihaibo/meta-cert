# 第十四章 侵犯公民人身权利、民主权利罪

## 本章概要

本章介绍侵犯公民人身权利、民主权利罪的主要罪名，包括故意杀人罪、故意伤害罪、强奸罪、绑架罪、非法拘禁罪、拐卖妇女儿童罪等。重点掌握故意杀人罪、故意伤害罪、强奸罪和绑架罪的构成要件。

## 高频考点

> 🔥 高频考点

### 考点1: 故意杀人罪
- 故意非法剥夺他人生命
- 与过失致人死亡罪的区分
- 与故意伤害罪（致人死亡）的区分

### 考点2: 故意伤害罪
- 故意非法损害他人身体健康
- 轻伤、重伤、致人死亡的不同法定刑

### 考点3: 强奸罪
- 以暴力、胁迫或者其他手段强奸妇女
- 奸淫幼女的从重处罚

### 考点4: 绑架罪
- 以勒索财物为目的绑架他人
- 绑架他人作为人质
- 致使被绑架人死亡或者杀害被绑架人的处死刑

### 考点5: 非法拘禁罪
- 非法拘禁他人或者以其他方法非法剥夺他人人身自由
- 使用暴力致人伤残、死亡的转化犯

## 重点难点

> ⚠️ 难点

### 难点1: 故意杀人罪与故意伤害罪（致人死亡）的区分
- 主观上是杀人故意还是伤害故意
- 客观行为是否足以致人死亡

### 难点2: 绑架罪与抢劫罪的区分
- 绑架罪：向第三人勒索财物
- 抢劫罪：直接向被害人劫取财物

### 难点3: 拐卖妇女、儿童罪的加重情节
- 拐卖妇女、儿童集团的首要分子
- 拐卖妇女、儿童三人以上
- 奸淫被拐卖的妇女

## 知识点详解

### 14.1 故意杀人罪

#### 14.1.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 他人的生命权 |
| 客观方面 | 故意非法剥夺他人生命 |
| 主体 | 一般主体（已满14周岁） |
| 主观方面 | 故意（直接故意或间接故意） |

#### 14.1.2 刑事责任

| 情节 | 刑罚 |
|------|------|
| 基本犯 | 死刑、无期徒刑或者10年以上有期徒刑 |
| 情节较轻 | 3年以上10年以下有期徒刑 |

**"情节较轻"的情形**：
- 义愤杀人
- 防卫过当杀人
- 受被害人嘱托杀人
- 帮助自杀

#### 14.1.3 相关罪名的区别

| 罪名 | 区别要点 |
|------|---------|
| 故意伤害罪（致人死亡） | 主观上只有伤害故意，对死亡结果是过失 |
| 过失致人死亡罪 | 主观上没有杀人的故意，对死亡结果是过失 |

### 14.2 故意伤害罪

#### 14.2.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 他人的身体健康权 |
| 客观方面 | 故意非法损害他人身体健康 |
| 主体 | 一般主体（已满16周岁；故意伤害致人重伤或死亡的，已满14周岁） |
| 主观方面 | 故意 |

#### 14.2.2 刑事责任

| 结果 | 刑罚 |
|------|------|
| 轻伤 | 3年以下有期徒刑、拘役或者管制 |
| 重伤 | 3年以上10年以下有期徒刑 |
| 致人死亡 | 10年以上有期徒刑、无期徒刑或者死刑 |
| 以特别残忍手段致人重伤造成严重残疾 | 10年以上有期徒刑、无期徒刑或者死刑 |

### 14.3 强奸罪

#### 14.3.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 妇女的性自主权 |
| 客观方面 | 以暴力、胁迫或者其他手段强奸妇女 |
| 主体 | 一般主体（已满14周岁） |
| 主观方面 | 故意 |

**"其他手段"的认定**：
- 醉酒、麻醉
- 利用妇女熟睡
- 冒充妇女丈夫
- 利用迷信、邪教

#### 14.3.2 加重情节

| 情节 | 刑罚 |
|------|------|
| 强奸妇女、奸淫幼女情节恶劣 | 10年以上有期徒刑、无期徒刑或者死刑 |
| 强奸妇女、奸淫幼女多人 | 10年以上有期徒刑、无期徒刑或者死刑 |
| 在公共场所当众强奸妇女 | 10年以上有期徒刑、无期徒刑或者死刑 |
| 二人以上轮奸 | 10年以上有期徒刑、无期徒刑或者死刑 |
| 致使被害人重伤、死亡或者造成其他严重后果 | 10年以上有期徒刑、无期徒刑或者死刑 |

#### 14.3.3 奸淫幼女的从重处罚

- 奸淫不满14周岁的幼女的，以强奸论，从重处罚
- 不论幼女是否同意，均构成强奸罪

### 14.4 绑架罪

#### 14.4.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 他人的人身自由权和财产权 |
| 客观方面 | 以勒索财物为目的绑架他人，或者绑架他人作为人质 |
| 主体 | 一般主体（已满16周岁） |
| 主观方面 | 故意，具有勒索财物或其他不法目的 |

#### 14.4.2 刑事责任

| 情节 | 刑罚 |
|------|------|
| 基本犯 | 10年以上有期徒刑或者无期徒刑 |
| 致使被绑架人死亡或者杀害被绑架人 | 死刑 |

#### 14.4.3 绑架罪与相关罪名的区别

| 罪名 | 区别要点 |
|------|---------|
| 抢劫罪 | 绑架罪向第三人勒索财物，抢劫罪直接向被害人劫取财物 |
| 非法拘禁罪 | 绑架罪有勒索财物或其他不法目的，非法拘禁罪没有 |
| 敲诈勒索罪 | 绑架罪实际控制被害人，敲诈勒索罪没有实际控制 |

### 14.5 非法拘禁罪

#### 14.5.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 他人的人身自由权 |
| 客观方面 | 非法拘禁他人或者以其他方法非法剥夺他人人身自由 |
| 主体 | 一般主体 |
| 主观方面 | 故意 |

#### 14.5.2 转化犯

- 使用暴力致人伤残的，以故意伤害罪定罪处罚
- 使用暴力致人死亡的，以故意杀人罪定罪处罚

#### 14.5.3 从重处罚情节

- 具有殴打、侮辱情节的，从重处罚
- 国家机关工作人员利用职权犯本罪的，从重处罚

### 14.6 拐卖妇女、儿童罪

#### 14.6.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 妇女、儿童的人身自由权和人格尊严 |
| 客观方面 | 拐卖妇女、儿童（以出卖为目的，有拐骗、绑架、收买、贩卖、接送、中转行为之一） |
| 主体 | 一般主体 |
| 主观方面 | 故意，以出卖为目的 |

#### 14.6.2 加重情节

| 情节 | 刑罚 |
|------|------|
| 拐卖妇女、儿童集团的首要分子 | 10年以上有期徒刑或者无期徒刑 |
| 拐卖妇女、儿童三人以上 | 10年以上有期徒刑或者无期徒刑 |
| 奸淫被拐卖的妇女 | 10年以上有期徒刑或者无期徒刑 |
| 诱骗、强迫被拐卖的妇女卖淫 | 10年以上有期徒刑或者无期徒刑 |
| 以出卖为目的，使用暴力、胁迫或者麻醉方法绑架妇女、儿童 | 10年以上有期徒刑或者无期徒刑 |
| 以出卖为目的，偷盗婴幼儿 | 10年以上有期徒刑或者无期徒刑 |
| 造成被拐卖的妇女、儿童或者其亲属重伤、死亡或者其他严重后果 | 10年以上有期徒刑或者无期徒刑 |
| 将妇女、儿童卖往境外 | 10年以上有期徒刑或者无期徒刑 |

### 14.7 侮辱罪、诽谤罪

#### 14.7.1 侮辱罪

**构成要件**：
- 以暴力或者其他方法公然侮辱他人
- 情节严重
- 告诉的才处理（严重危害社会秩序和国家利益的除外）

#### 14.7.2 诽谤罪

**构成要件**：
- 捏造事实诽谤他人
- 情节严重
- 告诉的才处理（严重危害社会秩序和国家利益的除外）

## 例题与解析

### 例题1

**题目**: 甲以勒索财物为目的，绑架乙作为人质，向乙的家属索要赎金。在绑架过程中，甲不慎致乙死亡。甲的行为构成：

**选项**:
A. 绑架罪的基本犯
B. 绑架罪，致使被绑架人死亡
C. 绑架罪和过失致人死亡罪，数罪并罚
D. 故意杀人罪

**答案**: B

**解析**: 甲以勒索财物为目的绑架乙，构成绑架罪。在绑架过程中致使被绑架人死亡，属于绑架罪的加重情节，处死刑，并处没收财产。

### 例题2

**题目**: 甲非法拘禁乙，在拘禁过程中对乙进行殴打，致乙重伤。甲的行为构成：

**选项**:
A. 非法拘禁罪
B. 非法拘禁罪和故意伤害罪，数罪并罚
C. 故意伤害罪
D. 非法拘禁罪，从重处罚

**答案**: C

**解析**: 根据《刑法》第238条，非法拘禁他人，使用暴力致人伤残的，以故意伤害罪定罪处罚。甲在非法拘禁过程中使用暴力致乙重伤，应以故意伤害罪定罪处罚。
