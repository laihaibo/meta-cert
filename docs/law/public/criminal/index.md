# 刑法

## 学科概述

刑法是法考中分值占比最高的学科之一，属于**公法**范畴，主要规定犯罪与刑罚。刑法学科理论性强、体系严密，是法考客观题和主观题的重点考查对象。

## 分值权重

| 客观题 | 35-40分 | ⭐⭐⭐⭐⭐ |
| 主观题 | 30-35分 | ⭐⭐⭐⭐⭐ |

## 备考建议

1. **总则部分**：重点掌握犯罪构成四要件、共同犯罪、犯罪形态
2. **分则部分**：重点掌握侵犯人身权利罪和侵犯财产罪
3. **学习方法**：以总则带动分则，注意罪名之间的对比
4. **做题策略**：先理解原理，再通过大量练习巩固
