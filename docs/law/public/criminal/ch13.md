# 第十三章 破坏社会主义市场经济秩序罪

## 本章概要

本章介绍破坏社会主义市场经济秩序罪的主要罪名，包括生产销售伪劣产品罪、走私罪、妨害对公司企业的管理秩序罪、破坏金融管理秩序罪、金融诈骗罪、危害税收征管罪、侵犯知识产权罪、扰乱市场秩序罪等。重点掌握合同诈骗罪和非法经营罪。

## 高频考点

> 🔥 高频考点

### 考点1: 生产、销售伪劣产品罪
- 销售金额5万元以上
- 货值金额15万元以上（未遂）

### 考点2: 合同诈骗罪
- 以非法占有为目的
- 在签订、履行合同过程中，骗取对方当事人财物

### 考点3: 非法经营罪
- 未经许可经营法律、行政法规规定的专营、专卖物品或者其他限制买卖的物品
- 买卖进出口许可证、进出口原产地证明以及其他法律、行政法规规定的经营许可证或者批准文件
- 未经国家有关主管部门批准非法经营证券、期货、保险业务，或者非法从事资金支付结算业务
- 其他严重扰乱市场秩序的非法经营行为

### 考点4: 信用卡诈骗罪
- 使用伪造的信用卡
- 使用作废的信用卡
- 冒用他人信用卡
- 恶意透支

## 重点难点

> ⚠️ 难点

### 难点1: 合同诈骗罪与诈骗罪的区分
- 合同诈骗罪发生在签订、履行合同过程中
- 诈骗罪不限于合同关系

### 难点2: 非法经营罪的"口袋罪"问题
- 非法经营罪的构成要件较为宽泛
- 司法实践中适用范围不断扩大

## 知识点详解

### 13.1 生产、销售伪劣产品罪

#### 13.1.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 国家产品质量管理秩序和消费者的合法权益 |
| 客观方面 | 生产者、销售者在产品中掺杂、掺假，以假充真，以次充好或者以不合格产品冒充合格产品，销售金额5万元以上 |
| 主体 | 一般主体（生产者、销售者） |
| 主观方面 | 故意 |

#### 13.1.2 刑事责任

| 销售金额 | 刑罚 |
|---------|------|
| 5万元以上不满20万元 | 2年以下有期徒刑或者拘役 |
| 20万元以上不满50万元 | 2年以上7年以下有期徒刑 |
| 50万元以上不满200万元 | 7年以上15年以下有期徒刑 |
| 200万元以上 | 15年有期徒刑或者无期徒刑 |

### 13.2 走私罪

#### 13.2.1 走私普通货物、物品罪

**构成要件**：
- 客体：国家对外贸易管制
- 客观方面：违反海关法规，逃避海关监管，走私货物、物品偷逃应缴税额较大或者一年内曾因走私被给予二次行政处罚后又走私
- 主体：一般主体
- 主观方面：故意

### 13.3 妨害信用卡管理罪

#### 13.3.1 构成要件

**妨害信用卡管理的行为类型**：
- 明知是伪造的信用卡而持有、运输
- 明知是伪造的空白信用卡而持有、运输，数量较大
- 非法持有他人信用卡，数量较大
- 使用虚假的身份证明骗领信用卡
- 出售、购买、为他人提供伪造的信用卡或者以虚假的身份证明骗领的信用卡

### 13.4 信用卡诈骗罪

#### 13.4.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 国家金融管理秩序和公私财产所有权 |
| 客观方面 | 使用伪造的信用卡、使用作废的信用卡、冒用他人信用卡、恶意透支 |
| 主体 | 一般主体 |
| 主观方面 | 故意，具有非法占有目的 |

**恶意透支的认定**：
- 持卡人以非法占有为目的
- 超过规定限额或者规定期限透支
- 经发卡银行两次有效催收后超过三个月仍不归还

### 13.5 合同诈骗罪

#### 13.5.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 国家对合同的管理秩序和公私财产所有权 |
| 客观方面 | 在签订、履行合同过程中，骗取对方当事人财物，数额较大 |
| 主体 | 一般主体 |
| 主观方面 | 故意，具有非法占有目的 |

**合同诈骗的行为方式**：
- 以虚构的单位或者冒用他人名义签订合同
- 以伪造、变造、作废的票据或者其他虚假的产权证明作担保
- 没有实际履行能力，以先履行小额合同或者部分履行合同的方法，诱骗对方当事人继续签订和履行合同
- 收受对方当事人给付的货物、货款、预付款或者担保财产后逃匿
- 以其他方法骗取对方当事人财物

#### 13.5.2 刑事责任

| 数额 | 刑罚 |
|------|------|
| 数额较大 | 3年以下有期徒刑或者拘役 |
| 数额巨大或有其他严重情节 | 3年以上10年以下有期徒刑 |
| 数额特别巨大或有其他特别严重情节 | 10年以上有期徒刑或者无期徒刑 |

### 13.6 非法经营罪

#### 13.6.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 国家市场管理秩序 |
| 客观方面 | 违反国家规定，非法经营，扰乱市场秩序，情节严重 |
| 主体 | 一般主体 |
| 主观方面 | 故意 |

**非法经营行为的类型**：
- 未经许可经营法律、行政法规规定的专营、专卖物品或者其他限制买卖的物品
- 买卖进出口许可证、进出口原产地证明以及其他法律、行政法规规定的经营许可证或者批准文件
- 未经国家有关主管部门批准非法经营证券、期货、保险业务，或者非法从事资金支付结算业务
- 其他严重扰乱市场秩序的非法经营行为

### 13.7 侵犯知识产权罪

#### 13.7.1 假冒注册商标罪

**构成要件**：
- 未经注册商标所有人许可
- 在同一种商品、服务上使用与其注册商标相同的商标
- 情节严重

#### 13.7.2 侵犯著作权罪

**构成要件**：
- 以营利为目的
- 未经著作权人许可，复制发行其作品
- 违法所得数额较大或者有其他严重情节

### 13.8 扰乱市场秩序罪

#### 13.8.1 损害商业信誉、商品声誉罪

**构成要件**：
- 捏造并散布虚伪事实
- 损害他人的商业信誉、商品声誉
- 给他人造成重大损失或者有其他严重情节

#### 13.8.2 虚假广告罪

**构成要件**：
- 广告主、广告经营者、广告发布者
- 违反国家规定，利用广告对商品或者服务作虚假宣传
- 情节严重

## 例题与解析

### 例题1

**题目**: 甲以非法占有为目的，冒用他人名义与乙公司签订合同，骗取乙公司货款100万元。甲的行为构成：

**选项**:
A. 诈骗罪
B. 合同诈骗罪
C. 侵占罪
D. 合同诈骗罪和诈骗罪的想象竞合

**答案**: B

**解析**: 甲在签订、履行合同过程中，冒用他人名义签订合同，骗取对方当事人财物，构成合同诈骗罪。合同诈骗罪是特殊规定，优先于诈骗罪适用。

### 例题2

**题目**: 甲未经许可，擅自经营烟草专卖品，情节严重。甲的行为构成：

**选项**:
A. 非法经营罪
B. 生产、销售伪劣产品罪
C. 走私罪
D. 不构成犯罪

**答案**: A

**解析**: 甲未经许可经营法律、行政法规规定的专营、专卖物品，扰乱市场秩序，情节严重，构成非法经营罪。
