# 第十五章 侵犯财产罪

## 本章概要

本章介绍侵犯财产罪的主要罪名，包括抢劫罪、盗窃罪、诈骗罪、抢夺罪、聚众哄抢罪、侵占罪、职务侵占罪、挪用资金罪、敲诈勒索罪、故意毁坏财物罪等。重点掌握抢劫罪、盗窃罪、诈骗罪和侵占罪的构成要件。

## 高频考点

> 🔥 高频考点

### 考点1: 抢劫罪
- 以暴力、胁迫或者其他方法抢劫公私财物
- 加重情节：入户抢劫、在公共交通工具上抢劫、抢劫银行或其他金融机构、多次抢劫、抢劫致人重伤或死亡

### 考点2: 盗窃罪
- 盗窃公私财物，数额较大，或者多次盗窃、入户盗窃、携带凶器盗窃、扒窃
- 盗窃罪的既遂标准：失控说或控制说

### 考点3: 诈骗罪
- 以非法占有为目的，用虚构事实或者隐瞒真相的方法，骗取数额较大的公私财物
- 诈骗罪的基本结构：欺骗行为→错误认识→处分财产→财产损害

### 考点4: 侵占罪
- 将代为保管的他人财物非法占为己有，数额较大，拒不退还
- 将他人的遗忘物或者埋藏物非法占为己有，数额较大，拒不交出

### 考点5: 敲诈勒索罪
- 以非法占有为目的，对被害人使用威胁或要挟的方法，强行索要公私财物

## 重点难点

> ⚠️ 难点

### 难点1: 抢劫罪与抢夺罪的区分
- 抢劫罪：对人使用暴力、胁迫或其他方法
- 抢夺罪：对物使用强力，对人没有使用暴力

### 难点2: 盗窃罪与侵占罪的区分
- 盗窃罪：将他人占有的财物转移为自己占有
- 侵占罪：将自己占有的他人财物非法占为己有

### 难点3: 诈骗罪与民事欺诈的区分
- 诈骗罪：以非法占有为目的，根本不想履行
- 民事欺诈：有一定的履行意愿，但存在欺诈行为

## 知识点详解

### 15.1 抢劫罪

#### 15.1.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 公私财产所有权和公民的人身权利 |
| 客观方面 | 以暴力、胁迫或者其他方法抢劫公私财物 |
| 主体 | 一般主体（已满14周岁） |
| 主观方面 | 故意，具有非法占有目的 |

**"其他方法"的认定**：
- 醉酒、麻醉
- 利用被害人熟睡、醉酒状态
- 使被害人陷入不能反抗的状态

#### 15.1.2 加重情节

| 情节 | 刑罚 |
|------|------|
| 入户抢劫 | 10年以上有期徒刑、无期徒刑或者死刑 |
| 在公共交通工具上抢劫 | 10年以上有期徒刑、无期徒刑或者死刑 |
| 抢劫银行或者其他金融机构 | 10年以上有期徒刑、无期徒刑或者死刑 |
| 多次抢劫或者抢劫数额巨大 | 10年以上有期徒刑、无期徒刑或者死刑 |
| 抢劫致人重伤、死亡 | 10年以上有期徒刑、无期徒刑或者死刑 |
| 冒充军警人员抢劫 | 10年以上有期徒刑、无期徒刑或者死刑 |
| 持枪抢劫 | 10年以上有期徒刑、无期徒刑或者死刑 |
| 抢劫军用物资或者抢险、救灾、救济物资 | 10年以上有期徒刑、无期徒刑或者死刑 |

#### 15.1.3 转化型抢劫

《刑法》第269条：犯盗窃、诈骗、抢夺罪，为窝藏赃物、抗拒抓捕或者毁灭罪证而当场使用暴力或者以暴力相威胁的，以抢劫罪定罪处罚。

**转化型抢劫的条件**：
- 前提条件：犯盗窃、诈骗、抢夺罪
- 主观条件：为窝藏赃物、抗拒抓捕或者毁灭罪证
- 客观条件：当场使用暴力或者以暴力相威胁

### 15.2 盗窃罪

#### 15.2.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 公私财产所有权 |
| 客观方面 | 盗窃公私财物，数额较大，或者多次盗窃、入户盗窃、携带凶器盗窃、扒窃 |
| 主体 | 一般主体 |
| 主观方面 | 故意，具有非法占有目的 |

**盗窃罪的行为方式**：

| 方式 | 说明 |
|------|------|
| 数额较大 | 盗窃公私财物价值1000元至3000元以上 |
| 多次盗窃 | 二年内盗窃三次以上 |
| 入户盗窃 | 非法进入供他人家庭生活，与外界相对隔离的住所盗窃 |
| 携带凶器盗窃 | 携带枪支、爆炸物、管制刀具等国家禁止个人携带的器械盗窃 |
| 扒窃 | 在公共场所或者公共交通工具上盗窃他人随身携带的财物 |

#### 15.2.2 既遂标准

- **失控说**：财物脱离被害人控制为既遂
- **控制说**：行为人实际控制财物为既遂
- **通说**：失控说或控制说（以行为人实际控制财物为既遂）

#### 15.2.3 刑事责任

| 数额 | 刑罚 |
|------|------|
| 数额较大 | 3年以下有期徒刑、拘役或者管制 |
| 数额巨大或有其他严重情节 | 3年以上10年以下有期徒刑 |
| 数额特别巨大或有其他特别严重情节 | 10年以上有期徒刑或者无期徒刑 |

### 15.3 诈骗罪

#### 15.3.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 公私财产所有权 |
| 客观方面 | 以虚构事实或者隐瞒真相的方法，骗取数额较大的公私财物 |
| 主体 | 一般主体 |
| 主观方面 | 故意，具有非法占有目的 |

**诈骗罪的基本结构**：
1. 行为人实施欺骗行为
2. 被害人陷入错误认识
3. 被害人基于错误认识处分财产
4. 行为人或第三人取得财产
5. 被害人遭受财产损失

#### 15.3.2 刑事责任

| 数额 | 刑罚 |
|------|------|
| 数额较大 | 3年以下有期徒刑、拘役或者管制 |
| 数额巨大或有其他严重情节 | 3年以上10年以下有期徒刑 |
| 数额特别巨大或有其他特别严重情节 | 10年以上有期徒刑或者无期徒刑 |

### 15.4 抢夺罪

#### 15.4.1 构成要件

- 客体：公私财产所有权
- 客观方面：乘人不备，公然夺取数额较大的公私财物
- 主体：一般主体
- 主观方面：故意，具有非法占有目的

#### 15.4.2 与抢劫罪的区别

| 比较项 | 抢劫罪 | 抢夺罪 |
|--------|--------|--------|
| 行为对象 | 对人使用暴力 | 对物使用强力 |
| 危害程度 | 更重 | 较轻 |
| 主体年龄 | 已满14周岁 | 已满16周岁 |

### 15.5 侵占罪

#### 15.5.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 公私财产所有权 |
| 客观方面 | 将代为保管的他人财物非法占为己有，数额较大，拒不退还；或者将他人的遗忘物或者埋藏物非法占为己有，数额较大，拒不交出 |
| 主体 | 一般主体 |
| 主观方面 | 故意，具有非法占有目的 |

#### 15.5.2 与盗窃罪的区别

| 比较项 | 盗窃罪 | 侵占罪 |
|--------|--------|--------|
| 犯罪对象 | 他人占有的财物 | 自己占有的他人财物 |
| 行为方式 | 转移占有 | 不退还或不交出 |
| 告诉方式 | 公诉 | 告诉才处理 |

### 15.6 职务侵占罪

#### 15.6.1 构成要件

- 客体：公司、企业或者其他单位的财产所有权
- 客观方面：公司、企业或者其他单位的人员，利用职务上的便利，将本单位财物非法占为己有，数额较大
- 主体：公司、企业或者其他单位的人员（非国家工作人员）
- 主观方面：故意，具有非法占有目的

### 15.7 挪用资金罪

#### 15.7.1 构成要件

- 客体：公司、企业或者其他单位的资金使用权
- 客观方面：公司、企业或者其他单位的工作人员，利用职务上的便利，挪用本单位资金归个人使用或者借贷给他人
- 主体：公司、企业或者其他单位的工作人员（非国家工作人员）
- 主观方面：故意

### 15.8 敲诈勒索罪

#### 15.8.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 公私财产所有权和公民的人身权利 |
| 客观方面 | 以威胁或者要挟的方法，强行索要公私财物 |
| 主体 | 一般主体 |
| 主观方面 | 故意，具有非法占有目的 |

#### 15.8.2 与抢劫罪的区别

| 比较项 | 抢劫罪 | 敲诈勒索罪 |
|--------|--------|-----------|
| 威胁内容 | 暴力威胁 | 暴力威胁或其他威胁 |
| 威胁时间 | 当场实现 | 将来实现 |
| 取得财物 | 当场取得 | 将来取得 |

### 15.9 故意毁坏财物罪

#### 15.9.1 构成要件

- 客体：公私财产所有权
- 客观方面：故意毁坏公私财物，数额较大或者有其他严重情节
- 主体：一般主体
- 主观方面：故意

## 例题与解析

### 例题1

**题目**: 甲入户盗窃，在盗窃过程中被户主乙发现，甲为抗拒抓捕将乙打成重伤。甲的行为构成：

**选项**:
A. 盗窃罪和故意伤害罪，数罪并罚
B. 抢劫罪（转化型抢劫）
C. 抢劫罪（入户抢劫）
D. 盗窃罪，从重处罚

**答案**: C

**解析**: 甲犯盗窃罪，为抗拒抓捕而当场使用暴力，转化为抢劫罪。甲入户盗窃，转化为抢劫后属于入户抢劫，应处10年以上有期徒刑、无期徒刑或者死刑。

### 例题2

**题目**: 甲将乙委托其保管的财物非法占为己有，拒不退还。甲的行为构成：

**选项**:
A. 盗窃罪
B. 诈骗罪
C. 侵占罪
D. 职务侵占罪

**答案**: C

**解析**: 甲将代为保管的他人财物非法占为己有，拒不退还，构成侵占罪。侵占罪的对象是行为人自己占有的他人财物，与盗窃罪（他人占有的财物）相区别。

### 例题3

**题目**: 甲以非法占有为目的，对乙说"你如果不给我10万元，我就把你贪污的事举报给纪委"，乙被迫给了甲10万元。甲的行为构成：

**选项**:
A. 诈骗罪
B. 抢劫罪
C. 敲诈勒索罪
D. 不构成犯罪

**答案**: C

**解析**: 甲以非法占有为目的，以举报乙的违法行为相威胁，迫使乙交出财物，构成敲诈勒索罪。甲的威胁内容是将来的不利后果，且乙有一定的意志自由，不构成抢劫罪。
