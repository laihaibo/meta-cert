# 第八章 刑罚概说

## 本章概要

本章介绍刑罚的基本概念、刑罚体系和各种刑罚种类。重点掌握主刑和附加刑的种类、适用条件。

## 高频考点

> 🔥 高频考点

### 考点1: 主刑的种类
- 管制、拘役、有期徒刑、无期徒刑、死刑
- 主刑只能独立适用，不能附加适用

### 考点2: 死刑的限制适用
- 死刑只适用于罪行极其严重的犯罪分子
- 犯罪时不满18周岁的人和审判时怀孕的妇女不适用死刑
- 审判时已满75周岁的人不适用死刑，但以特别残忍手段致人死亡的除外

### 考点3: 附加刑的种类
- 罚金、剥夺政治权利、没收财产、驱逐出境
- 附加刑可以独立适用，也可以附加适用

## 知识点详解

### 8.1 刑罚概述

#### 8.1.1 刑罚的概念与特征

刑罚是刑法规定的由国家审判机关依法对犯罪分子适用的限制或剥夺其某种权益的最严厉的强制方法。

**刑罚的特征**：
- 刑罚是最严厉的法律制裁方法
- 刑罚只能由人民法院依法适用
- 刑罚只能对犯罪分子适用

#### 8.1.2 刑罚的目的

| 目的 | 内容 |
|------|------|
| 特殊预防 | 预防犯罪分子再次犯罪 |
| 一般预防 | 预防社会上不稳定分子犯罪 |

### 8.2 主刑

#### 8.2.1 管制

| 项目 | 内容 |
|------|------|
| 期限 | 3个月以上2年以下，数罪并罚最高不超过3年 |
| 执行机关 | 司法行政机关（社区矫正机构） |
| 权利限制 | 未经执行机关批准，不得行使言论、出版、集会、结社、游行、示威自由的权利 |
| 禁止令 | 可以同时宣告禁止令 |
| 折抵 | 判决执行前先行羁押的，羁押1日折抵刑期2日 |

#### 8.2.2 拘役

| 项目 | 内容 |
|------|------|
| 期限 | 1个月以上6个月以下，数罪并罚最高不超过1年 |
| 执行机关 | 公安机关就近执行 |
| 待遇 | 每月可以回家1天至2天；参加劳动的，可以酌量发给报酬 |
| 折抵 | 判决执行前先行羁押的，羁押1日折抵刑期1日 |

#### 8.2.3 有期徒刑

| 项目 | 内容 |
|------|------|
| 期限 | 6个月以上15年以下，数罪并罚最高不超过25年 |
| 执行机关 | 监狱或其他执行场所 |
| 待遇 | 凡有劳动能力的，都应当参加劳动 |
| 折抵 | 判决执行前先行羁押的，羁押1日折抵刑期1日 |

#### 8.2.4 无期徒刑

| 项目 | 内容 |
|------|------|
| 期限 | 剥夺终身自由 |
| 执行机关 | 监狱 |
| 减刑 | 减刑后实际执行的刑期不能少于13年 |
| 假释 | 实际执行13年以上，可以假释 |

#### 8.2.5 死刑

**死刑的适用条件**：
- 死刑只适用于罪行极其严重的犯罪分子
- 死刑除依法由最高人民法院判决的以外，都应当报请最高人民法院核准

**不适用死刑的情形**：
- 犯罪时不满18周岁的人
- 审判时怀孕的妇女
- 审判时已满75周岁的人（以特别残忍手段致人死亡的除外）

**死缓制度**：

| 情形 | 法律后果 |
|------|---------|
| 死缓期间没有故意犯罪 | 减为无期徒刑 |
| 死缓期间确有重大立功表现 | 减为25年有期徒刑 |
| 死缓期间故意犯罪，情节恶劣 | 报请最高人民法院核准执行死刑 |
| 死缓期间故意犯罪，情节不恶劣 | 死刑缓期执行的期间重新计算 |

### 8.3 附加刑

#### 8.3.1 罚金

**罚金的适用方式**：
- 选处罚金
- 单处罚金
- 并处罚金
- 并处或者单处罚金

**罚金的数额**：
- 无限额罚金制
- 限额罚金制
- 倍比罚金制

#### 8.3.2 剥夺政治权利

**剥夺政治权利的内容**：
- 选举权和被选举权
- 言论、出版、集会、结社、游行、示威自由的权利
- 担任国家机关职务的权利
- 担任国有公司、企业、事业单位和人民团体领导职务的权利

**剥夺政治权利的适用情形**：
- 对于危害国家安全的犯罪分子应当附加剥夺政治权利
- 对于被判处死刑、无期徒刑的犯罪分子，应当剥夺政治权利终身
- 对于故意杀人、强奸、放火、爆炸、投毒、抢劫等严重破坏社会秩序的犯罪分子，可以附加剥夺政治权利

**剥夺政治权利的期限**：
- 独立适用或判处有期徒刑、拘役附加适用：1年以上5年以下
- 判处管制附加适用：与管制期限相同
- 判处死刑、无期徒刑附加适用：剥夺政治权利终身
- 死缓、无期徒刑减为有期徒刑时：3年以上10年以下

#### 8.3.3 没收财产

**没收财产的范围**：没收犯罪分子个人所有财产的一部分或全部

**没收财产的限制**：
- 没收财产以前犯罪分子所负的正当债务，需要以没收的财产偿还的，经债权人请求，应当偿还
- 不得没收属于犯罪分子家属所有或者应有的财产

#### 8.3.4 驱逐出境

驱逐出境是强迫犯罪的外国人离开中国国（边）境的刑罚方法。

- 只适用于犯罪的外国人
- 可以独立适用，也可以附加适用
- 附加适用时，在主刑执行完毕后执行

### 8.4 非刑罚处罚措施

#### 8.4.1 非刑罚处罚措施的种类

| 类型 | 内容 |
|------|------|
| 训诫 | 对犯罪分子进行批评教育 |
| 责令具结悔过 | 责令犯罪分子写出书面保证 |
| 赔礼道歉 | 责令犯罪分子向被害人赔礼道歉 |
| 赔偿损失 | 责令犯罪分子赔偿被害人经济损失 |
| 行政处罚 | 由主管部门给予行政处罚或行政处分 |
| 禁止令 | 禁止从事特定活动、进入特定区域、接触特定人 |

## 例题与解析

### 例题1

**题目**: 下列关于死刑的说法，正确的是：

**选项**:
A. 犯罪时不满18周岁的人不适用死刑，包括不适用死缓
B. 审判时怀孕的妇女可以适用死缓
C. 审判时已满75周岁的人一律不适用死刑
D. 死刑缓期执行期间故意犯罪的，一律执行死刑

**答案**: A

**解析**: A项正确，犯罪时不满18周岁的人不适用死刑，包括死刑立即执行和死刑缓期执行。B项错误，审判时怀孕的妇女不适用死刑，包括死缓。C项错误，审判时已满75周岁的人不适用死刑，但以特别残忍手段致人死亡的除外。D项错误，死缓期间故意犯罪，情节恶劣的，报请最高人民法院核准执行死刑；情节不恶劣的，重新计算死缓期间。
