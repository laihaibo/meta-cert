# 第十二章 危害公共安全罪

## 本章概要

本章介绍危害公共安全罪的主要罪名，包括放火罪、爆炸罪、投放危险物质罪、以危险方法危害公共安全罪、交通肇事罪、危险驾驶罪等。重点掌握交通肇事罪和危险驾驶罪的构成要件。

## 高频考点

> 🔥 高频考点

### 考点1: 放火罪、爆炸罪、投放危险物质罪
- 危险犯，不要求造成实际损害
- 与故意杀人罪的竞合关系

### 考点2: 交通肇事罪
- 违反交通运输管理法规，因而发生重大事故，致人重伤、死亡或者使公私财产遭受重大损失
- 逃逸的加重情节
- 因逃逸致人死亡的加重处罚

### 考点3: 危险驾驶罪
- 追逐竞驶，情节恶劣
- 醉酒驾驶机动车
- 从事校车业务或者旅客运输，严重超过额定乘员载客，或者严重超过规定时速行驶
- 违反危险化学品安全管理规定运输危险化学品，危及公共安全

### 考点4: 以危险方法危害公共安全罪
- 与放火罪、爆炸罪、投放危险物质罪的关系
- "其他危险方法"的认定

## 重点难点

> ⚠️ 难点

### 难点1: 交通肇事罪与危险驾驶罪的关系
- 危险驾驶罪是抽象危险犯
- 交通肇事罪是结果犯
- 危险驾驶行为同时构成交通肇事罪的，依照处罚较重的规定定罪处罚

### 难点2: 交通肇事罪中的逃逸
- "交通运输肇事后逃逸"：加重处罚
- "因逃逸致人死亡"：更重的加重处罚

## 知识点详解

### 12.1 放火罪

#### 12.1.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 公共安全 |
| 客观方面 | 实施放火行为，危害公共安全 |
| 主体 | 一般主体（已满14周岁） |
| 主观方面 | 故意 |

#### 12.1.2 既遂标准

放火罪是危险犯，以足以危害公共安全为既遂标准，不要求造成实际损害。

#### 12.1.3 与故意杀人罪的区分

| 情形 | 定性 |
|------|------|
| 以放火方式杀人，危害公共安全 | 放火罪 |
| 以放火方式杀人，不危害公共安全 | 故意杀人罪 |

### 12.2 爆炸罪

#### 12.2.1 构成要件

- 客体：公共安全
- 客观方面：实施爆炸行为，危害公共安全
- 主体：一般主体（已满14周岁）
- 主观方面：故意

### 12.3 投放危险物质罪

#### 12.3.1 构成要件

- 客体：公共安全
- 客观方面：投放毒害性、放射性、传染病病原体等物质，危害公共安全
- 主体：一般主体（已满14周岁）
- 主观方面：故意

### 12.4 以危险方法危害公共安全罪

#### 12.4.1 构成要件

- 客体：公共安全
- 客观方面：以放火、决水、爆炸、投放危险物质以外的其他危险方法危害公共安全
- 主体：一般主体
- 主观方面：故意

**"其他危险方法"的认定**：
- 与放火、决水、爆炸、投放危险物质的危险性相当
- 一次性行为能够造成不特定多数人伤亡或重大财产损失

**典型情形**：
- 私设电网
- 驾车冲撞人群
- 在公共场所驾车撞人
- 向人群开枪

### 12.5 交通肇事罪

#### 12.5.1 构成要件

| 要件 | 内容 |
|------|------|
| 客体 | 交通运输安全 |
| 客观方面 | 违反交通运输管理法规，因而发生重大事故，致人重伤、死亡或者使公私财产遭受重大损失 |
| 主体 | 一般主体 |
| 主观方面 | 过失 |

#### 12.5.2 刑事责任

| 情节 | 刑罚 |
|------|------|
| 基本犯 | 3年以下有期徒刑或者拘役 |
| 交通运输肇事后逃逸 | 3年以上7年以下有期徒刑 |
| 因逃逸致人死亡 | 7年以上有期徒刑 |

#### 12.5.3 交通肇事罪的认定

**罪与非罪的界限**：
- 是否发生重大事故
- 是否违反交通运输管理法规
- 违章行为与事故之间是否有因果关系

**"逃逸"的认定**：
- 在发生交通事故后，为逃避法律追究而逃跑的行为
- "因逃逸致人死亡"：行为人在交通肇事后为逃避法律追究而逃跑，致使被害人因得不到救助而死亡

### 12.6 危险驾驶罪

#### 12.6.1 构成要件

**危险驾驶的行为类型**：

| 类型 | 内容 |
|------|------|
| 追逐竞驶 | 情节恶劣的 |
| 醉酒驾驶 | 醉酒驾驶机动车的 |
| 超员超速 | 从事校车业务或者旅客运输，严重超过额定乘员载客，或者严重超过规定时速行驶的 |
| 违规运输危险化学品 | 违反危险化学品安全管理规定运输危险化学品，危及公共安全的 |

#### 12.6.2 刑事责任

- 处拘役，并处罚金
- 同时构成其他犯罪的，依照处罚较重的规定定罪处罚

#### 12.6.3 醉酒驾驶的认定

- 血液酒精含量达到80mg/100ml以上
- 属于抽象危险犯，不要求造成实际危害结果

### 12.7 重大责任事故罪

#### 12.7.1 构成要件

- 客体：生产、作业安全
- 客观方面：在生产、作业中违反有关安全管理的规定，因而发生重大伤亡事故或者造成其他严重后果
- 主体：对生产、作业负有组织、指挥或者管理职责的负责人、管理人员、实际控制人、投资人等人员，以及直接从事生产、作业的人员
- 主观方面：过失

## 例题与解析

### 例题1

**题目**: 甲醉酒驾驶机动车在道路上行驶，被交警查获。经检测，甲血液酒精含量为100mg/100ml。甲的行为构成：

**选项**:
A. 交通肇事罪
B. 危险驾驶罪
C. 以危险方法危害公共安全罪
D. 不构成犯罪

**答案**: B

**解析**: 甲血液酒精含量达到80mg/100ml以上，属于醉酒驾驶，构成危险驾驶罪。危险驾驶罪是抽象危险犯，不要求造成实际危害结果。甲未发生交通事故，不构成交通肇事罪。

### 例题2

**题目**: 甲交通肇事后逃逸，致使被害人因得不到救助而死亡。甲的行为构成：

**选项**:
A. 交通肇事罪的基本犯
B. 交通肇事罪，交通运输肇事后逃逸
C. 交通肇事罪，因逃逸致人死亡
D. 故意杀人罪

**答案**: C

**解析**: 甲在交通肇事后为逃避法律追究而逃跑，致使被害人因得不到救助而死亡，属于"因逃逸致人死亡"，应处7年以上有期徒刑。如果甲在肇事后将被害人带离事故现场隐藏或者遗弃，致使被害人无法得到救助而死亡的，则以故意杀人罪定罪处罚。
