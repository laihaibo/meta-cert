# 第十章 刑罚的执行与消灭

## 本章概要

本章介绍刑罚执行的基本制度和刑罚消灭制度，包括缓刑、减刑、假释和时效。重点掌握缓刑的适用条件、减刑和假释的条件以及追诉时效的计算。

## 高频考点

> 🔥 高频考点

### 考点1: 缓刑
- 适用条件：被判处拘役或3年以下有期徒刑，犯罪情节较轻，有悔罪表现，没有再犯罪的危险，宣告缓刑对所居住社区没有重大不良影响
- 不适用缓刑：累犯和犯罪集团的首要分子

### 考点2: 减刑
- 适用对象：被判处管制、拘役、有期徒刑、无期徒刑的犯罪分子
- 减刑的限度：减刑后实际执行的刑期不能少于法定最低期限

### 考点3: 假释
- 适用条件：被判处有期徒刑执行原判刑期1/2以上，被判处无期徒刑实际执行13年以上
- 不得假释：累犯以及因故意杀人、强奸、抢劫、绑架、放火、爆炸、投放危险物质或者有组织的暴力性犯罪被判处10年以上有期徒刑、无期徒刑的犯罪分子

### 考点4: 追诉时效
- 法定最高刑不满5年有期徒刑的，经过5年
- 法定最高刑为5年以上不满10年有期徒刑的，经过10年
- 法定最高刑为10年以上有期徒刑的，经过15年
- 法定最高刑为无期徒刑、死刑的，经过20年

## 知识点详解

### 10.1 缓刑

#### 10.1.1 缓刑的概念

缓刑是指对被判处拘役、3年以下有期徒刑的犯罪分子，在符合法定条件的情况下，规定一定的考验期，暂缓其刑罚的执行，如果犯罪分子在考验期内没有发生法定撤销缓刑的情形，原判刑罚就不再执行的制度。

#### 10.1.2 缓刑的适用条件

| 条件 | 内容 |
|------|------|
| 刑度条件 | 被判处拘役或者3年以下有期徒刑 |
| 实质条件 | 犯罪情节较轻，有悔罪表现，没有再犯罪的危险，宣告缓刑对所居住社区没有重大不良影响 |
| 消极条件 | 不是累犯和犯罪集团的首要分子 |

#### 10.1.3 缓刑的考验期

| 刑种 | 考验期 |
|------|--------|
| 拘役 | 原判刑期以上1年以下，但是不能少于2个月 |
| 有期徒刑 | 原判刑期以上5年以下，但是不能少于1年 |

缓刑考验期从判决确定之日起计算。

#### 10.1.4 缓刑的撤销

**应当撤销缓刑的情形**：
- 在缓刑考验期限内犯新罪
- 在缓刑考验期限内发现判决宣告以前还有其他罪没有判决
- 在缓刑考验期限内，违反法律、行政法规或者国务院有关部门关于缓刑的监督管理规定，情节严重

**撤销后的处理**：
- 犯新罪或发现漏罪：对新罪或漏罪作出判决，与前罪数罪并罚
- 违反规定：执行原判刑罚

### 10.2 减刑

#### 10.2.1 减刑的概念

减刑是指对被判处管制、拘役、有期徒刑、无期徒刑的犯罪分子，在执行期间，如果认真遵守监规，接受教育改造，确有悔改表现或者立功表现的，适当减轻其原判刑罚的制度。

#### 10.2.2 减刑的条件

**可以减刑的条件**：
- 认真遵守监规
- 接受教育改造
- 确有悔改表现或者立功表现

**应当减刑的条件**：
- 有重大立功表现

#### 10.2.3 减刑的限度

| 原判刑罚 | 减刑后实际执行的最低刑期 |
|---------|----------------------|
| 有期徒刑 | 不能少于原判刑期的1/2 |
| 无期徒刑 | 不能少于13年 |
| 死缓减为无期徒刑 | 不能少于25年 |
| 死缓减为25年有期徒刑 | 不能少于25年 |

#### 10.2.4 减刑的程序

- 由执行机关向中级以上人民法院提出减刑建议书
- 人民法院组成合议庭进行审理
- 对确有悔改或者立功事实的，裁定予以减刑

### 10.3 假释

#### 10.3.1 假释的概念

假释是指对被判处有期徒刑、无期徒刑的犯罪分子，在执行一定刑期之后，如果认真遵守监规，接受教育改造，确有悔改表现，没有再犯罪的危险的，附条件地将其提前释放的制度。

#### 10.3.2 假释的适用条件

| 条件 | 内容 |
|------|------|
| 对象条件 | 被判处有期徒刑、无期徒刑的犯罪分子 |
| 刑期条件 | 有期徒刑执行原判刑期1/2以上，无期徒刑实际执行13年以上 |
| 实质条件 | 认真遵守监规，接受教育改造，确有悔改表现，没有再犯罪的危险 |
| 消极条件 | 累犯以及因故意杀人、强奸、抢劫、绑架、放火、爆炸、投放危险物质或者有组织的暴力性犯罪被判处10年以上有期徒刑、无期徒刑的犯罪分子，不得假释 |

#### 10.3.3 假释的考验期

| 原判刑罚 | 考验期 |
|---------|--------|
| 有期徒刑 | 没有执行完毕的刑期 |
| 无期徒刑 | 10年 |

假释考验期从假释之日起计算。

#### 10.3.4 假释的撤销

**应当撤销假释的情形**：
- 在假释考验期限内犯新罪
- 在假释考验期限内发现判决宣告以前还有其他罪没有判决
- 在假释考验期限内，有违反法律、行政法规或者国务院有关部门关于假释的监督管理规定的行为

**撤销后的处理**：
- 犯新罪：对新罪作出判决，与前罪未执行完毕的刑罚数罪并罚
- 发现漏罪：对漏罪作出判决，与前罪数罪并罚
- 违反规定：收监执行未执行完毕的刑罚

### 10.4 时效

#### 10.4.1 追诉时效

**追诉时效的期限**：

| 法定最高刑 | 追诉期限 |
|-----------|---------|
| 不满5年有期徒刑 | 5年 |
| 5年以上不满10年有期徒刑 | 10年 |
| 10年以上有期徒刑 | 15年 |
| 无期徒刑、死刑 | 20年（超过20年须追诉的，报最高人民检察院核准） |

**追诉期限的计算**：
- 从犯罪之日起计算
- 犯罪行为有连续或者继续状态的，从犯罪行为终了之日起计算

**追诉时效的中断**：
- 在追诉期限以内又犯罪的，前罪追诉的期限从犯后罪之日起计算

**追诉时效的延长**：
- 在人民检察院、公安机关、国家安全机关立案侦查或者在人民法院受理案件以后，逃避侦查或者审判的，不受追诉期限的限制
- 被害人在追诉期限内提出控告，人民法院、人民检察院、公安机关应当立案而不予立案的，不受追诉期限的限制

#### 10.4.2 赦免

**赦免的类型**：
- 大赦：赦免特定犯罪或一般犯罪的全部刑罚
- 特赦：赦免特定罪犯的全部或部分刑罚

## 例题与解析

### 例题1

**题目**: 甲因盗窃罪被判处有期徒刑3年，缓刑5年。在缓刑考验期内，甲又犯故意伤害罪。下列说法正确的是：

**选项**:
A. 应当撤销缓刑，对故意伤害罪作出判决，与盗窃罪数罪并罚
B. 应当撤销缓刑，执行原判刑罚3年
C. 不撤销缓刑，对故意伤害罪单独处罚
D. 应当延长缓刑考验期

**答案**: A

**解析**: 根据《刑法》第77条，被宣告缓刑的犯罪分子，在缓刑考验期限内犯新罪的，应当撤销缓刑，对新犯的罪作出判决，把前罪和后罪所判处的刑罚，依照数罪并罚的规定，决定执行的刑罚。

### 例题2

**题目**: 甲因抢劫罪被判处无期徒刑，执行10年后被假释。假释考验期第3年，甲又犯盗窃罪。下列说法正确的是：

**选项**:
A. 应当撤销假释，对盗窃罪作出判决，与无期徒刑数罪并罚
B. 应当撤销假释，收监执行未执行完毕的刑罚
C. 应当撤销假释，对盗窃罪作出判决，与未执行完毕的刑罚数罪并罚
D. 不撤销假释，对盗窃罪单独处罚

**答案**: C

**解析**: 根据《刑法》第86条，在假释考验期限内犯新罪的，应当撤销假释，对新犯的罪作出判决，把前罪没有执行的刑罚和后罪所判处的刑罚，依照数罪并罚的规定，决定执行的刑罚。无期徒刑假释考验期为10年，甲在考验期内犯新罪，应当撤销假释。
