# 习近平法治思想

## 科目概述

习近平法治思想是全面依法治国的根本遵循和行动指南，是法考公共科目的重要组成部分。本部分系统阐述习近平法治思想的核心要义、理论体系和实践要求。

## 考情分析

- 分值占比：约10-15分
- 题型：以选择题为主，偶有论述题
- 难度：中等，重在理解与记忆
