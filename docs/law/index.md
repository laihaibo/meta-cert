# 法律职业资格考试（客观题）

法律职业资格考试是国家统一法律职业资格考试，是初任法官、初任检察官，申请律师执业、公证员执业和初次担任法律类仲裁员，以及行政机关中初次从事行政处罚决定法制审核、行政复议、行政裁决的公务员的资格考试。

## 备考建议

1. **重点突破** — 刑法、民法分值最高，优先学习
2. **体系掌握** — 法理学和宪法是理解其他法律的基础
3. **刷题巩固** — 每科配套题库练习，关注高频考点
4. **对比记忆** — 刑诉与民诉、行政法与行政诉讼法对比学习
5. **定期复习** — 法律史、国际法等记忆型科目需要反复巩固
