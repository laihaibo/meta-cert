# 一级/高级技师 · 人工智能训练师

> 职业定位：在人工智能产品实际使用过程中，负责业务战略规划、指标体系设计、算法创新应用、团队管理与项目落地的高级技术管理人才。

## 学习建议

一级/高级技师是人工智能训练师的最高级别，要求具备战略思维、业务创新能力和团队管理能力。学习时需：

1. **把握全局** — 从业务战略高度理解 AI 产品全生命周期
2. **融会贯通** — 将前面四级的内容整合应用于复杂场景
3. **关注前沿** — 了解大模型、多模态等最新技术趋势
4. **强化实操** — 高级技师考核偏重方案设计和团队协作能力
