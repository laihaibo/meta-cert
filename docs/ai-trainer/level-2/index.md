# 二级/技师 · 人工智能训练师

> 定位：业务分析与模型优化 | 重要程度：★★★★★
> 参考学时：40学时 | 及格线：60分

## 考情分析

二级/技师是人工智能训练师的中高级别，要求考生在掌握基础数据处理与模型训练能力的基础上，进一步具备业务分析与设计、模型优化与评估的能力。考试对实操能力要求较高，重点考查业务流程梳理、指标体系设计、模型评估与调优等综合能力。

| 考试模块 | 分值占比 | 考试形式 |
|----------|----------|----------|
| 职业道德与法律法规 | 约10% | 选择题 |
| 业务分析与设计 | 约35% | 选择题 + 案例分析 |
| 模型优化与评估 | 约35% | 选择题 + 实操 |
| 项目管理与协作 | 约20% | 选择题 |

## 备考建议

1. **理解为主** — 业务分析与模型优化章节概念较多，需在理解的基础上记忆
2. **案例驱动** — 结合实际项目案例学习，理解理论在实践中的应用
3. **指标体系** — 重点掌握常见业务场景的指标体系设计方法
4. **实操练习** — 模型评估与调优需结合实际代码练习，加深理解
