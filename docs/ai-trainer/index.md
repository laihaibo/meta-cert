# 人工智能训练师

> 使用智能训练软件，在人工智能产品实际使用过程中进行数据库管理、算法参数设置、人机交互设计、性能测试跟踪及其他辅助作业。

## 考试概述

| 项目 | 详情 |
|------|------|
| 职业名称 | 人工智能训练师 |
| 级别设置 | 五级/初级工、四级/中级工、三级/高级工、二级/技师、一级/高级技师 |
| 考试形式 | 理论知识考试（计算机在线作答）+ 操作技能考核 |
| 及格线 | 60分及以上 |
| 参考学时 | 五级60学时、四级50学时、三级40学时、二级40学时、一级30学时 |

## 级别与章节

| 级别 | 定位 | 核心章节 | 重要程度 |
|------|------|----------|----------|
| [五级/初级工](./level-5/) | 数据标注基础 | 职业道德、数据标注基础、通用工具操作 | ★★★ |
| [四级/中级工](./level-4/) | 数据处理与标注 | 职业道德、数据采集与清洗、数据标注方法、通用知识 | ★★★★ |
| [三级/高级工](./level-3/) | 模型训练与调优 | 职业道德、智能训练基础、模型训练、模型评估 | ★★★★ |
| [二级/技师](./level-2/) | 业务分析与模型优化 | 职业道德、业务分析设计、模型优化、项目管理 | ★★★★★ |
| [一级/高级技师](./level-1/) | 战略规划与落地 | 职业道德、业务战略、模型创新、团队管理 | ★★★★★ |

## 备考建议

1. **循序渐进** — 从五级开始逐步提升，每个级别的内容都是下一级的基础
2. **重视实操** — 操作技能考核占比较大，需结合实践工具（Label Studio、Python等）进行练习
3. **关注大纲** — 依据官方考试大纲复习，重点掌握各级别核心知识点
4. **真题训练** — 每章学完后完成配套练习题，巩固知识点
