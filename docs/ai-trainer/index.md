# 人工智能训练师

> 使用智能训练软件，在人工智能产品实际使用过程中进行数据库管理、算法参数设置、人机交互设计、性能测试跟踪及其他辅助作业。

## 考试概述

| 项目 | 详情 |
|------|------|
| 职业名称 | 人工智能训练师 |
| 级别 | 三级/高级工 |
| 考试形式 | 理论知识考试（计算机在线作答）+ 操作技能考核 |
| 及格线 | 60分及以上 |
| 参考学时 | 40学时 |

## 级别与章节

| 级别 | 定位 | 核心章节 | 重要程度 |
|------|------|----------|----------|
| [三级/高级工](./) | 模型训练与调优 | 职业道德、智能训练基础、模型训练、模型评估 | ★★★★ |

## 备考建议

1. **夯实基础** — 重点掌握智能训练基础、模型训练流程与调优方法
2. **重视实操** — 操作技能考核占比较大，需结合实践工具（Label Studio、Python等）进行练习
3. **关注大纲** — 依据官方考试大纲复习，重点掌握各级别核心知识点
4. **真题训练** — 每章学完后完成配套练习题，巩固知识点
