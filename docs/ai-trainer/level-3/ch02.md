## 2.1 数据处理规范

### 2.1.1 数据预处理流程

**标准预处理流程：**

```
原始数据 → 数据清洗 → 数据标注 → 特征工程 → 数据集划分 → 训练数据
```

**各环节要点：**

| 环节 | 主要工作 | 关键要点 |
|------|----------|----------|
| 数据清洗 | 去重、去噪、补全缺失值 | 处理异常值、格式统一 |
| 数据标注 | 分类标注、实体标注、关系标注 | 标注规范、一致性校验 |
| 特征工程 | 特征提取、特征选择、特征变换 | 降维、归一化、标准化 |
| 数据集划分 | 训练集、验证集、测试集 | 比例合理、分布一致 |

### 2.1.2 数据清洗方法

**常见数据问题及处理方法：**

| 问题类型 | 识别方法 | 处理方法 |
|----------|----------|----------|
| 缺失值 | 统计空值比例 | 删除、均值/中位数填充、模型预测填充 |
| 异常值 | 箱线图、Z-score | 删除、修正、分箱处理 |
| 重复数据 | 唯一性校验 | 去重保留一条 |
| 格式不一致 | 数据探查 | 统一格式、正则匹配 |

**数据质量评价指标：**
- 完整性：缺失值比例
- 一致性：数据格式统一程度
- 准确性：数据与真实情况吻合程度
- 时效性：数据的新鲜程度

### 2.1.3 数据集划分原则

**常用划分比例：**
- 训练集：60%-80%
- 验证集：10%-20%
- 测试集：10%-20%

**划分方法：**
- 随机划分：适用于数据分布均匀的情况
- 分层划分：按类别比例划分，保持分布一致性
- 时间划分：按时间顺序划分，适用于时序数据

::: tip 高频考点
1. 数据预处理标准流程：清洗 → 标注 → 特征工程 → 划分
2. 缺失值处理方法：删除、填充、预测
3. 数据集划分比例：训练集60%-80%，验证集10%-20%，测试集10%-20%
4. 分层划分可保持类别分布一致性
:::

## 2.2 算法测试评价

### 2.2.1 算法评价指标

**分类任务指标：**

| 指标 | 公式 | 说明 |
|------|------|------|
| 准确率（Accuracy） | (TP+TN)/(TP+FP+TN+FN) | 预测正确的样本占比 |
| 精确率（Precision） | TP/(TP+FP) | 预测为正例中实际为正例的比例 |
| 召回率（Recall） | TP/(TP+FN) | 实际为正例中被预测正确的比例 |
| F1-score | 2×Precision×Recall/(Precision+Recall) | 精确率与召回率的调和平均 |
| AUC-ROC | ROC曲线下面积 | 衡量分类器综合性能 |

**混淆矩阵：**

```
                预测正例    预测负例
实际正例    TP（真正例）  FN（假负例）
实际负例    FP（假正例）  TN（真负例）
```

### 2.2.2 回归任务指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 均方误差（MSE） | Σ(y-ŷ)²/n | 预测值与真实值差平方的均值 |
| 均方根误差（RMSE） | √MSE | MSE的平方根，量纲与原始数据一致 |
| 平均绝对误差（MAE） | Σ|y-ŷ|/n | 预测值与真实值绝对差的均值 |
| R²（决定系数） | 1-SSres/SStot | 模型解释方差的比例 |

### 2.2.3 交叉验证

**K折交叉验证：**
- 将数据集分为K个子集
- 每次用K-1个子集训练，1个子集验证
- 重复K次，取平均性能

**常用K值：**
- K=5：常用选择，平衡计算量与稳定性
- K=10：更稳定，计算量较大
- 留一法（LOOCV）：K等于样本数，计算量最大

::: warning 注意事项
- 测试集只能用于最终评估，不能用于调参
- 验证集用于调参和模型选择
- 交叉验证可减少因数据划分不同带来的性能波动
- 类别不平衡时，准确率不能全面反映模型性能
:::

## 2.3 深度学习基础

### 2.3.1 神经网络基本结构

**基本组成：**
- **输入层**：接收特征输入
- **隐藏层**：进行特征变换与抽象
- **输出层**：输出预测结果

**关键概念：**
- **神经元**：基本计算单元，接收输入、加权求和、激活输出
- **权重（Weight）**：连接强度参数，通过训练更新
- **偏置（Bias）**：调节神经元激活阈值
- **激活函数**：引入非线性，增强模型表达能力

### 2.3.2 常见激活函数

| 激活函数 | 公式 | 特点 |
|----------|------|------|
| Sigmoid | f(x)=1/(1+e^-x) | 输出0-1，易饱和 |
| Tanh | f(x)=(e^x-e^-x)/(e^x+e^-x) | 输出-1到1，零中心 |
| ReLU | f(x)=max(0,x) | 计算简单，缓解梯度消失 |
| Leaky ReLU | f(x)=max(0.01x,x) | 解决ReLU死亡问题 |
| Softmax | f(x_i)=e^x_i/Σe^x_j | 多分类输出，概率分布 |

### 2.3.3 前向传播与反向传播

**前向传播：**
- 输入数据从输入层逐层传递到输出层
- 每层计算：输出 = 激活函数(权重·输入 + 偏置)
- 最终得到预测值

**反向传播（Backpropagation）：**
- 计算输出层损失相对于权重的梯度
- 从输出层反向传播梯度到各隐藏层
- 利用链式法则逐层计算梯度

**权重更新：**
- 新权重 = 旧权重 - 学习率 × 梯度

### 2.3.4 过拟合与欠拟合

**过拟合（Overfitting）：**
- 训练集表现好，验证/测试集表现差
- 原因：模型复杂度过高、数据量不足
- 解决方法：增加数据、正则化、Dropout、早停

**欠拟合（Underfitting）：**
- 训练集和验证集表现都差
- 原因：模型复杂度过低、特征不足
- 解决方法：增加模型复杂度、特征工程、减少正则化

**偏差-方差权衡：**
- 总误差 = 偏差² + 方差 + 噪声
- 目标：找到偏差与方差的最佳平衡点

## 2.4 特征工程基础

### 2.4.1 特征类型

**数值特征：**
- 连续值：身高、体重、温度
- 离散值：次数、人数

**类别特征：**
- 有序类别：学历等级（小学、初中、高中）
- 无序类别：颜色（红、蓝、绿）

**文本特征：**
- 词袋模型（Bag of Words）
- TF-IDF
- Word2Vec、BERT词向量

### 2.4.2 特征变换方法

| 方法 | 适用场景 | 说明 |
|------|----------|------|
| 标准化（Z-score） | 特征分布接近正态 | 均值为0，方差为1 |
| 归一化（Min-Max） | 特征范围差异大 | 缩放到[0,1]区间 |
| 对数变换 | 右偏分布 | 压缩大值，拉伸小值 |
| 独热编码（One-Hot） | 无序类别特征 | 每个类别对应一维 |
| 标签编码（Label） | 有序类别特征 | 映射为整数 |

### 2.4.3 特征选择

**过滤法：**
- 基于统计指标选择特征
- 方差过滤、相关系数、卡方检验
- 优点：快速，不依赖模型
- 缺点：忽略特征间交互

**包裹法：**
- 通过模型性能选择特征
- 递归特征消除（RFE）
- 优点：考虑特征组合
- 缺点：计算量大

**嵌入法：**
- 训练过程中自动选择特征
- L1正则化、树模型特征重要性
- 优点：效率与效果平衡
