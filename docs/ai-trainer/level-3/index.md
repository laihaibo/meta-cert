# 三级/高级工

> 定位：模型训练与调优 | 重要程度：★★★★

## 考情分析

三级/高级工是人工智能训练师的中高级阶段，重点考查智能训练基础、模型训练流程、超参数调整、损失函数与优化器等核心能力。考试既包含职业道德与法律法规，也覆盖算法测试评价、深度学习基础、训练流程实操等内容，适合具备一定数据处理基础、希望向模型训练方向发展的考生。

| 项目 | 详情 |
|------|------|
| 职业名称 | 人工智能训练师 |
| 级别 | 三级/高级工 |
| 核心章节 | 职业道德与法律法规、智能训练基础、模型训练与调优 |
| 重要程度 | ★★★★ |
| 参考学时 | 40学时 |

## 章节概览

| 章节 | 名称 | 重点 |
|------|------|------|
| 第一章 | 职业道德与法律法规 | 职业守则、数据安全、知识产权、相关法律法规 |
| 第二章 | 智能训练基础 | 数据处理规范、算法测试评价、深度学习基础 |
| 第三章 | 模型训练与调优 | 训练流程、超参数调整、损失函数、优化器 |

## 备考建议

1. **夯实基础** — 熟练掌握数据处理规范和深度学习基础概念
2. **理解流程** — 深入理解模型训练全流程及各环节作用
3. **辨析概念** — 重点区分各类损失函数与优化器的适用场景
4. **实操结合** — 通过实际项目加深对超参数调整、模型评估的理解
5. **合规意识** — 牢记职业道德与数据安全相关法规要求
