## 公式速查

| 公式名称 | 公式 |
|----------|------|
| 准确率 | Accuracy = (TP+TN)/(TP+FP+TN+FN) |
| 精确率 | Precision = TP/(TP+FP) |
| 召回率 | Recall = TP/(TP+FN) |
| F1-score | F1 = 2×P×R/(P+R) |
| MSE | MSE = Σ(y-ŷ)²/n |
| RMSE | RMSE = √MSE |
| R² | R² = 1 - SSres/SStot |
| 参数更新 | θ = θ - η × ∇J(θ) |
| 标准化 | Z = (X-μ)/σ |
| 归一化 | X' = (X-X_min)/(X_max-X_min) |
| L2正则化 | Loss = 原损失 + λ × Σw² |
| L1正则化 | Loss = 原损失 + λ × Σ|w| |
| 梯度爆炸判断 | 梯度范数 > 阈值（如10） |
| 梯度消失判断 | 梯度范数 < 阈值（如1e-7） |

## 重点概念辨析

| 概念对 | 区别 | 联系 |
|--------|------|------|
| 过拟合 vs 欠拟合 | 过拟合：训练好验证差；欠拟合：训练验证都差 | 都是模型泛化能力不足的表现 |
| 偏差 vs 方差 | 偏差：模型预测与真实的差距；方差：预测的波动 | 总误差 = 偏差² + 方差 + 噪声 |
| L1 vs L2正则化 | L1稀疏，L2平滑 | 都用于防止过拟合 |
| Adam vs SGD | Adam自适应，SGD统一 | Adam收敛快，SGD泛化可能更好 |
| 训练集 vs 验证集 | 训练集学习参数；验证集调参 | 都用训练数据划分 |
| 验证集 vs 测试集 | 验证集调参；测试集最终评估 | 测试集只能用一次 |
