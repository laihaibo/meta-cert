# 人工智能训练师（五级/初级工）

> 职业技能等级认定 · 初级

## 考试概述

| 项目 | 详情 |
|------|------|
| 职业名称 | 人工智能训练师 |
| 等级 | 五级 / 初级工 |
| 考试形式 | 理论知识（闭卷笔试 / 机考） |
| 题型 | 单项选择题、判断题 |
| 及格线 | 60分（满分100分） |
| 考试时间 | 120分钟 |

## 章节结构与分值分布

| 章节 | 主要内容 | 分值占比 | 重要程度 |
|------|---------|----------|----------|
| [第一章](./ch01) | 职业道德与法律法规 | 5%-10% | ★★ |
| [第二章](./ch02) | 数据标注基础 | 30%-35% | ★★★★★ |
| [第三章](./ch03) | 通用工具操作 | 30%-35% | ★★★★★ |

## 必背考点速记

| 类型 | 考点 | 关键内容 |
|------|------|---------|
| 概念 | 数据标注 | 对原始数据进行分类、标记、转写等预处理 |
| 概念 | 标注类型 | 文本分类、图像标注、语音转写、实体识别 |
| 规范 | 标注一致性 | 同一标注员对不同样本标注结果应一致 |
| 操作 | Office基础 | Word文档编辑、Excel数据录入、PPT制作 |
| 操作 | 文件管理 | 文件夹创建、文件重命名、路径查找 |
| 法律 | 职业道德 | 诚实守信、保守秘密、公平竞争 |
| 法律 | 数据安全 | 不得泄露训练数据，遵守数据保护法规 |

## 备考建议

1. **重点章节**：第二章（数据标注基础）和第三章（通用工具操作）分值最高，务必重点掌握实际操作技能
2. **理论题**：侧重职业道德、数据标注概念、标注规范等基础概念的记忆
3. **实操题**：重点练习Office软件操作、文件管理、浏览器使用等通用技能
4. **标注规范**：熟记数据标注的基本流程和一致性要求
5. **时间管理**：考试时间较充裕，建议先做实操题再做理论题
