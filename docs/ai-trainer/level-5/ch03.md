> 分值占比：30%-35% | 重要程度：★★★★★

## 考情分析

本章是人工智能训练师五级的核心实操章节，主要介绍日常工作中常用的Office软件操作、操作系统基础知识和浏览器使用。考试重点考查实际操作能力，包括文档编辑、数据处理、文件管理等实用技能。

## 3.1 Office操作

### 3.1.1 Word文档操作

**基本操作：**

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 新建文档 | Ctrl+N | 创建空白文档 |
| 打开文档 | Ctrl+O | 打开已有文档 |
| 保存文档 | Ctrl+S | 保存当前文档 |
| 另存为 | F12 | 另存为新文件 |
| 复制 | Ctrl+C | 复制选中内容 |
| 粘贴 | Ctrl+V | 粘贴复制内容 |
| 剪切 | Ctrl+X | 剪切选中内容 |
| 撤销 | Ctrl+Z | 撤销上一步操作 |
| 查找 | Ctrl+F | 查找文档内容 |
| 替换 | Ctrl+H | 替换文档内容 |

**常用编辑功能：**
- 字体设置：字体、字号、加粗、斜体、下划线
- 段落设置：对齐方式、行距、段间距
- 页面设置：页边距、纸张大小、页眉页脚
- 插入功能：表格、图片、形状、超链接

### 3.1.2 Excel数据处理

**基本操作：**

| 操作 | 说明 |
|------|------|
| 单元格选择 | 单击选择单个单元格，拖动选择区域 |
| 行列操作 | 右键插入/删除行或列 |
| 公式输入 | 以"="开头输入公式 |
| 常用函数 | SUM（求和）、AVERAGE（平均）、MAX（最大）、MIN（最小） |
| 数据排序 | 选中数据区域，点击排序按钮 |
| 数据筛选 | 启用筛选功能，按条件筛选数据 |

**常用函数示例：**
- `=SUM(A1:A10)`：计算A1到A10单元格的和
- `=AVERAGE(B1:B5)`：计算B1到B5单元格的平均值
- `=MAX(C1:C10)`：找出C1到C10中的最大值
- `=IF(D1>60,"及格","不及格")`：条件判断

### 3.1.3 PPT制作

**基本操作：**
- 新建演示文稿：选择空白演示文稿或模板
- 添加幻灯片：右键或"开始"选项卡中新建幻灯片
- 添加内容：标题、正文、图片、图表、表格
- 设置动画：为元素添加入场、退出、强调动画
- 设置切换：为幻灯片添加切换效果
- 放映幻灯片：F5从头放映，Shift+F5从当前页放映

## 3.2 操作系统基础

### 3.2.1 文件与文件夹管理

**基本概念：**
- 文件：存储数据的基本单位，有文件名和扩展名
- 文件夹（目录）：用于组织和管理文件的容器
- 路径：文件在计算机中的位置标识

**常见文件类型：**

| 扩展名 | 文件类型 | 说明 |
|--------|---------|------|
| .txt | 文本文件 | 纯文本内容 |
| .doc/.docx | Word文档 | 微软Word格式 |
| .xls/.xlsx | Excel表格 | 微软Excel格式 |
| .ppt/.pptx | PPT演示 | 微软PowerPoint格式 |
| .jpg/.png | 图片文件 | 图像文件 |
| .pdf | PDF文档 | 便携式文档格式 |
| .zip | 压缩文件 | 压缩归档文件 |

**文件操作：**
- 创建文件夹：右键→新建→文件夹
- 重命名文件：右键→重命名 或 F2
- 复制/移动文件：Ctrl+C/V/X 或 拖动（配合Ctrl/SHIFT）
- 删除文件：选中后按Delete键
- 查找文件：搜索框输入文件名

### 3.2.2 系统常用操作

**Windows系统：**

| 操作 | 快捷键/方法 | 说明 |
|------|------------|------|
| 打开开始菜单 | Win键或Ctrl+Esc | 打开Windows开始菜单 |
| 运行程序 | Win+R | 打开"运行"对话框 |
| 切换窗口 | Alt+Tab | 在打开的窗口间切换 |
| 任务管理器 | Ctrl+Shift+Esc | 查看和管理运行中的程序 |
| 截图 | Win+Shift+S | 区域截图 |
| 锁屏 | Win+L | 锁定计算机 |

**常用系统设置：**
- 显示设置：分辨率、亮度、缩放比例
- 网络设置：Wi-Fi连接、有线网络配置
- 声音设置：音量调节、音频设备选择
- 时间设置：时区、日期时间格式

## 3.3 浏览器使用

### 3.3.1 浏览器基本操作

**常用浏览器：**
- Chrome（谷歌浏览器）
- Edge（微软浏览器）
- Firefox（火狐浏览器）
- Safari（苹果浏览器）

**基本操作：**

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 打开新标签页 | Ctrl+T | 新建空白标签页 |
| 关闭当前标签页 | Ctrl+W | 关闭当前标签 |
| 切换标签页 | Ctrl+Tab | 切换到下一个标签 |
| 打开新窗口 | Ctrl+N | 新建浏览器窗口 |
| 前进 | Alt+Right | 前进到下一页 |
| 后退 | Alt+Left | 后退到上一页 |
| 刷新页面 | F5 或 Ctrl+R | 重新加载当前页 |
| 强制刷新 | Ctrl+F5 | 忽略缓存刷新页面 |
| 打开网址 | Ctrl+L | 选中地址栏输入网址 |
| 收藏网页 | Ctrl+D | 将当前页添加到收藏夹 |
| 查找页面内容 | Ctrl+F | 在当前页查找文字 |
| 打开开发者工具 | F12 | 查看网页源代码和调试信息 |

### 3.3.2 浏览器使用技巧

**标签页管理：**
- 右键标签页可以固定、静音、复制标签页
- 使用Ctrl+数字键可快速切换到第N个标签页
- 浏览器可设置标签页分组管理

**书签管理：**
- 将常用网站添加到书签栏
- 可以创建文件夹分类管理书签
- 导入/导出书签进行备份

**扩展插件：**
- 广告拦截：提高浏览体验
- 密码管理：安全存储密码
- 截图工具：快速截取网页内容
- 翻译插件：网页内容翻译

::: warning 易混淆点
- Ctrl+S 保存 vs Ctrl+Shift+S 另存为：保存是覆盖原文件，另存为创建新文件
- Alt+Tab 切换窗口 vs Ctrl+Tab 切换标签：前者切换程序窗口，后者切换浏览器标签
- Delete 删除 vs Shift+Delete 永久删除：前者放入回收站，后者直接删除
- F5 刷新 vs Ctrl+F5 强制刷新：前者使用缓存，后者忽略缓存
:::
