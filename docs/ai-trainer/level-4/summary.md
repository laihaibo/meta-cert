## 第二章：数据采集与清洗

### 核心考点

| 序号 | 考点 | 重要程度 | 备注 |
|------|------|----------|------|
| 1 | 缺失值处理方法（删除、均值/中位数/众数填充、模型预测） | ★★★★★ | 必考，区分适用场景 |
| 2 | 重复值去重方法 | ★★★★ | drop_duplicates，主键去重 |
| 3 | 异常值检测：IQR 法 | ★★★★★ | Q1-1.5IQR ~ Q3+1.5IQR |
| 4 | 数据质量六维度 | ★★★★ | 完整性、准确性、一致性、时效性、唯一性、有效性 |
| 5 | 数据质量评估指标 | ★★★★ | 完整率、准确率、一致率 |
| 6 | 抽样检验方法 | ★★★ | 简单随机 vs 分层抽样 |
| 7 | 数据存储格式选择 | ★★★ | CSV/JSON/Parquet 适用场景 |
| 8 | 数据清洗流程 | ★★★★★ | 加载→缺失→重复→异常→标准化 |

### 公式速记

| 公式 | 含义 |
|------|------|
| IQR = Q3 - Q1 | 四分位距 |
| 下界 = Q1 - 1.5 × IQR | 异常值检测下界 |
| 上界 = Q3 + 1.5 × IQR | 异常值检测上界 |
| 数据完整率 = 完整记录数 / 总记录数 × 100% | 数据完整性指标 |

### 关键词记忆

| 关键词 | 对应考点 |
|--------|----------|
| MCAR/MAR/MNAR | 缺失值三种类型 |
| IQR | 四分位距，异常值检测核心指标 |
| fillna | Pandas 缺失值填充方法 |
| drop_duplicates | Pandas 去重方法 |
| 分层抽样 | 类别不平衡场景首选 |

## 第三章：数据标注方法

### 核心考点

| 序号 | 考点 | 重要程度 | 备注 |
|------|------|----------|------|
| 1 | IoU 计算与含义 | ★★★★★ | 目标检测核心指标 |
| 2 | mAP 计算原理 | ★★★★ | 目标检测评估指标 |
| 3 | mIoU 计算原理 | ★★★★ | 语义分割核心指标 |
| 4 | Precision / Recall / F1 | ★★★★★ | 所有标注类型的通用指标 |
| 5 | NER 标注方案（BIO） | ★★★★ | B-PER、I-PER、O |
| 6 | 混淆矩阵 | ★★★★ | TP/FP/FN/TN |
| 7 | Cohen's Kappa | ★★★ | 标注一致性评估 |
| 8 | 三审质量控制流程 | ★★★ | 自检→互检→终审 |
| 9 | 标注工具选择 | ★★★ | Label Studio / CVAT |

### 公式速记

| 公式 | 含义 |
|------|------|
| IoU = 交集面积 / 并集面积 | 目标检测框匹配度 |
| Precision = TP / (TP + FP) | 准确率 |
| Recall = TP / (TP + FN) | 召回率 |
| F1 = 2 × P × R / (P + R) | F1 分数 |
| mIoU = Σ IoU_i / C | 语义分割平均 IoU |
| Kappa = (Po - Pe) / (1 - Pe) | 标注一致性系数 |

### 关键词记忆

| 关键词 | 对应考点 |
|--------|----------|
| B-PER / I-PER / O | NER 标注标签体系 |
| mAP | 目标检测评估指标 |
| mIoU | 语义分割评估指标 |
| 混淆矩阵 | TP/FP/FN/TN 四格表 |
| 三审流程 | 自检→互检→终审 |

### 常见陷阱

| 陷阱 | 正确做法 |
|------|----------|
| 把缺失值直接删除而不考虑缺失率 | 先评估缺失率，<5% 且样本充足才考虑删除 |
| 混淆 IQR 上下界公式 | 下界 = Q1 - 1.5×IQR，上界 = Q3 + 1.5×IQR |
| 把准确率当召回率 | Precision 关注"预测对了多少"，Recall 关注"找全了多少" |
| BIO 标注时 I 接 B | 同一实体的连续 token 用 B 开头，后续用 I |
| 爬虫采集不遵守 robots.txt | 必须遵守，否则违规 |
