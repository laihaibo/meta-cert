# 软件设计师

> 计算机技术与软件专业技术资格（软考·中级）

## 考试概述

| 项目 | 详情 |
|------|------|
| 科目名称 | 软件设计师 |
| 题型 | 单项选择题（75题）+ 问答题 |
| 总分 | 75分（上午选择题）+ 75分（下午案例分析） |
| 及格线 | 45分（上午+下午均需及格） |
| 考试时间 | 上午120分钟，下午120分钟 |

| 章节 | 主要内容 | 分值占比 | 重要程度 |

| [第二章](./ch02) | 程序设计语言基础 | 5%-8% | ★★★ |
| [第三章](./ch03) | 数据结构 | 10%-12% | ★★★★ |
| [第四章](./ch04) | 操作系统基础 | 5%-8% | ★★★ |
| [第五章](./ch05) | 软件工程 | 10%-12% | ★★★★ |
| [第六章](./ch06) | 数据库系统基础 | 5%-8% | ★★★ |
| [第七章](./ch07) | 计算机网络基础 | 5%-8% | ★★★ |

| [第九章](./ch09) | 标准化与知识产权 | 5%-8% | ★★★ |

## 必背数字考点

| 类型 | 考点 | 数值 |
|------|------|------|
| 时间 | 时间片轮转调度 | 时间片大小决定 |
| 比例 | 缓存命中率 | 越高越好 |
| 比例 | 代码覆盖率 | 建议≥80% |
| 级别 | 软件能力成熟度模型（CMMI） | 5级 |
| 代码 | 极限编程（XP）核心实践 | 12种 |
| 协议 | TCP三次握手 | 3次 |
| 地址 | IPv4地址长度 | 32位 |
| 地址 | IPv6地址长度 | 128位 |

## 备考建议

1. **重点章节**：第三章（数据结构）、第五章（软件工程）分值最高，务必重点掌握
2. **上午题**：侧重基础知识记忆，注意计算机系统、操作系统、数据结构等概念题
3. **下午题**：侧重案例分析，掌握软件设计流程、UML建模、数据流图等
4. **编程题**：C语言或C++算法题，重点掌握常用排序、查找算法
5. **时间管理**：上午题时间较紧，建议每题不超过1.5分钟
