# 第五章 基金与衍生工具

## 本章概要

本章是重点章节，介绍证券投资基金和金融衍生工具。重点掌握基金的分类、估值方法，以及期货、期权、互换等衍生工具的特点和基本策略。分值占比15%-20%。

## 高频考点

> :fire: 高频考点

### 考点1: 基金的分类

**按运作方式**：
| 类型 | 特点 |
|------|------|
| 开放式基金 | 可随时申购赎回，规模不固定 |
| 封闭式基金 | 封闭期内不能赎回，规模固定 |

**按投资对象**：
| 类型 | 投资对象 |
|------|---------|
| 股票基金 | 80%以上投资于股票 |
| 债券基金 | 80%以上投资于债券 |
| 货币市场基金 | 投资于货币市场工具 |
| 混合基金 | 投资于股票、债券等 |
| 基金中基金（FOF） | 投资于其他基金 |

**按募集方式**：
- 公募基金：面向社会公众
- 私募基金：面向合格投资者

### 考点2: 基金估值

**基金资产净值（NAV）**：
$$NAV = 基金总资产 - 基金总负债$$

**基金份额净值**：
$$基金份额净值 = \frac{基金资产净值}{基金总份额}$$

### 考点3: 期货与远期的区别

| 特征 | 远期 | 期货 |
|------|------|------|
| 标准化 | 非标准化 | 标准化 |
| 交易场所 | 场外（OTC） | 交易所 |
| 违约风险 | 较高 | 较低（清算所担保） |
| 保证金 | 无或协商 | 必须缴纳 |
| 结算方式 | 到期结算 | 逐日盯市 |

### 考点4: 期权基本策略

| 策略 | 预期 | 最大收益 | 最大损失 |
|------|------|---------|---------|
| 买入看涨期权 | 看涨 | 无限 | 权利金 |
| 卖出看涨期权 | 看跌/中性 | 权利金 | 无限 |
| 买入看跌期权 | 看跌 | 行权价-权利金 | 权利金 |
| 卖出看跌期权 | 看涨/中性 | 权利金 | 行权价-权利金 |

## 重点难点

> :warning: 难点

### 难点1: 期权价格的影响因素

| 因素 | 看涨期权 | 看跌期权 |
|------|---------|---------|
| 标的资产价格上升 | 价格上升 | 价格下降 |
| 行权价格上升 | 价格下降 | 价格上升 |
| 到期时间延长 | 价格上升 | 价格上升 |
| 波动率增大 | 价格上升 | 价格上升 |
| 无风险利率上升 | 价格上升 | 价格下降 |

### 难点2: 期权盈亏分析

**买入看涨期权**：
- 盈亏 = max(S - X, 0) - C
- 盈亏平衡点 = X + C
- S：标的资产价格，X：行权价格，C：权利金

**买入看跌期权**：
- 盈亏 = max(X - S, 0) - P
- 盈亏平衡点 = X - P
- P：权利金

## 知识点详解

### 5.1 证券投资基金

#### 5.1.1 基金的分类

**按运作方式分类**：

| 类型 | 开放式基金 | 封闭式基金 |
|------|-----------|-----------|
| 规模 | 不固定 | 固定 |
| 申购赎回 | 可随时 | 封闭期内不能 |
| 价格 | 净值 | 市价（可能折价或溢价） |
| 流动性 | 高 | 较低 |

**按投资对象分类**：
| 类型 | 投资比例 | 风险等级 |
|------|---------|---------|
| 股票基金 | ≥80%股票 | 较高 |
| 债券基金 | ≥80%债券 | 中等 |
| 货币市场基金 | 货币市场工具 | 最低 |
| 混合基金 | 股票+债券 | 中等 |
| 基金中基金（FOF） | ≥80%其他基金 | 分散 |

**按投资理念分类**：
- **主动型基金**：基金经理主动选股择时
- **被动型基金（指数基金）**：跟踪指数，被动投资

#### 5.1.2 基金的当事人

**基金份额持有人**：
- 基金投资者
- 享有收益分配权、表决权等

**基金管理人**：
- 负责基金的投资管理
- 收取管理费

**基金托管人**：
- 负责基金资产的保管和监督
- 与基金管理人相互制衡

#### 5.1.3 基金的估值与费用

**基金资产净值（NAV）**：
$$NAV = 基金总资产 - 基金总负债$$

**基金份额净值**：
$$基金份额净值 = \frac{基金资产净值}{基金总份额}$$

**基金费用**：
| 费用类型 | 说明 |
|----------|------|
| 管理费 | 支付给基金管理人 |
| 托管费 | 支付给基金托管人 |
| 申购费 | 买入基金时支付 |
| 赎回费 | 卖出基金时支付 |
| 销售服务费 | 支付给销售机构 |

#### 5.1.4 基金投资顾问业务

**概念**：
- 为投资者提供基金投资建议
- 帮助投资者构建基金组合

**试点要求**：
- 具备一定资产规模
- 具有专业投顾团队
- 建立完善的风险管理制度

### 5.2 金融衍生工具

#### 5.2.1 衍生工具概述

**四大基本衍生工具**：
1. **远期合约（Forwards）**
2. **期货合约（Futures）**
3. **期权合约（Options）**
4. **互换合约（Swaps）**

#### 5.2.2 远期与期货

**远期合约**：
- 非标准化
- 场外交易
- 违约风险较高
- 到期结算

**期货合约**：
- 标准化
- 交易所交易
- 保证金制度
- 逐日盯市

**期货与远期的区别**：
| 特征 | 远期 | 期货 |
|------|------|------|
| 标准化 | 非标准化 | 标准化 |
| 交易场所 | 场外（OTC） | 交易所 |
| 违约风险 | 较高 | 较低（清算所担保） |
| 保证金 | 无或协商 | 必须缴纳 |
| 结算方式 | 到期结算 | 逐日盯市 |
| 流动性 | 较低 | 较高 |

#### 5.2.3 期权合约

**基本类型**：
| 类型 | 买方权利 | 卖方义务 |
|------|---------|---------|
| 看涨期权（Call） | 有权以约定价格买入 | 有义务以约定价格卖出 |
| 看跌期权（Put） | 有权以约定价格卖出 | 有义务以约定价格买入 |

**期权的基本策略**：
| 策略 | 预期 | 最大收益 | 最大损失 |
|------|------|---------|---------|
| 买入看涨期权 | 看涨 | 无限 | 权利金 |
| 卖出看涨期权 | 看跌/中性 | 权利金 | 无限 |
| 买入看跌期权 | 看跌 | 行权价-权利金 | 权利金 |
| 卖出看跌期权 | 看涨/中性 | 权利金 | 行权价-权利金 |

**期权价格的影响因素**：
| 因素 | 看涨期权 | 看跌期权 |
|------|---------|---------|
| 标的资产价格上升 | 价格上升 | 价格下降 |
| 行权价格上升 | 价格下降 | 价格上升 |
| 到期时间延长 | 价格上升 | 价格上升 |
| 波动率增大 | 价格上升 | 价格上升 |
| 无风险利率上升 | 价格上升 | 价格下降 |

#### 5.2.4 互换合约

**利率互换**：
- 交换固定利率和浮动利率的利息支付
- 降低融资成本

**货币互换**：
- 交换不同货币的本金和利息
- 规避汇率风险

**互换的功能**：
- 降低融资成本
- 管理风险
- 满足不同融资需求

#### 5.2.5 衍生工具的功能

**套期保值**：
- 规避价格风险
- 锁定成本或收益

**价格发现**：
- 反映市场预期
- 提供价格参考

**投机**：
- 利用价格波动获利
- 高风险高收益

## 例题与解析

### 例题1

**题目**: 开放式基金与封闭式基金的主要区别是：

**选项**:
- A. 投资对象不同
- B. 规模是否固定
- C. 风险水平不同
- D. 收益水平不同

**答案**: B

**解析**: 开放式基金与封闭式基金的主要区别在于规模是否固定。开放式基金规模不固定，可随时申购赎回；封闭式基金规模固定，封闭期内不能赎回。

### 例题2

**题目**: 期货合约与远期合约的主要区别不包括：

**选项**:
- A. 标准化程度不同
- B. 交易场所不同
- C. 违约风险不同
- D. 投资对象不同

**答案**: D

**解析**: 期货合约与远期合约的主要区别包括标准化程度、交易场所、违约风险、保证金要求、结算方式等。投资对象不是两者的主要区别。

### 例题3

**题目**: 买入看涨期权的最大损失为：

**选项**:
- A. 无限
- B. 行权价格
- C. 权利金
- D. 标的资产价格

**答案**: C

**解析**: 买入看涨期权的最大损失为期权权利金。如果标的资产价格低于行权价格，期权不会被行权，买方损失全部权利金。最大收益理论上是无限的。

### 例题4

**题目**: 下列关于期权价格影响因素的说法，正确的是：

**选项**:
- A. 标的资产价格上升，看涨期权价格下降
- B. 行权价格上升，看涨期权价格上升
- C. 波动率增大，看涨期权价格上升
- D. 无风险利率上升，看涨期权价格下降

**答案**: C

**解析**: 波动率增大，期权价格上升（无论看涨还是看跌）。标的资产价格上升，看涨期权价格上升；行权价格上升，看涨期权价格下降；无风险利率上升，看涨期权价格上升。

### 例题5

**题目**: 基金份额净值的计算公式为：

**选项**:
- A. 基金总资产 / 基金总份额
- B. 基金总负债 / 基金总份额
- C. (基金总资产 - 基金总负债) / 基金总份额
- D. 基金总资产 - 基金总负债

**答案**: C

**解析**: 基金份额净值 = 基金资产净值 / 基金总份额 = (基金总资产 - 基金总负债) / 基金总份额。基金资产净值 = 基金总资产 - 基金总负债。
