# 第四章 债券市场

## 本章概要

本章是重点章节，介绍债券的基本概念、分类、发行、交易、估值和信用评级。重点掌握债券收益率计算（当期收益率、到期收益率）、利率期限结构理论和可转换债券。分值占比15%-18%。

## 高频考点

> :fire: 高频考点

### 考点1: 债券的特征

1. **偿还性**：有到期日，需偿还本金
2. **流动性**：可以在市场上买卖
3. **安全性**：风险相对较低
4. **收益性**：获取利息收入和资本利得

### 考点2: 债券的分类

**按发行主体**：
| 类型 | 发行主体 | 特点 |
|------|---------|------|
| 政府债券 | 政府 | 安全性最高 |
| 金融债券 | 金融机构 | 信用较高 |
| 公司债券 | 上市公司 | 收益较高 |
| 企业债券 | 非上市公司 | 发改委审批 |

**按付息方式**：
- 零息债券
- 附息债券（固定利率、浮动利率）
- 贴现债券

### 考点3: 债券收益率计算

**当期收益率**：
$$当期收益率 = \frac{年利息收入}{债券市场价格} \times 100\%$$

**到期收益率（YTM）**：
- 使债券未来现金流现值等于当前价格的折现率
- 考虑了利息收入和资本利得

### 考点4: 利率期限结构理论

| 理论 | 核心观点 |
|------|---------|
| 预期理论 | 长期利率等于预期未来短期利率的平均值 |
| 市场分割理论 | 不同期限市场相互独立 |
| 流动性偏好理论 | 长期债券需要流动性溢价 |

### 考点5: 可转换债券

**双重属性**：债权性 + 股权性

**核心要素**：
- 转股价格
- 转股比例 = 100 / 转股价格
- 赎回条款（保护发行人）
- 回售条款（保护投资者）
- 下修条款（转股价格向下修正）

## 重点难点

> :warning: 难点

### 难点1: 到期收益率（YTM）计算

**公式**：
$$P = \sum_{t=1}^{n} \frac{C}{(1+YTM)^t} + \frac{F}{(1+YTM)^n}$$

其中：
- P：债券市场价格
- C：每期利息
- F：面值
- n：剩余期数
- YTM：到期收益率

**试错法**：
- 先假设一个折现率
- 计算现值
- 与市场价格比较
- 调整折现率直到现值等于市场价格

### 难点2: 久期与凸性

**久期**：衡量债券价格对利率变动的敏感性

**麦考利久期**：
$$D = \frac{\sum_{t=1}^{n} t \times \frac{C_t}{(1+y)^t}}{P}$$

**修正久期**：
$$D_{mod} = \frac{D}{1+y}$$

**价格变动近似公式**：
$$\frac{\Delta P}{P} \approx -D_{mod} \times \Delta y$$

**凸性**：衡量久期随利率变化的程度

$$\frac{\Delta P}{P} \approx -D_{mod} \times \Delta y + \frac{1}{2} \times C \times (\Delta y)^2$$

## 知识点详解

### 4.1 债券概述

#### 4.1.1 债券的概念与特征

**定义**：债券是发行人向投资者发行的债务凭证，承诺按一定利率支付利息并按约定条件偿还本金。

**四大特征**：
1. **偿还性**：有到期日，需偿还本金
2. **流动性**：可以在市场上买卖
3. **安全性**：风险相对较低
4. **收益性**：获取利息收入和资本利得

#### 4.1.2 债券的分类

**按发行主体分类**：

| 类型 | 发行主体 | 特点 | 风险等级 |
|------|---------|------|---------|
| 政府债券 | 中央政府、地方政府 | 安全性最高，免税 | 最低 |
| 金融债券 | 银行、证券公司等 | 信用较高 | 较低 |
| 公司债券 | 上市公司 | 收益较高 | 中等 |
| 企业债券 | 非上市公司 | 发改委审批 | 中等 |

**按付息方式分类**：
| 类型 | 特点 |
|------|------|
| 零息债券 | 不支付利息，折价发行 |
| 附息债券 | 定期支付利息 |
| 贴现债券 | 低于面值发行，到期按面值偿还 |

**按嵌入选择权分类**：
| 类型 | 特点 |
|------|------|
| 可赎回债券 | 发行人有权提前赎回 |
| 可回售债券 | 投资者有权提前回售 |
| 可转换债券 | 可转换为股票 |

#### 4.1.3 债券的要素

- **面值**：债券票面标明的金额
- **票面利率**：债券票面标明的利率
- **期限**：债券从发行到到期的时间
- **偿还方式**：到期一次性偿还或分期偿还
- **发行价格**：债券发行时的价格

### 4.2 债券发行

#### 4.2.1 发行条件

| 债券类型 | 发行主体 | 审批机构 |
|----------|---------|---------|
| 国债 | 财政部 | 国务院 |
| 地方政府债 | 省级政府 | 国务院 |
| 金融债 | 金融机构 | 人民银行 |
| 公司债 | 上市公司 | 交易所/证监会 |
| 企业债 | 非上市公司 | 发改委 |

#### 4.2.2 发行方式

**招标发行**：
- **荷兰式招标**：单一价格招标
- **美国式招标**：多重价格招标

**定向发行**：
- 向特定投资者发行
- 不公开发行

**公开发行**：
- 向社会公众发行
- 需满足发行条件

### 4.3 债券交易

#### 4.3.1 交易方式

**现券交易**：
- 买卖双方以约定价格即时交割
- 最常见的交易方式

**回购交易**：
| 类型 | 特点 |
|------|------|
| 质押式回购 | 债券作为质押品 |
| 买断式回购 | 债券所有权转移 |

**远期交易**：
- 约定未来某日以约定价格交易
- 场外交易

**期货交易**：
- 标准化合约
- 交易所交易

#### 4.3.2 交易市场

**银行间债券市场**：
- 机构投资者参与
- 交易规模最大
- 做市商制度

**交易所债券市场**：
- 机构和个人投资者均可参与
- 竞价交易和大宗交易

### 4.4 债券估值

#### 4.4.1 影响债券价值的因素

- **票面利率**：票面利率越高，债券价值越高
- **市场利率（折现率）**：市场利率越高，债券价值越低
- **剩余期限**：期限越长，对利率变动越敏感
- **信用风险**：信用风险越高，债券价值越低
- **流动性**：流动性越好，债券价值越高

#### 4.4.2 债券收益率

**当期收益率**：
$$当期收益率 = \frac{年利息收入}{债券市场价格} \times 100\%$$

**到期收益率（YTM）**：
- 使债券未来现金流现值等于当前价格的折现率
- 考虑了利息收入和资本利得

$$P = \sum_{t=1}^{n} \frac{C}{(1+YTM)^t} + \frac{F}{(1+YTM)^n}$$

**持有期收益率**：
$$持有期收益率 = \frac{利息收入 + 资本利得}{购买价格} \times 100\%$$

#### 4.4.3 久期与凸性

**久期**：衡量债券价格对利率变动的敏感性

**麦考利久期**：
$$D = \frac{\sum_{t=1}^{n} t \times \frac{C_t}{(1+y)^t}}{P}$$

**修正久期**：
$$D_{mod} = \frac{D}{1+y}$$

**价格变动近似公式**：
$$\frac{\Delta P}{P} \approx -D_{mod} \times \Delta y$$

**凸性**：衡量久期随利率变化的程度

$$\frac{\Delta P}{P} \approx -D_{mod} \times \Delta y + \frac{1}{2} \times C \times (\Delta y)^2$$

#### 4.4.4 利率期限结构理论

| 理论 | 核心观点 | 局限性 |
|------|---------|--------|
| 预期理论 | 长期利率等于预期未来短期利率的平均值 | 忽略风险因素 |
| 市场分割理论 | 不同期限市场相互独立 | 过于绝对 |
| 流动性偏好理论 | 长期债券需要流动性溢价 | 较为合理 |

### 4.5 信用评级

#### 4.5.1 信用评级等级

**长期信用等级**：
| 等级 | 含义 |
|------|------|
| AAA | 最高信用等级，极低违约风险 |
| AA | 高信用等级，很低违约风险 |
| A | 中上信用等级，较低违约风险 |
| BBB | 中等信用等级，有一定违约风险 |
| BB | 中下信用等级，较高违约风险 |
| B | 较低信用等级，高违约风险 |
| CCC | 很低信用等级，很高违约风险 |
| CC | 极低信用等级 |
| C | 最低信用等级 |
| D | 已违约 |

**短期信用等级**：A-1、A-2、A-3、B、C、D

#### 4.5.2 信用评级程序

1. 评级准备
2. 实地调查
3. 初评
4. 评级委员会讨论
5. 结果发布与跟踪

### 4.6 可转换债券

#### 4.6.1 可转债特征

**双重属性**：
- **债权性**：定期支付利息，到期偿还本金
- **股权性**：可转换为股票

**核心要素**：
| 要素 | 含义 |
|------|------|
| 转股价格 | 转换为股票的价格 |
| 转股比例 | 100 / 转股价格 |
| 赎回条款 | 保护发行人，可强制赎回 |
| 回售条款 | 保护投资者，可回售给发行人 |
| 下修条款 | 转股价格向下修正 |

## 例题与解析

### 例题1

**题目**: 某债券面值100元，票面利率5%，市场价格95元，当期收益率为：

**选项**:
- A. 4.75%
- B. 5.00%
- C. 5.26%
- D. 5.50%

**答案**: C

**解析**: 当期收益率 = 年利息收入 / 债券市场价格 × 100% = 5 / 95 × 100% = 5.26%。注意当期收益率使用市场价格而非面值。

### 例题2

**题目**: 利率期限结构理论中，认为长期利率等于预期未来短期利率平均值的是：

**选项**:
- A. 预期理论
- B. 市场分割理论
- C. 流动性偏好理论
- D. 购买力平价理论

**答案**: A

**解析**: 预期理论认为长期利率等于预期未来短期利率的平均值。市场分割理论认为不同期限市场相互独立。流动性偏好理论认为长期债券需要流动性溢价。

### 例题3

**题目**: 可转换债券的转股比例计算公式为：

**选项**:
- A. 转股比例 = 转股价格 / 100
- B. 转股比例 = 100 / 转股价格
- C. 转股比例 = 面值 / 转股价格
- D. 转股比例 = 转股价格 / 面值

**答案**: B

**解析**: 可转换债券的转股比例 = 100 / 转股价格。例如转股价格为20元，则转股比例 = 100 / 20 = 5，即每张可转债可转换为5股股票。

### 例题4

**题目**: 久期衡量的是：

**选项**:
- A. 债券的到期时间
- B. 债券价格对利率变动的敏感性
- C. 债券的信用风险
- D. 债券的流动性

**答案**: B

**解析**: 久期是衡量债券价格对利率变动敏感性的指标。修正久期越大，债券价格对利率变动越敏感。久期与到期时间相关，但不等同于到期时间。

### 例题5

**题目**: 信用评级中，最高信用等级为：

**选项**:
- A. AA
- B. AAA
- C. A
- D. BBB

**答案**: B

**解析**: 信用评级中，AAA为最高信用等级，表示极低违约风险。AA为高信用等级，A为中上信用等级，BBB为中等信用等级。BBB以下为投机级债券。
