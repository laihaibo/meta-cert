# 第六章 金融风险管理

## 本章概要

本章介绍金融风险的类型和度量方法。重点掌握系统性风险与非系统性风险的区别，以及VaR、Beta系数等风险度量方法。分值占比8%-10%。

## 高频考点

> :fire: 高频考点

### 考点1: 系统性风险 vs 非系统性风险

| 类型 | 特点 | 能否分散 |
|------|------|---------|
| 系统性风险 | 影响整个市场 | 不可分散 |
| 非系统性风险 | 影响个别证券 | 可通过分散投资降低 |

**系统性风险**：
- 市场风险
- 利率风险
- 汇率风险
- 通胀风险
- 政策风险

**非系统性风险**：
- 信用风险
- 经营风险
- 财务风险
- 流动性风险

### 考点2: VaR（在险价值）

**定义**：在一定置信水平下，某一时期内最大可能损失

**计算方法**：
- 历史模拟法
- 蒙特卡洛模拟法
- 方差-协方差法

### 考点3: Beta系数

**定义**：衡量系统性风险的指标

**含义**：
| Beta值 | 含义 |
|--------|------|
| Beta > 1 | 风险高于市场 |
| Beta = 1 | 风险等于市场 |
| Beta < 1 | 风险低于市场 |

## 重点难点

> :warning: 难点

### 难点1: VaR的计算

**方差-协方差法**：
$$VaR = Z_{\alpha} \times \sigma \times \sqrt{t} \times P$$

其中：
- $Z_{\alpha}$：置信水平对应的Z值（如95%对应1.65，99%对应2.33）
- $\sigma$：收益率标准差
- $t$：时间期限
- $P$：资产价值

### 难点2: Beta系数的计算

**公式**：
$$\beta_i = \frac{Cov(R_i, R_m)}{Var(R_m)}$$

其中：
- $R_i$：证券i的收益率
- $R_m$：市场收益率
- $Cov$：协方差
- $Var$：方差

## 知识点详解

### 6.1 风险类型

#### 6.1.1 系统性风险

**定义**：影响整个市场的风险，不可通过分散投资消除。

**类型**：
| 风险类型 | 说明 |
|----------|------|
| 市场风险 | 证券市场价格波动风险 |
| 利率风险 | 利率变动导致的风险 |
| 汇率风险 | 汇率变动导致的风险 |
| 通胀风险 | 通货膨胀导致的风险 |
| 政策风险 | 政策变动导致的风险 |

#### 6.1.2 非系统性风险

**定义**：影响个别证券的风险，可通过分散投资降低。

**类型**：
| 风险类型 | 说明 |
|----------|------|
| 信用风险 | 违约风险 |
| 经营风险 | 公司经营不善风险 |
| 财务风险 | 财务结构不合理风险 |
| 流动性风险 | 无法及时变现风险 |

### 6.2 风险度量

#### 6.2.1 VaR（在险价值）

**定义**：在一定置信水平下，某一时期内最大可能损失。

**计算方法**：

**历史模拟法**：
- 基于历史数据
- 不需要假设分布
- 简单直观

**蒙特卡洛模拟法**：
- 基于随机模拟
- 适用于复杂情况
- 计算量大

**方差-协方差法**：
$$VaR = Z_{\alpha} \times \sigma \times \sqrt{t} \times P$$

**应用**：
- 风险管理
- 资本配置
- 绩效评估

#### 6.2.2 标准差

**定义**：衡量收益率的波动程度。

**公式**：
$$\sigma = \sqrt{\frac{\sum_{i=1}^{n}(R_i - \bar{R})^2}{n-1}}$$

**应用**：
- 标准差越大，风险越高
- 适用于衡量总风险

#### 6.2.3 Beta系数

**定义**：衡量系统性风险的指标。

**公式**：
$$\beta_i = \frac{Cov(R_i, R_m)}{Var(R_m)}$$

**含义**：
| Beta值 | 含义 | 投资策略 |
|--------|------|---------|
| Beta > 1 | 风险高于市场 | 进攻型 |
| Beta = 1 | 风险等于市场 | 中性 |
| Beta < 1 | 风险低于市场 | 防守型 |
| Beta = 0 | 无系统性风险 | 避险 |

**应用**：
- 衡量证券的系统性风险
- 构建投资组合
- 计算资本资产定价模型（CAPM）

### 6.3 风险管理方法

#### 6.3.1 分散投资

**原理**：
- 非系统性风险可通过分散投资降低
- 组合风险低于单个证券风险的加权平均

**方法**：
- 投资不同行业
- 投资不同地区
- 投资不同资产类别

#### 6.3.2 对冲

**原理**：
- 利用衍生工具转移风险
- 锁定成本或收益

**方法**：
- 期货对冲
- 期权对冲
- 互换对冲

#### 6.3.3 保险

**原理**：
- 支付保费转移风险
- 保险公司承担风险

**方法**：
- 财产保险
- 责任保险
- 信用保险

## 例题与解析

### 例题1

**题目**: 下列属于系统性风险的是：

**选项**:
- A. 信用风险
- B. 经营风险
- C. 市场风险
- D. 财务风险

**答案**: C

**解析**: 系统性风险包括市场风险、利率风险、汇率风险、通胀风险和政策风险。信用风险、经营风险、财务风险属于非系统性风险，可通过分散投资降低。

### 例题2

**题目**: Beta系数大于1表示：

**选项**:
- A. 风险低于市场
- B. 风险等于市场
- C. 风险高于市场
- D. 无风险

**答案**: C

**解析**: Beta系数衡量证券的系统性风险。Beta > 1表示风险高于市场，Beta = 1表示风险等于市场，Beta < 1表示风险低于市场，Beta = 0表示无系统性风险。

### 例题3

**题目**: VaR的含义是：

**选项**:
- A. 在一定置信水平下，某一时期内最大可能收益
- B. 在一定置信水平下，某一时期内最大可能损失
- C. 在一定置信水平下，某一时期内平均收益
- D. 在一定置信水平下，某一时期内平均损失

**答案**: B

**解析**: VaR（在险价值）是指在一定置信水平下，某一时期内最大可能损失。例如，95%置信水平下日VaR为100万元，表示有95%的把握认为一天内的最大损失不超过100万元。

### 例题4

**题目**: 下列风险度量方法中，不需要假设收益率分布的是：

**选项**:
- A. 历史模拟法
- B. 蒙特卡洛模拟法
- C. 方差-协方差法
- D. Beta系数法

**答案**: A

**解析**: 历史模拟法基于历史数据，不需要假设收益率的分布形式。蒙特卡洛模拟法需要假设随机过程，方差-协方差法需要假设正态分布。

### 例题5

**题目**: 非系统性风险的特点是：

**选项**:
- A. 影响整个市场
- B. 不可通过分散投资消除
- C. 影响个别证券
- D. 无法度量

**答案**: C

**解析**: 非系统性风险影响个别证券，可通过分散投资降低。系统性风险影响整个市场，不可通过分散投资消除。两种风险都可以度量。
