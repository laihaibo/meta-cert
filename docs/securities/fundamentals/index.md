# 金融市场基础知识

## 科目介绍

《金融市场基础知识》是证券从业资格考试的科目二，主要考查考生对金融市场体系、证券市场运行机制、股票债券基金等金融产品的理解和掌握程度。考试形式为机考，共100道单选题（含组合型单选），总分100分，60分及格，考试时间120分钟。

## 备考建议

1. **重点章节优先**：第三章至第五章分值占比最高（合计45%-56%），应优先掌握
2. **计算题突破**：债券收益率、股票估值、期权定价等计算题是难点
3. **对比记忆**：不同金融产品的特点、不同市场的上市条件需对比记忆
4. **刷题巩固**：建议结合题库反复练习，特别是计算题和综合题
