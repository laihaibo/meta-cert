# 第一章 金融市场体系

## 本章概要

本章介绍金融市场的基本概念、功能、分类以及全球主要金融市场。重点掌握金融市场的五大功能和分类方式。分值占比8%-10%。

## 知识点详解

### 1.1 金融市场概述

#### 1.1.1 金融市场的概念与功能

**定义**：金融市场是指以金融资产为交易对象，以金融资产的供给方和需求方为交易主体形成的交易机制及其关系的总和。

**五大功能**：
1. **资金融通功能**
   - 资金从盈余方流向短缺方
   - 实现资源的优化配置
   - 提高资金使用效率

2. **价格发现功能**
   - 通过交易形成资产价格
   - 反映市场供求关系
   - 引导资源配置

3. **提供流动性功能**
   - 资产可以快速变现
   - 降低交易成本
   - 提高市场效率

4. **风险管理功能**
   - 通过衍生工具转移风险
   - 实现风险的分散和对冲
   - 提高市场稳定性

5. **降低搜寻成本和信息成本功能**
   - 减少交易成本
   - 提高市场透明度
   - 促进市场发展

#### 1.1.2 金融市场的分类

**按交易标的物分类**：
| 市场类型 | 交易对象 | 特点 |
|----------|---------|------|
| 货币市场 | 短期金融工具 | 期限1年以内，流动性强 |
| 资本市场 | 长期金融工具 | 期限1年以上，收益较高 |
| 外汇市场 | 外汇 | 全球最大金融市场 |
| 金融衍生品市场 | 衍生工具 | 风险管理工具 |

**按发行流通分类**：
- **一级市场（发行市场）**：新证券首次发行的市场
- **二级市场（交易市场）**：已发行证券进行交易的市场

**按组织方式分类**：
- **交易所市场**：有固定场所，集中交易
- **场外市场（OTC）**：无固定场所，分散交易

### 1.2 全球金融市场

#### 1.2.1 全球主要金融市场

**纽约证券交易所（NYSE）**：
- 全球最大股票交易所
- 上市公司总市值全球第一
- 采用专家制度

**纳斯达克（NASDAQ）**：
- 科技股为主的电子交易市场
- 采用做市商制度
- 上市门槛相对较低

**伦敦证券交易所（LSE）**：
- 历史最悠久的交易所之一
- 国际化程度高
- 采用连续交易制度

**东京证券交易所（TSE）**：
- 亚洲主要交易所
- 上市公司以日本企业为主

**香港交易所（HKEX）**：
- 连接中国内地与国际市场
- 同时上市H股和红筹股
- 国际化程度高

#### 1.2.2 全球金融体系的主要参与者

**主要参与者**：
- 各国中央银行
- 商业银行和投资银行
- 保险公司和养老基金
- 对冲基金和私募股权基金
- 主权财富基金

#### 1.2.3 国际资金流动方式

**国际短期资金流动（热钱）**：
| 类型 | 特点 |
|------|------|
| 贸易资金流动 | 与国际贸易相关 |
| 套利性资金流动 | 利用利差获利 |
| 避险性资金流动 | 规避风险 |
| 投机性资金流动 | 利用汇率波动获利 |

**国际长期资金流动**：
| 类型 | 特点 |
|------|------|
| 国际直接投资 | 参与经营管理 |
| 国际证券投资 | 购买股票债券 |
| 国际贷款 | 借贷关系 |

## 例题与解析

### 例题1

**题目**: 金融市场的功能不包括：

**选项**:
- A. 资金融通功能
- B. 价格发现功能
- C. 风险消除功能
- D. 提供流动性功能

**答案**: C

**解析**: 金融市场具有资金融通、价格发现、提供流动性、风险管理和降低搜寻成本和信息成本五大功能。注意是风险管理而非风险消除，金融市场可以帮助转移和分散风险，但不能完全消除风险。

### 例题2

**题目**: 按交易标的物分类，金融市场不包括：

**选项**:
- A. 货币市场
- B. 资本市场
- C. 一级市场
- D. 外汇市场

**答案**: C

**解析**: 按交易标的物分类，金融市场包括货币市场、资本市场、外汇市场和金融衍生品市场。一级市场是按发行流通分类的，不属于按交易标的物分类。

### 例题3

**题目**: 全球最大的股票交易所是：

**选项**:
- A. 纳斯达克
- B. 纽约证券交易所
- C. 伦敦证券交易所
- D. 东京证券交易所

**答案**: B

**解析**: 纽约证券交易所（NYSE）是全球最大的股票交易所，上市公司总市值全球第一。纳斯达克是科技股为主的电子交易市场，规模仅次于纽约证券交易所。

### 例题4

**题目**: 国际短期资金流动不包括：

**选项**:
- A. 贸易资金流动
- B. 套利性资金流动
- C. 国际直接投资
- D. 投机性资金流动

**答案**: C

**解析**: 国际短期资金流动（热钱）包括贸易资金流动、套利性资金流动、避险性资金流动和投机性资金流动。国际直接投资属于国际长期资金流动。

### 例题5

**题目**: 交易所市场与场外市场的主要区别是：

**选项**:
- A. 交易对象不同
- B. 交易时间不同
- C. 是否有固定交易场所
- D. 交易费用不同

**答案**: C

**解析**: 交易所市场有固定交易场所，采用集中交易方式；场外市场（OTC）没有固定交易场所，采用分散交易方式。两者在交易对象、交易时间、交易费用等方面也可能存在差异，但主要区别在于是否有固定交易场所。
