# 第二章 证券市场主体

## 本章概要

本章介绍中国多层次资本市场结构和证券市场的主要参与者，包括交易所市场、场外市场、证券发行人、投资者、中介机构和监管机构。重点掌握各板块的特点和投资者门槛。分值占比10%-12%。

## 高频考点

> :fire: 高频考点

### 考点1: 中国多层次资本市场

**交易所市场**：
| 板块 | 设立时间 | 特点 | 投资者门槛 |
|------|---------|------|-----------|
| 主板 | 最早 | 上市门槛最高，大型成熟企业 | 无特殊门槛 |
| 科创板 | 2019年7月 | 注册制，允许未盈利企业 | 50万元+2年经验 |
| 创业板 | 2009年 | 成长型创新创业企业 | 10万元+2年经验 |
| 北交所 | 2021年9月 | 创新型中小企业 | 50万元+2年经验 |

**场外市场**：
- **新三板**：服务中小微企业，基础层、创新层
- **区域性股权交易市场**：各省市设立，门槛最低

### 考点2: 科创板与创业板特殊规定

**科创板**：
- IPO前**5个交易日**不设涨跌幅限制
- 之后涨跌幅限制为**20%**
- 投资者门槛：**50万元**证券资产 + **2年**交易经验

**创业板**：
- 注册制改革后涨跌幅限制为**20%**
- 投资者门槛：**10万元**证券资产 + **2年**交易经验

### 考点3: 证券市场参与者

**证券发行人**：上市公司、非上市公众公司、政府、金融机构

**投资者**：
- 个人投资者
- 机构投资者：证券投资基金、保险公司、社保基金、QFII/RQFII、企业年金

**证券中介机构**：证券公司、会计师事务所、律师事务所、资产评估机构、信用评级机构

**自律性组织**：证券交易所、中国证券业协会、中国证券登记结算有限公司

**监管机构**：中国证券监督管理委员会（CSRC）

### 考点4: 商业银行

**主要业务**：
- 负债业务
- 资产业务
- 中间业务

**经营原则**：安全性、流动性、盈利性

## 重点难点

> :warning: 难点

### 难点1: 各板块上市条件对比

| 条件 | 主板 | 科创板 | 创业板 | 北交所 |
|------|------|--------|--------|--------|
| 市值要求 | 较高 | 多元化 | 中等 | 较低 |
| 盈利要求 | 连续3年盈利 | 允许未盈利 | 连续2年盈利 | 较宽松 |
| 股本总额 | ≥5000万元 | ≥3000万元 | ≥3000万元 | ≥3000万元 |
| 涨跌幅 | ±10% | ±20% | ±20% | ±30% |

### 难点2: QFII与RQFII的区别

| 项目 | QFII | RQFII |
|------|------|-------|
| 全称 | 合格境外机构投资者 | 人民币合格境外机构投资者 |
| 资金来源 | 外币 | 人民币 |
| 投资范围 | 交易所上市证券 | 交易所上市证券+银行间债券 |
| 额度管理 | 有额度限制 | 有额度限制 |

## 知识点详解

### 2.1 中国资本市场层次结构

#### 2.1.1 交易所市场

**主板市场**：
- 上海证券交易所主板
- 深圳证券交易所主板
- 特点：上市门槛最高，适合大型成熟企业
- 涨跌幅限制：±10%

**科创板**（2019年7月22日开板）：
- 设立于上海证券交易所
- 定位：面向世界科技前沿、面向经济主战场、面向国家重大需求
- 注册制试点
- 上市条件多元化（允许未盈利企业上市）
- IPO前5个交易日不设涨跌幅限制
- 之后涨跌幅限制为20%
- 投资者门槛：50万元证券资产 + 2年交易经验

**创业板**（注册制改革后）：
- 设立于深圳证券交易所
- 定位：服务成长型创新创业企业
- 注册制改革后涨跌幅限制为20%
- 投资者门槛：10万元证券资产 + 2年交易经验

**北京证券交易所（北交所）**（2021年9月成立）：
- 定位：服务创新型中小企业
- 由新三板精选层变更而来
- 投资者门槛：50万元证券资产 + 2年交易经验
- 涨跌幅限制：±30%

#### 2.1.2 场外市场

**全国中小企业股份转让系统（新三板）**：
- 定位：服务中小微企业
- 基础层、创新层、精选层（精选层已转为北交所）
- 挂牌条件相对宽松

**区域性股权交易市场**：
- 各省市设立
- 门槛最低
- 服务本地中小企业

### 2.2 证券市场参与者

#### 2.2.1 证券发行人

**上市公司**：
- 股票在证券交易所上市交易的股份有限公司
- 需满足上市条件并经交易所审核同意

**非上市公众公司**：
- 股票未在证券交易所上市交易的公众公司
- 股东人数超过200人

**政府**：
- 发行国债、地方政府债
- 筹集财政资金

**金融机构**：
- 发行金融债券
- 补充资本金

#### 2.2.2 投资者

**个人投资者**：
- 以个人名义参与证券投资
- 资金规模相对较小
- 需要适当性管理

**机构投资者**：
| 类型 | 特点 |
|------|------|
| 证券投资基金 | 集合投资，专业管理 |
| 保险公司 | 长期资金，稳健投资 |
| 社保基金 | 安全性要求高 |
| QFII/RQFII | 境外机构投资者 |
| 企业年金 | 长期资金，安全性要求高 |

#### 2.2.3 证券中介机构

**证券公司**：
- 证券经纪、投资咨询、承销保荐等
- 连接投资者与融资者

**会计师事务所**：
- 审计上市公司财务报告
- 提供会计咨询服务

**律师事务所**：
- 提供法律意见
- 出具法律意见书

**资产评估机构**：
- 评估资产价值
- 为并购重组提供估值服务

**信用评级机构**：
- 评估证券信用风险
- 为投资者提供决策参考

#### 2.2.4 自律性组织

**证券交易所**：
- 上海证券交易所
- 深圳证券交易所
- 北京证券交易所
- 组织证券交易，实施自律监管

**中国证券业协会**：
- 证券业自律性组织
- 制定行业规范
- 组织从业人员资格考试

**中国证券登记结算有限公司**：
- 证券的登记、存管、结算
- 维护投资者权益

**中国证券投资者保护基金公司**：
- 筹集、管理和运作投资者保护基金
- 保护投资者合法权益

#### 2.2.5 监管机构

**中国证券监督管理委员会（CSRC）**：
- 监管职责：制定规章规则、监管证券期货市场、查处违法违规行为
- 派出机构设置：各省证监局

### 2.3 商业银行

#### 2.3.1 商业银行概述

**性质与职能**：
- 以营利为目的
- 经营货币资金
- 充当信用中介

**主要业务**：
| 业务类型 | 内容 |
|----------|------|
| 负债业务 | 吸收存款、发行债券 |
| 资产业务 | 发放贷款、投资证券 |
| 中间业务 | 结算、代理、咨询 |

**经营原则**：
- **安全性**：保障资金安全
- **流动性**：保持支付能力
- **盈利性**：获取合理利润

## 例题与解析

### 例题1

**题目**: 科创板的投资者门槛为：

**选项**:
- A. 10万元证券资产 + 2年交易经验
- B. 30万元证券资产 + 2年交易经验
- C. 50万元证券资产 + 2年交易经验
- D. 100万元证券资产 + 2年交易经验

**答案**: C

**解析**: 科创板的投资者门槛为50万元证券资产 + 2年交易经验。创业板的门槛为10万元证券资产 + 2年交易经验。北交所的门槛也为50万元证券资产 + 2年交易经验。

### 例题2

**题目**: 下列不属于证券中介机构的是：

**选项**:
- A. 证券公司
- B. 会计师事务所
- C. 中国证监会
- D. 信用评级机构

**答案**: C

**解析**: 中国证监会是证券监管机构，不属于证券中介机构。证券中介机构包括证券公司、会计师事务所、律师事务所、资产评估机构、信用评级机构等。

### 例题3

**题目**: 北交所的涨跌幅限制为：

**选项**:
- A. ±10%
- B. ±20%
- C. ±30%
- D. 不设限制

**答案**: C

**解析**: 北交所的涨跌幅限制为±30%。主板为±10%，科创板和创业板为±20%。科创板和创业板IPO前5个交易日不设涨跌幅限制。

### 例题4

**题目**: 商业银行的经营原则不包括：

**选项**:
- A. 安全性
- B. 流动性
- C. 盈利性
- D. 创新性

**答案**: D

**解析**: 商业银行的经营原则是安全性、流动性、盈利性。创新性不是商业银行的基本经营原则。

### 例题5

**题目**: QFII是指：

**选项**:
- A. 合格境内机构投资者
- B. 合格境外机构投资者
- C. 人民币合格境外机构投资者
- D. 合格个人投资者

**答案**: B

**解析**: QFII（Qualified Foreign Institutional Investor）是指合格境外机构投资者。RQFII是指人民币合格境外机构投资者。QDII是指合格境内机构投资者。
