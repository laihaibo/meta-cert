# 第三章 股票市场

## 本章概要

本章是重点章节，介绍股票的基本概念、分类、发行、交易和估值方法。重点掌握股票的特征、发行制度、交易机制和估值方法（DDM、P/E、P/B）。分值占比15%-18%。

## 高频考点

> :fire: 高频考点

### 考点1: 股票的特征

1. **收益性**：获取股息、红利和资本利得
2. **风险性**：投资可能亏损
3. **流动性**：可以在市场上买卖
4. **永久性**：无到期日，不可偿还
5. **参与性**：股东有权参与公司决策

### 考点2: 股票的分类

**按股东权利**：
| 类型 | 特点 |
|------|------|
| 普通股 | 投票权、分红权、优先认股权、剩余资产分配权 |
| 优先股 | 优先分红、优先清偿、一般无投票权 |

**按上市地域**：
| 类型 | 特点 |
|------|------|
| A股 | 境内上市，人民币交易 |
| B股 | 境内上市，外币交易 |
| H股 | 香港上市的内地企业股票 |
| N股 | 纽约上市 |
| S股 | 新加坡上市 |

### 考点3: 股票发行制度

**注册制**（现行制度）：
- 审核机构：交易所
- 注册机构：证监会
- 核心：信息披露

### 考点4: 涨跌幅限制

| 板块 | 涨跌幅限制 | 特殊情况 |
|------|-----------|----------|
| 主板 | ±10% | ST股票±5% |
| 科创板 | ±20% | 前5个交易日不设限制 |
| 创业板 | ±20% | 前5个交易日不设限制 |
| 北交所 | ±30% | 上市首日不设限制 |

### 考点5: 股票估值方法

**绝对估值法**：
- 股利贴现模型（DDM）
- 自由现金流贴现模型（DCF）

**相对估值法**：
- 市盈率（P/E）= 股价 / 每股收益
- 市净率（P/B）= 股价 / 每股净资产
- 市销率（P/S）= 股价 / 每股销售收入

## 重点难点

> :warning: 难点

### 难点1: 股利贴现模型（DDM）

**基本公式**：
$$P_0 = \sum_{t=1}^{\infty} \frac{D_t}{(1+r)^t}$$

**零增长模型**（股利不变）：
$$P_0 = \frac{D}{r}$$

**固定增长模型**（戈登增长模型）：
$$P_0 = \frac{D_1}{r-g}$$

其中：
- $P_0$：股票内在价值
- $D_t$：第t期股利
- $r$：折现率（必要收益率）
- $g$：股利增长率

### 难点2: 市盈率估值法

**公式**：P/E = 股价 / 每股收益

**应用**：
- 股票价值 = 每股收益 × 市盈率
- 适用于盈利稳定的公司

**局限性**：
- 不适用于亏损公司
- 受会计政策影响
- 不同行业市盈率差异大

## 知识点详解

### 3.1 股票概述

#### 3.1.1 股票的概念与特征

**定义**：股票是股份有限公司签发的证明股东所持股份的凭证。

**五大特征**：
1. **收益性**
   - 股息、红利收入
   - 资本利得（买卖差价）
   - 公积金转增收益

2. **风险性**
   - 公司经营风险
   - 市场波动风险
   - 政策风险

3. **流动性**
   - 可以在证券交易所买卖
   - 变现能力强
   - 流动性因股票而异

4. **永久性**
   - 无到期日
   - 不可偿还
   - 只能转让

5. **参与性**
   - 股东有权参与公司决策
   - 参加股东大会
   - 行使投票权

#### 3.1.2 股票的分类

**按股东权利分类**：

**普通股**：
- 投票权：参与公司重大决策
- 分红权：获取股息红利
- 优先认股权：新股发行时优先认购
- 剩余资产分配权：公司清算时分配剩余资产

**优先股**：
- 优先分红：先于普通股分红
- 优先清偿：公司清算时先于普通股
- 一般无投票权
- 股息率固定

**按上市地域分类**：

| 类型 | 上市地点 | 交易货币 | 特点 |
|------|---------|----------|------|
| A股 | 境内交易所 | 人民币 | 境内投资者为主 |
| B股 | 境内交易所 | 外币 | 面向境外投资者 |
| H股 | 香港交易所 | 港币 | 内地企业香港上市 |
| N股 | 纽约交易所 | 美元 | 内地企业纽约上市 |
| S股 | 新加坡交易所 | 新加坡元 | 内地企业新加坡上市 |

#### 3.1.3 股票的价值

**票面价值（面值）**：
- 股票票面标明的金额
- A股面值通常为1元

**账面价值（每股净资产）**：
- 公式：(总资产 - 无形资产 - 负债) / 总股本
- 反映公司净资产

**清算价值**：
- 公司清算时每股的实际价值
- 通常低于账面价值

**内在价值**：
- 未来现金流的现值
- 通过估值模型计算

### 3.2 股票发行

#### 3.2.1 发行制度

**注册制**（现行制度）：
- 审核机构：证券交易所
- 注册机构：中国证监会
- 核心：信息披露
- 特点：形式审核为主

**核准制**（已逐步退出）：
- 实质审核
- 发审委审核
- 特点：实质性判断

#### 3.2.2 发行方式

**首次公开发行（IPO）**：
- 公司首次向社会公众公开发行股票
- 需满足发行条件

**增发**：
- **公开增发**：向全体股东配售
- **定向增发**：向特定投资者配售

**配股**：
- 向原股东配售新股
- 原股东有优先认购权

#### 3.2.3 发行定价

**定价方式**：
- 向网下投资者询价
- 向战略投资者配售
- 网上定价发行

### 3.3 股票交易

#### 3.3.1 交易机制

**竞价交易**：
| 类型 | 时间 | 特点 |
|------|------|------|
| 集合竞价 | 9:15-9:25（开盘）| 最大成交量原则 |
| 连续竞价 | 9:30-11:30，13:00-15:00 | 价格优先、时间优先 |

**大宗交易**：
- 单笔交易数量或金额达到一定标准
- 在大宗交易系统完成
- 不影响连续竞价

**协议转让**：
- 买卖双方协商确定价格
- 适用于特定情况

#### 3.3.2 涨跌幅限制

| 板块 | 涨跌幅限制 | 特殊情况 |
|------|-----------|----------|
| 主板 | ±10% | ST股票±5% |
| 科创板 | ±20% | 前5个交易日不设限制 |
| 创业板 | ±20% | 前5个交易日不设限制 |
| 北交所 | ±30% | 上市首日不设限制 |

#### 3.3.3 T+1交易制度

- 当日买入的股票，下一交易日才能卖出
- 当日卖出股票的资金，当日可用于买入

### 3.4 股票价格指数

#### 3.4.1 主要指数

| 指数 | 样本范围 | 特点 |
|------|---------|------|
| 上证综指 | 上海全部上市股票 | 综合反映沪市走势 |
| 深证成指 | 深圳500家代表性公司 | 代表深市走势 |
| 沪深300 | 沪深300只大盘蓝筹股 | 反映大盘股走势 |
| 中证500 | 扣除沪深300后的500只中盘股 | 反映中盘股走势 |
| 创业板指 | 创业板100只代表性股票 | 反映创业板走势 |
| 科创50 | 科创板50只代表性股票 | 反映科创板走势 |

#### 3.4.2 指数编制方法

**市值加权法**：
- 以市值为权重
- 市值越大，权重越高

**等权重法**：
- 每只股票权重相同
- 不考虑市值差异

**价格加权法**：
- 以价格为权重
- 价格越高，权重越高

### 3.5 股票估值

#### 3.5.1 绝对估值法

**股利贴现模型（DDM）**：

基本公式：
$$P_0 = \sum_{t=1}^{\infty} \frac{D_t}{(1+r)^t}$$

**零增长模型**（股利不变）：
$$P_0 = \frac{D}{r}$$

**固定增长模型**（戈登增长模型）：
$$P_0 = \frac{D_1}{r-g}$$

其中：
- $P_0$：股票内在价值
- $D_t$：第t期股利
- $r$：折现率（必要收益率）
- $g$：股利增长率

**自由现金流贴现模型（DCF）**：
$$P_0 = \sum_{t=1}^{\infty} \frac{FCF_t}{(1+r)^t}$$

其中：
- $FCF_t$：第t期自由现金流
- $r$：加权平均资本成本

#### 3.5.2 相对估值法

**市盈率（P/E）**：
$$P/E = \frac{股价}{每股收益}$$

应用：股票价值 = 每股收益 × 市盈率

**市净率（P/B）**：
$$P/B = \frac{股价}{每股净资产}$$

应用：股票价值 = 每股净资产 × 市净率

**市销率（P/S）**：
$$P/S = \frac{股价}{每股销售收入}$$

应用：适用于亏损公司

**EV/EBITDA**：
$$EV/EBITDA = \frac{企业价值}{息税折旧摊销前利润}$$

应用：适用于资本密集型行业

## 例题与解析

### 例题1

**题目**: 股票的特征不包括：

**选项**:
- A. 收益性
- B. 风险性
- C. 有期限性
- D. 参与性

**答案**: C

**解析**: 股票具有收益性、风险性、流动性、永久性和参与性五大特征。股票是永久性证券，没有到期日，因此不具有有期限性。

### 例题2

**题目**: 根据戈登增长模型，股票内在价值的计算公式为：

**选项**:
- A. P = D / r
- B. P = D₁ / (r - g)
- C. P = D × r
- D. P = D / (r + g)

**答案**: B

**解析**: 戈登增长模型（固定增长模型）的公式为P₀ = D₁ / (r - g)，其中D₁为下一期股利，r为折现率，g为股利增长率。零增长模型的公式为P = D / r。

### 例题3

**题目**: 市盈率的计算公式为：

**选项**:
- A. 市盈率 = 股价 / 每股净资产
- B. 市盈率 = 股价 / 每股收益
- C. 市盈率 = 股价 / 每股销售收入
- D. 市盈率 = 股价 / 每股现金流

**答案**: B

**解析**: 市盈率（P/E）= 股价 / 每股收益。市净率（P/B）= 股价 / 每股净资产。市销率（P/S）= 股价 / 每股销售收入。

### 例题4

**题目**: 科创板IPO前5个交易日的涨跌幅限制为：

**选项**:
- A. ±10%
- B. ±20%
- C. ±30%
- D. 不设限制

**答案**: D

**解析**: 科创板IPO前5个交易日不设涨跌幅限制，之后涨跌幅限制为±20%。创业板注册制改革后也有同样规定。主板涨跌幅限制为±10%，北交所为±30%。

### 例题5

**题目**: 下列不属于相对估值法的是：

**选项**:
- A. 市盈率（P/E）
- B. 市净率（P/B）
- C. 股利贴现模型（DDM）
- D. 市销率（P/S）

**答案**: C

**解析**: 股利贴现模型（DDM）属于绝对估值法，通过预测未来现金流并折现来计算股票价值。市盈率、市净率、市销率属于相对估值法，通过与同类公司比较来估值。
