# 第七章 金融创新与监管

## 本章概要

本章介绍金融创新和金融监管的基本概念。重点掌握资产证券化和中国金融监管体制（"一行一局一会"架构）。分值占比5%-8%。

## 高频考点

> :fire: 高频考点

### 考点1: 资产证券化

**定义**：将缺乏流动性但能够产生可预见的稳定现金流的资产，通过结构性重组，转化为可以在金融市场上销售和流通的证券。

**核心要素**：
- **特殊目的载体（SPV）**：实现破产隔离
- **基础资产**：能够产生现金流的资产
- **资产支持证券（ABS）**：证券化产品

**主要产品**：
| 产品 | 基础资产 |
|------|---------|
| 资产支持证券（ABS） | 企业应收账款、租赁债权等 |
| 住房抵押贷款支持证券（MBS） | 住房抵押贷款 |
| 资产支持票据（ABN） | 企业资产 |

### 考点2: 中国金融监管体制

**"一行一局一会"架构**（2023年改革后）：
| 机构 | 职责 |
|------|------|
| 中国人民银行 | 制定和执行货币政策，维护金融稳定 |
| 国家金融监督管理总局 | 监管银行业、保险业 |
| 中国证券监督管理委员会 | 监管证券期货市场 |

### 考点3: 证券监管目标

- 保护投资者
- 维护市场公平
- 降低系统性风险

## 重点难点

> :warning: 难点

### 难点1: 资产证券化流程

**基本流程**：
1. 发起人将基础资产转让给SPV
2. SPV以基础资产为支撑发行证券
3. 投资者购买证券
4. 基础资产产生的现金流用于偿付证券本息

**关键环节**：
- **破产隔离**：基础资产与发起人破产风险隔离
- **信用增级**：提高证券信用等级
- **真实出售**：资产转让为真实出售

### 难点2: 分业监管与功能监管

**分业监管**：
- 按机构类型进行监管
- 银行、保险、证券分别监管

**功能监管**：
- 按业务功能进行监管
- 相同功能适用相同监管规则

## 知识点详解

### 7.1 金融创新

#### 7.1.1 资产证券化

**定义**：将缺乏流动性但能够产生可预见的稳定现金流的资产，通过结构性重组，转化为可以在金融市场上销售和流通的证券。

**核心要素**：
- **特殊目的载体（SPV）**：实现破产隔离
- **基础资产**：能够产生现金流的资产
- **资产支持证券（ABS）**：证券化产品

**主要产品**：
| 产品 | 基础资产 | 特点 |
|------|---------|------|
| 资产支持证券（ABS） | 企业应收账款、租赁债权等 | 企业融资工具 |
| 住房抵押贷款支持证券（MBS） | 住房抵押贷款 | 房地产金融 |
| 资产支持票据（ABN） | 企业资产 | 银行间市场 |

**资产证券化流程**：
1. 发起人将基础资产转让给SPV
2. SPV以基础资产为支撑发行证券
3. 投资者购买证券
4. 基础资产产生的现金流用于偿付证券本息

**关键环节**：
- **破产隔离**：基础资产与发起人破产风险隔离
- **信用增级**：提高证券信用等级
- **真实出售**：资产转让为真实出售

#### 7.1.2 金融创新的影响

**积极影响**：
- 提高市场效率
- 分散和转移风险
- 满足多样化需求
- 促进金融市场发展

**消极影响**：
- 可能带来新的风险
- 增加市场复杂性
- 可能导致监管套利

### 7.2 金融监管

#### 7.2.1 中国金融监管体制

**"一行一局一会"架构**（2023年改革后）：

**中国人民银行**：
- 制定和执行货币政策
- 维护金融稳定
- 管理外汇储备
- 维护支付系统运行

**国家金融监督管理总局**：
- 监管银行业
- 监管保险业
- 保护金融消费者权益
- 防范化解金融风险

**中国证券监督管理委员会**：
- 监管证券期货市场
- 制定规章规则
- 查处违法违规行为
- 保护投资者权益

#### 7.2.2 证券市场监管

**监管目标**：
1. **保护投资者**
   - 保护投资者合法权益
   - 防止欺诈和操纵
   - 确保市场公平

2. **维护市场公平**
   - 确保信息公平披露
   - 防止内幕交易
   - 维护市场秩序

3. **降低系统性风险**
   - 防范市场风险
   - 维护市场稳定
   - 保护金融体系安全

**监管方式**：
| 方式 | 说明 |
|------|------|
| 注册制 | 以信息披露为核心 |
| 信息披露 | 要求上市公司公开信息 |
| 持续监管 | 对市场持续监督 |
| 行政处罚 | 对违法违规行为处罚 |

**自律管理与行政监管的关系**：
- **行政监管**：证监会及其派出机构
- **自律管理**：证券交易所、证券业协会
- 两者相互配合，共同维护市场秩序

#### 7.2.3 国际金融监管

**主要国际监管组织**：
| 组织 | 职责 |
|------|------|
| 国际证监会组织（IOSCO） | 证券监管国际合作 |
| 巴塞尔银行监管委员会 | 银行监管标准 |
| 金融稳定理事会（FSB） | 维护金融稳定 |

**国际监管趋势**：
- 加强跨境监管合作
- 提高监管标准
- 加强系统性风险防范

## 例题与解析

### 例题1

**题目**: 资产证券化的核心要素不包括：

**选项**:
- A. 特殊目的载体（SPV）
- B. 基础资产
- C. 信用增级
- D. 股票发行

**答案**: D

**解析**: 资产证券化的核心要素包括特殊目的载体（SPV）、基础资产、资产支持证券等。信用增级是提高证券信用等级的重要环节。股票发行不属于资产证券化的核心要素。

### 例题2

**题目**: 2023年金融监管改革后，中国金融监管体制为：

**选项**:
- A. 一行三会
- B. 一行两会
- C. 一行一局一会
- D. 一行两会一局

**答案**: C

**解析**: 2023年金融监管改革后，中国金融监管体制为"一行一局一会"架构，即中国人民银行、国家金融监督管理总局、中国证券监督管理委员会。

### 例题3

**题目**: 证券监管的目标不包括：

**选项**:
- A. 保护投资者
- B. 维护市场公平
- C. 降低系统性风险
- D. 提高市场效率

**答案**: D

**解析**: 证券监管的三大目标是保护投资者、维护市场公平、降低系统性风险。提高市场效率虽然重要，但不是证券监管的核心目标。

### 例题4

**题目**: MBS是指：

**选项**:
- A. 资产支持证券
- B. 住房抵押贷款支持证券
- C. 资产支持票据
- D. 公司债券

**答案**: B

**解析**: MBS（Mortgage-Backed Securities）是指住房抵押贷款支持证券，以住房抵押贷款为基础资产发行的证券化产品。ABS是资产支持证券，ABN是资产支持票据。

### 例题5

**题目**: 资产证券化中，实现破产隔离的关键是：

**选项**:
- A. 信用增级
- B. 真实出售
- C. 特殊目的载体（SPV）
- D. 资产评估

**答案**: C

**解析**: 特殊目的载体（SPV）是实现破产隔离的关键。通过将基础资产转让给SPV，使基础资产与发起人的破产风险隔离，保护投资者利益。真实出售是资产转让的方式，信用增级是提高证券信用等级的手段。
