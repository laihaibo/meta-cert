# 第三章 证券公司业务规范

## 本章概要

本章详细介绍了证券公司的主要业务规范，包括证券经纪、自营、资产管理、融资融券和投资银行等业务。重点掌握融资融券业务规则、资产管理业务规范和经纪业务禁止行为。分值占比20%-25%，是考试的重点章节。

## 高频考点

> :fire: 高频考点

### 考点1: 融资融券业务规则

**保证金与担保比例**：
| 指标 | 标准 |
|------|------|
| 保证金比例 | ≥100% |
| 维持担保比例预警线 | 150% |
| 维持担保比例平仓线 | 130% |

**强制平仓规则**：
- 维持担保比例低于130%时，证券公司应在**2个交易日内**通知客户追加担保物
- 客户追加担保物后的维持担保比例不得低于**150%**
- 客户未在规定时间内追加担保物的，证券公司有权强制平仓

### 考点2: 证券经纪业务禁止行为

- **挪用客户资金**
- **接受客户全权委托**
- **代理客户操作**
- 未经客户委托买卖证券
- 诱导客户进行不必要的证券买卖

### 考点3: 资产管理业务类型

| 类型 | 特点 | 投资者 |
|------|------|--------|
| 定向资产管理 | 为单一客户办理 | 签订定向合同 |
| 集合资产管理 | 为多个客户办理 | 设立集合计划 |
| 专项资产管理 | 特定目的设立 | 特定投资者 |

### 考点4: 合格投资者标准

- 金融资产不低于**300万元**
- 或近3年年均收入不低于**40万元**
- 或法人单位最近1年末净资产不低于**1000万元**

## 重点难点

> :warning: 难点

### 难点1: 自营业务投资限制

- 持有一种权益类证券的成本不得超过净资本的**30%**
- 持有一种权益类证券的市值不得超过该证券总市值的**5%**
- 证券自营业务规模不得超过净资本的**100%**

### 难点2: 资管新规要点

- **打破刚性兑付**：不得承诺保本保收益
- **消除多层嵌套**：资管产品可以再投资一层资管产品
- **禁止资金池**：每只资管产品单独管理、单独建账、单独核算
- **净值化管理**：资管产品应实行净值化管理

## 知识点详解

### 3.1 证券经纪业务

#### 3.1.1 开户与委托

**证券账户开立**：
- 实行**一码通**制度，一个投资者只能开立一个一码通账户
- 开户需进行适当性评估
- 开户需签署风险揭示书

**委托方式**：
| 方式 | 特点 |
|------|------|
| 柜台委托 | 到证券公司柜台办理 |
| 电话委托 | 通过电话系统委托 |
| 网上委托 | 通过互联网委托 |
| 手机委托 | 通过手机APP委托 |

**委托指令类型**：
- **市价委托**：按市场价格买卖，不指定价格
- **限价委托**：指定价格买卖，达到指定价格才能成交

#### 3.1.2 经纪业务规范

**禁止行为**：
1. 挪用客户所委托买卖的证券或者客户账户上的资金
2. 接受客户的全权委托而决定证券买卖、选择证券种类、决定买卖数量或者买卖价格
3. 未经客户的委托，擅自为客户买卖证券
4. 为牟取佣金收入，诱使客户进行不必要的证券买卖
5. 其他违背客户真实意思表示，损害客户利益的行为

**客户信息保密义务**：
- 证券公司应当对客户资料保密
- 非因法定原因不得向任何单位和个人提供客户资料

**风险提示义务**：
- 证券公司应当向客户充分揭示证券投资风险
- 在客户开户时签署风险揭示书

### 3.2 证券自营业务

#### 3.2.1 投资范围

**自营投资品种**：
- 股票
- 债券
- 基金
- 金融衍生品（需满足特定条件）
- 证监会认可的其他证券

#### 3.2.2 风险控制

**投资比例限制**：
| 限制项目 | 比例限制 |
|----------|---------|
| 持有一种权益类证券的成本/净资本 | ≤30% |
| 持有一种权益类证券的市值/该证券总市值 | ≤5% |
| 自营股票规模/净资本 | ≤100% |

**禁止行为**：
- 内幕交易
- 操纵市场
- 使用他人账户自营
- 将自营账户借给他人使用

### 3.3 融资融券业务

#### 3.3.1 基本规则

**保证金制度**：
- 保证金比例不得低于**100%**
- 保证金可以是现金，也可以是可充抵保证金的证券
- 可充抵保证金的证券折算率由交易所规定

**维持担保比例**：
| 指标 | 比例 | 处置措施 |
|------|------|----------|
| 预警线 | 150% | 提醒客户关注 |
| 平仓线 | 130% | 通知追加担保物 |
| 追加后标准 | ≥150% | 追加后的最低标准 |

**标的证券**：
- 由交易所规定
- 需满足流通股本、市值、股东人数等条件
- 交易所定期公布标的证券名单

#### 3.3.2 风险控制

**逐日盯市制度**：
- 证券公司应当逐日计算客户维持担保比例
- 维持担保比例低于预警线时，通知客户关注

**强制平仓制度**：
- 维持担保比例低于平仓线（130%）
- 证券公司应在2个交易日内通知客户追加担保物
- 客户未在规定时间内追加的，证券公司有权强制平仓

**限仓制度**：
- 单一客户融资规模不得超过证券公司净资本的一定比例
- 全部客户融资规模不得超过证券公司净资本的一定比例

### 3.4 证券资产管理业务

#### 3.4.1 业务类型

**定向资产管理**：
- 为单一客户办理
- 签订定向资产管理合同
- 投资范围由合同约定

**集合资产管理**：
- 为多个客户办理
- 设立集合资产管理计划
- 投资范围由计划说明书约定

**专项资产管理**：
- 特定目的设立
- 如资产证券化业务

#### 3.4.2 资管新规要点

**核心要求**：
| 要点 | 具体内容 |
|------|---------|
| 打破刚性兑付 | 不得承诺保本保收益 |
| 消除多层嵌套 | 可再投资一层资管产品 |
| 禁止资金池 | 单独管理、单独建账、单独核算 |
| 统一监管标准 | 消除监管套利 |
| 净值化管理 | 定期公布净值 |

**合格投资者标准**：
- **个人**：金融资产不低于300万元，或近3年年均收入不低于40万元
- **法人**：最近1年末净资产不低于1000万元
- **非法人组织**：最近1年末净资产不低于1000万元

#### 3.4.3 投资限制

**集合计划投资限制**：
- 不得违规将集合计划资产用于资金拆借、贷款、抵押融资或者对外担保
- 不得违规将集合计划资产用于可能承担无限责任的投资
- 单个集合计划持有一家公司发行的证券不得超过集合计划资产净值的**10%**

### 3.5 投资银行业务

#### 3.5.1 保荐制度

**保荐机构资格**：
- 注册资本不低于1亿元
- 净资本不低于5000万元
- 具有完善的内部控制制度

**保荐代表人资格**：
- 具有证券从业资格
- 从事证券发行、交易相关业务3年以上
- 最近3年内在境内证券发行项目中担任过项目协办人

**保荐职责**：
- 尽职推荐发行人证券发行上市
- 持续督导发行人履行相关义务

**持续督导期限**：
| 项目 | 主板 | 科创板/创业板 |
|------|------|--------------|
| IPO | 上市当年剩余时间+2个完整会计年度 | 上市当年剩余时间+3个完整会计年度 |
| 再融资 | 上市当年剩余时间+1个完整会计年度 | 上市当年剩余时间+2个完整会计年度 |

#### 3.5.2 承销方式

| 方式 | 代销 | 包销 |
|------|------|------|
| 定义 | 承销期结束，未售出的证券退还发行人 | 承销商购买未售出的证券 |
| 风险承担 | 发行人承担 | 承销商承担 |
| 承销期限 | 最长90天 | 最长90天 |

**超额配售选择权（绿鞋机制）**：
- 发行人授予主承销商的一项选择权
- 主承销商可以超额发售不超过包销数额15%的股份
- 目的：稳定新股上市后的股价

### 3.6 财务顾问业务

#### 3.6.1 财务顾问的监管

**资格条件**：
- 具有健全且运行良好的内部控制机制和管理制度
- 具有良好的诚信记录，最近3年无重大违法违规记录
- 具有担任财务顾问所需的专业团队

**财务顾问职责**：
- 对上市公司收购及相关股份权益变动活动进行尽职调查
- 对收购人的主体资格、收购目的、实力、诚信记录等进行核查
- 出具专业意见

**法律责任**：
- 财务顾问未勤勉尽责，所制作、出具的文件有虚假记载、误导性陈述或者重大遗漏的，责令改正，没收业务收入，并处以业务收入1倍以上5倍以下的罚款

## 例题与解析

### 例题1

**题目**: 证券公司融资融券业务中，维持担保比例的平仓线为：

**选项**:
- A. 100%
- B. 130%
- C. 150%
- D. 200%

**答案**: B

**解析**: 融资融券业务中，维持担保比例的预警线为150%，平仓线为130%。当维持担保比例低于130%时，证券公司应在2个交易日内通知客户追加担保物，使维持担保比例不低于150%。客户未在规定时间内追加的，证券公司有权强制平仓。

### 例题2

**题目**: 下列属于证券经纪业务禁止行为的是：

**选项**:
- A. 向客户揭示投资风险
- B. 接受客户全权委托
- C. 对客户资料保密
- D. 按照客户指令买卖证券

**答案**: B

**解析**: 证券经纪业务禁止接受客户全权委托，即不得由证券公司决定证券买卖、选择证券种类、决定买卖数量或者买卖价格。证券公司应当按照客户的委托买卖证券，并向客户充分揭示投资风险，对客户资料保密。

### 例题3

**题目**: 证券公司持有一种权益类证券的成本不得超过净资本的：

**选项**:
- A. 10%
- B. 20%
- C. 30%
- D. 50%

**答案**: C

**解析**: 根据证券自营业务的风险控制要求，证券公司持有一种权益类证券的成本不得超过净资本的30%，持有一种权益类证券的市值不得超过该证券总市值的5%。

### 例题4

**题目**: 资管新规中，合格投资者的金融资产门槛为：

**选项**:
- A. 100万元
- B. 200万元
- C. 300万元
- D. 500万元

**答案**: C

**解析**: 根据资管新规，合格投资者的个人金融资产不低于300万元，或近3年年均收入不低于40万元。法人单位最近1年末净资产不低于1000万元。

### 例题5

**题目**: 关于保荐制度，下列表述错误的是：

**选项**:
- A. 保荐机构注册资本不低于1亿元
- B. 保荐代表人需具有证券从业资格
- C. IPO持续督导期限为上市当年剩余时间+2个完整会计年度（主板）
- D. 保荐机构可以同时为发行人提供审计服务

**答案**: D

**解析**: 保荐机构不得同时为发行人提供审计服务，以保证保荐业务的独立性。保荐机构注册资本不低于1亿元，保荐代表人需具有证券从业资格，主板IPO持续督导期限为上市当年剩余时间加2个完整会计年度。
