# 证券市场基本法律法规

## 科目介绍

《证券市场基本法律法规》是证券从业资格考试的科目一，主要考查考生对证券市场相关法律法规、行业规范和职业道德的掌握程度。考试形式为机考，共100道单选题（含组合型单选），总分100分，60分及格，考试时间120分钟。

## 备考建议

1. **重点章节优先**：第一章和第三章分值占比最高（合计40%-50%），应优先掌握
2. **数字记忆**：罚款金额、时间期限、比例限制等数字是高频考点
3. **对比记忆**：不同违法行为的法律责任、不同业务的资格条件需对比记忆
4. **刷题巩固**：建议结合题库反复练习，特别是易混淆的知识点
