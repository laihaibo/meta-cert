# 第二章 证券经营机构管理规范

## 本章概要

本章主要介绍证券公司的设立条件、业务范围、风险控制指标、合规管理和客户资产保护等内容。重点掌握净资本管理、风险控制指标和合规管理体系。分值占比15%-20%。

## 高频考点

> :fire: 高频考点

### 考点1: 证券公司设立条件

**主要股东条件**：
- 净资产不低于人民币2亿元
- 最近3年无重大违法违规记录
- 具有持续盈利能力

**注册资本要求**：
| 业务类型 | 最低注册资本 |
|----------|-------------|
| 证券经纪 | 5000万元 |
| 证券投资咨询 | 5000万元 |
| 证券承销与保荐 | 1亿元 |
| 证券自营 | 1亿元 |
| 证券资产管理 | 1亿元 |
| 融资融券 | 1亿元 |

### 考点2: 风险控制指标

**四大核心指标**：
| 指标 | 标准 | 含义 |
|------|------|------|
| 风险覆盖率 | ≥100% | 净资本/各项风险资本准备之和 |
| 资本杠杆率 | ≥8% | 核心净资本/表内外资产总额 |
| 流动性覆盖率 | ≥100% | 优质流动性资产/未来30天现金净流出量 |
| 净稳定资金率 | ≥100% | 可用稳定资金/所需稳定资金 |

### 考点3: 净资本计算

**净资本 = 净资产 - 金融资产的风险调整 - 其他资产的风险调整 - 或有负债的风险调整 - 中国证监会认定的其他调整项目**

### 考点4: 分类监管

证券公司分为**A（AAA、AA、A）、B（BBB、BB、B）、C（CCC、CC、C）、D、E**共11个级别。

## 重点难点

> :warning: 难点

### 难点1: 净资本与业务规模的比例要求

- 净资本与净资产的比例不得低于**40%**
- 净资本与负债的比例不得低于**8%**
- 净资产与负债的比例不得低于**10%**

### 难点2: 合规总监的职责与独立性

- 合规总监由董事会聘任和解聘
- 合规总监直接向董事会负责
- 合规总监不得兼任与合规管理职责相冲突的职务
- 合规总监有权列席公司有关会议

## 知识点详解

### 2.1 证券公司设立与管理

#### 2.1.1 设立条件

**基本条件**：
- 有符合法律、行政法规规定的公司章程
- 主要股东及控股股东具有持续盈利能力，信誉良好，最近3年无重大违法违规记录，净资产不低于人民币2亿元
- 有符合《证券法》规定的注册资本
- 董事、监事、高级管理人员具备任职资格，从业人员具有证券从业资格
- 有完善的风险管理与内部控制制度
- 有合格的经营场所和业务设施

**高级管理人员任职资格**：
- 正直诚实，品行良好
- 熟悉证券法律、行政法规
- 具有大学本科以上学历
- 从事证券工作3年以上，或从事金融、法律、会计工作5年以上

#### 2.1.2 业务范围

证券公司可以经营下列部分或者全部业务：
1. 证券经纪
2. 证券投资咨询
3. 与证券交易、证券投资活动有关的财务顾问
4. 证券承销与保荐
5. 证券自营
6. 证券资产管理
7. 融资融券
8. 证券做市交易
9. 其他证券业务

### 2.2 风险控制指标

#### 2.2.1 净资本管理

**净资本计算公式**：
```
净资本 = 净资产
       - 金融资产的风险调整
       - 其他资产的风险调整
       - 或有负债的风险调整
       - 中国证监会认定的其他调整项目
       + 中国证监会认定的其他调整项目
```

**净资本的最低标准**：
- 经纪业务：净资本不低于**2000万元**
- 承销与保荐、自营、资产管理、其他业务之一：净资本不低于**1亿元**
- 经纪+承销与保荐、自营、资产管理、其他业务中两项以上：净资本不低于**2亿元**

#### 2.2.2 风险控制指标体系

| 指标 | 计算公式 | 预警标准 | 监管标准 |
|------|---------|----------|----------|
| 风险覆盖率 | 净资本/各项风险资本准备之和 | ≥120% | ≥100% |
| 资本杠杆率 | 核心净资本/表内外资产总额 | ≥9.6% | ≥8% |
| 流动性覆盖率 | 优质流动性资产/未来30天现金净流出量 | ≥120% | ≥100% |
| 净稳定资金率 | 可用稳定资金/所需稳定资金 | ≥120% | ≥100% |

#### 2.2.3 分类监管

**评价指标**：
- 资本充足
- 公司治理与合规管理
- 全面风险管理
- 信息系统安全
- 客户权益保护
- 信息披露

**分类结果**：
| 级别 | 含义 | 监管措施 |
|------|------|----------|
| A类（AAA、AA、A） | 风险管理能力在行业内最高 | 常规监管 |
| B类（BBB、BB、B） | 风险管理能力在行业内较高 | 常规监管 |
| C类（CCC、CC、C） | 风险管理能力在行业内与其现有业务相匹配 | 关注监管 |
| D类 | 风险管理能力在行业内较低 | 重点关注 |
| E类 | 风险管理能力在行业内极低 | 风险处置 |

### 2.3 合规管理

#### 2.3.1 合规管理体系

**合规管理架构**：
- **董事会**：对合规管理承担最终责任
- **高级管理人员**：负责落实合规管理目标
- **合规总监**：全面负责合规管理工作
- **合规管理部门**：具体执行合规管理工作

**合规总监职责**：
- 对公司及其工作人员的经营管理和执业行为的合规性进行审查、监督和检查
- 组织制定合规管理的基本制度
- 对公司内部规章制度、重大决策、新产品和新业务方案进行合规审查
- 组织合规培训
- 处理涉及公司和工作人员违法违规行为的投诉和举报

**合规总监的独立性**：
- 由董事会聘任和解聘
- 直接向董事会负责
- 不得兼任与合规管理职责相冲突的职务
- 有权列席公司有关会议
- 有权查阅公司相关文件、资料

#### 2.3.2 合规管理制度

**合规审查**：
- 对公司内部规章制度进行合规审查
- 对公司重大决策进行合规审查
- 对新产品和新业务方案进行合规审查

**合规检查**：
- 定期或不定期对公司各部门、分支机构进行合规检查
- 检查结果应向董事会报告

**合规报告**：
- 合规总监每年向董事会提交合规报告
- 发现重大合规风险应及时报告

### 2.4 客户资产保护

#### 2.4.1 三方存管制度

**客户交易结算资金第三方存管**：
- 证券公司将客户交易结算资金存放在指定的商业银行
- 以每个客户的名义单独立户管理
- 证券公司不得挪用客户交易结算资金

**存管银行的选择**：
- 证券公司应选择符合条件的商业银行作为存管银行
- 存管银行应具备健全的内部控制制度
- 存管银行应具备安全的资金划转系统

#### 2.4.2 客户资产隔离

- 证券公司客户资产与证券公司自有资产相互独立
- 证券公司破产或清算时，客户资产不属于破产财产或清算财产
- 不同客户的资产相互独立

## 例题与解析

### 例题1

**题目**: 根据规定，证券公司的风险覆盖率不得低于：

**选项**:
- A. 80%
- B. 100%
- C. 120%
- D. 150%

**答案**: B

**解析**: 风险覆盖率 = 净资本/各项风险资本准备之和，监管标准为不得低于100%，预警标准为120%。这是证券公司四大核心风险控制指标之一，需要牢记。

### 例题2

**题目**: 下列关于证券公司合规总监的说法，错误的是：

**选项**:
- A. 合规总监由董事会聘任和解聘
- B. 合规总监直接向董事会负责
- C. 合规总监可以兼任公司财务负责人
- D. 合规总监有权列席公司有关会议

**答案**: C

**解析**: 合规总监不得兼任与合规管理职责相冲突的职务。财务负责人属于高级管理人员，其职责可能与合规管理存在冲突，因此合规总监不得兼任。合规总监由董事会聘任和解聘，直接向董事会负责，有权列席公司有关会议。

### 例题3

**题目**: 证券公司经营证券经纪业务的，净资本不得低于人民币：

**选项**:
- A. 1000万元
- B. 2000万元
- C. 5000万元
- D. 1亿元

**答案**: B

**解析**: 根据《证券公司风险控制指标管理办法》，证券公司经营证券经纪业务的，净资本不得低于人民币2000万元。经营承销与保荐、自营、资产管理等业务之一的，净资本不得低于1亿元。经营两项以上业务的，净资本不得低于2亿元。

### 例题4

**题目**: 证券公司分类评价共分为几个级别？

**选项**:
- A. 5个
- B. 7个
- C. 9个
- D. 11个

**答案**: D

**解析**: 证券公司分为A（AAA、AA、A）、B（BBB、BB、B）、C（CCC、CC、C）、D、E共11个级别。评价指标包括资本充足、公司治理与合规管理、全面风险管理、信息系统安全、客户权益保护、信息披露等六个方面。

### 例题5

**题目**: 关于客户交易结算资金第三方存管，下列表述正确的是：

**选项**:
- A. 客户交易结算资金由证券公司自行管理
- B. 客户交易结算资金应存放在指定的商业银行
- C. 证券公司可以临时借用客户交易结算资金
- D. 不同客户的交易结算资金可以混合管理

**答案**: B

**解析**: 客户交易结算资金第三方存管制度要求证券公司将客户交易结算资金存放在指定的商业银行，以每个客户的名义单独立户管理。证券公司不得挪用客户交易结算资金，不同客户的资产应当相互独立。
