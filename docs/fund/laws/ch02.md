# 第二章：证券投资基金概述

> 分值占比：8%-10% | 重要程度：★★★★

## 考情分析

本章是基金法律法规的核心章节，主要介绍证券投资基金的概念、运作机制、参与主体及其关系。考试重点在于基金运作的基本框架和各参与主体的权利义务。

## 2.1 证券投资基金的概念与特点

### 2.1.1 证券投资基金的定义

证券投资基金（简称"基金"）是指通过发售基金份额，将众多投资者的资金集中起来，形成独立财产，由基金管理人管理、基金托管人托管，以投资组合的方式进行证券投资，基金份额持有人按其所持份额享受收益和承担风险的集合投资方式。

### 2.1.2 证券投资基金的特点

| 特点 | 说明 |
|------|------|
| 集合理资、专业管理 | 汇集众多投资者资金，由专业基金管理人管理，降低投资门槛 |
| 组合投资、分散风险 | 通过构建投资组合分散非系统性风险 |
| 利益共享、风险共担 | 投资者按持有份额比例享有收益和承担风险 |
| 严格监管、信息透明 | 受证监会严格监管，定期披露信息 |
| 独立托管、保障安全 | 基金财产独立于管理人和托管人固有财产 |

### 2.1.3 基金财产的独立性

**核心原则：基金财产独立于各方固有财产**

- 基金财产独立于基金管理人的固有财产
- 基金财产独立于基金托管人的固有财产
- 基金管理人、托管人不得将基金财产归入其固有财产
- 基金管理人、托管人因依法解散、被依法撤销或被依法宣告破产等原因进行清算的，基金财产不属于其清算财产
- 基金财产的债权，不得与基金管理人、托管人固有财产的债务相抵销
- 不同基金财产的债权债务，不得相互抵销

## 2.2 基金的参与主体

### 2.2.1 基金当事人

**1. 基金份额持有人（投资者）**
- 基金的出资人、基金资产的所有者、基金投资回报的受益人
- **权利：**
  - 分享基金财产收益
  - 参与分配清算后的剩余基金财产
  - 依法转让或者申请赎回其持有的基金份额
  - 按照规定要求召开基金份额持有人大会
  - 对基金份额持有人大会审议事项行使表决权
  - 查阅或者复制公开披露的基金信息资料
  - 对基金管理人、托管人、销售机构损害其合法权益的行为依法提起诉讼

**2. 基金管理人（基金公司）**
- 负责基金的投资管理
- **职责：**
  - 依法募集基金，办理基金份额的发售和登记事宜
  - 办理基金备案手续
  - 对所管理的不同基金财产分别管理、分别记账
  - 按照基金合同的约定确定基金收益分配方案
  - 进行基金会计核算并编制基金财务会计报告
  - 编制中期和年度基金报告
  - 计算并公告基金资产净值
  - 办理与基金财产管理业务活动有关的信息披露事项
  - 保存基金财产管理业务活动的记录、账册、报表和其他相关资料

**3. 基金托管人（商业银行）**
- 负责基金资产的保管和监督
- **职责：**
  - 安全保管基金财产
  - 按照规定开设基金财产的资金账户和证券账户
  - 对所托管的不同基金财产分别设置账户
  - 保存基金托管业务活动的记录、账册、报表和其他相关资料
  - 按照基金合同的约定，根据基金管理人的投资指令，及时办理清算、交割事宜
  - 办理与基金托管业务活动有关的信息披露事项
  - 对基金财务会计报告、中期和年度基金报告出具意见
  - 复核、审查基金管理人计算的基金资产净值和基金份额申购、赎回价格
  - 按照规定召集基金份额持有人大会
  - 按照规定监督基金管理人的投资运作

### 2.2.2 基金市场服务机构

| 服务机构 | 主要职责 |
|---------|---------|
| 基金销售机构 | 基金份额的发售、申购、赎回等 |
| 基金销售支付机构 | 基金销售资金的支付结算 |
| 基金份额登记机构 | 基金份额的登记、存管、过户 |
| 基金估值核算机构 | 基金资产估值、核算 |
| 基金投资顾问机构 | 为投资者提供投资建议 |
| 基金评价机构 | 对基金进行评级和评价 |
| 律师事务所 | 出具法律意见书 |
| 会计师事务所 | 审计基金财务报告 |

### 2.2.3 基金监管机构

- **中国证监会**：基金行业的主管部门
- **中国证券投资基金业协会**：基金行业的自律组织

## 2.3 基金的运作方式

### 2.3.1 封闭式基金与开放式基金的比较

| 比较项目 | 封闭式基金 | 开放式基金 |
|---------|-----------|-----------|
| 份额总额 | 合同期限内固定不变 | 不固定，随申购赎回变动 |
| 期限 | 有固定封闭期 | 无固定期限 |
| 交易方式 | 证券交易所上市交易 | 通过基金管理人或代销机构申购赎回 |
| 价格形成 | 市场供求决定（可能溢价或折价） | 按基金份额净值 |
| 信息披露 | 每周公布一次份额净值 | 每个交易日公布份额净值 |
| 投资策略 | 可进行长期投资 | 需保留一定现金应对赎回 |

### 2.3.2 基金的运作流程

```
基金募集 → 基金成立 → 基金运作（投资管理） → 基金信息披露 → 基金终止
```

## 2.4 基金的起源与发展

### 2.4.1 基金的起源

- **1868年**：英国成立"海外及殖民地政府信托"，被公认为最早的基金
- **1924年**：美国成立"马萨诸塞投资信托基金"，是第一只开放式基金
- **1940年**：美国《投资公司法》颁布，奠定基金监管法律基础

### 2.4.2 中国基金业的发展

| 时间 | 事件 |
|------|------|
| 1998年 | 国泰基金、南方基金成立，首批封闭式基金发行 |
| 2001年 | 首只开放式基金"华安创新"成立 |
| 2004年 | 《证券投资基金法》正式实施 |
| 2012年 | 《证券投资基金法》修订，私募基金纳入监管 |
| 2013年 | 新《证券投资基金法》正式实施 |
