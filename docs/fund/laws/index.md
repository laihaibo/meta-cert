| 项目 | 详情 |
|------|------|
| 科目名称 | 基金法律法规、职业道德与业务规范 |

| 题量 | 100题 |

## 必背数字考点

| 类型 | 考点 | 数值 |
|------|------|------|
| 时间 | 基金管理人变更重大事项报告 | 5个工作日内 |
| 时间 | 年报披露 | 90日内 |
| 时间 | 半年报披露 | 60日内 |
| 时间 | 季报披露 | 15日内 |
| 比例 | 单只基金持有一家公司证券 | ≤基金资产净值10% |
| 比例 | 股票基金投资股票比例 | ≥80% |
| 金额 | 基金合同生效最低募集规模 | 2亿份/2亿元 |
| 人数 | 公开募集基金持有人 | ≥200人 |
| 金额 | 私募合格投资者（个人） | 金融资产≥300万或年均收入≥50万 |

## 备考建议

1. **重点章节**：第六章（基金募集、交易与登记）分值最高，务必重点掌握
2. **数字考点**：本科目涉及大量时间、比例、金额等数字考点，需专门记忆
3. **法规条文**：注意关键法规条文的具体规定
4. **职业道德**：第五章职业道德部分虽然分值不高，但考试常考，不可忽视
