# 第六章：基金的募集、交易与登记

> 分值占比：10%-12% | 重要程度：★★★★★

## 考情分析

本章是本科目最重要的章节，分值占比最高。重点掌握基金募集的条件和程序、基金合同生效条件、封闭式基金上市交易条件、开放式基金申购赎回规则等。数字考点密集，务必重点记忆。

## 6.1 基金的募集

### 6.1.1 基金募集的条件

**公开募集基金的条件：**
- 已经取得基金托管资格的商业银行担任基金托管人
- 基金管理人已取得基金管理资格
- 基金合同草案、招募说明书草案、托管协议草案已获批准
- 基金份额发售公告已获批准
- 中国证监会规定的其他条件

### 6.1.2 基金募集的程序

```
申请 → 注册 → 发售 → 基金合同生效
```

**1. 申请**
- 向中国证监会提交募集申请材料
- 包括：基金合同草案、招募说明书草案、托管协议草案等

**2. 注册**
- 中国证监会自收到完整申请材料之日起6个月内作出批准或不批准的决定
- 对常规基金产品，按照简易程序注册

**3. 发售**
- 基金管理人应自收到核准文件之日起6个月内进行基金募集
- 基金募集期限不得超过3个月
- 基金募集期限自基金份额发售之日起计算

**4. 基金合同生效**
- **封闭式基金**：募集份额总额达到核准规模的**80%以上**
- **开放式基金**：募集份额总额不少于**2亿份**，募集金额不少于**2亿元**人民币
- 基金份额持有人的人数不少于**200人**

### 6.1.3 基金募集失败

**情形：**
- 基金募集期限届满，不满足基金合同生效条件
- 基金管理人应以固有财产承担因募集行为而产生的债务和费用
- 在基金募集期限届满后30日内返还投资者已缴纳的款项，并加计银行同期存款利息

## 6.2 封闭式基金的上市交易

### 6.2.1 上市交易条件

- 基金的募集符合《基金法》规定
- 基金合同期限为5年以上
- 基金募集金额不低于2亿元人民币
- 基金份额持有人不少于1000人
- 基金份额上市交易规则规定的其他条件

### 6.2.2 交易规则

| 项目 | 内容 |
|------|------|
| 交易场所 | 证券交易所 |
| 交易方式 | 竞价交易 |
| 涨跌幅限制 | 10%（上市首日除外） |
| 交易单位 | 100份基金份额为1手 |
| 价格形成 | 市场供求决定 |

### 6.2.3 终止上市交易

- 不再具备上市交易条件
- 基金合同期限届满
- 基金份额持有人大会决定提前终止上市交易
- 基金合同约定的其他情形

## 6.3 开放式基金的申购与赎回

### 6.3.1 申购与赎回的原则

- **"未知价"原则**：申购、赎回价格以申请当日收市后计算的基金份额净值为基准
- **"金额申购、份额赎回"原则**：申购以金额申请，赎回以份额申请

### 6.3.2 申购与赎回的费用

**申购费：**
- 不超过申购金额的**5%**
- 前端收费模式：申购时扣除
- 后端收费模式：赎回时扣除，持有时间越长费率越低

**赎回费：**
- 不超过赎回金额的**5%**
- 赎回费在扣除手续费后，余额归入基金财产
- 对持续持有期少于7日的投资者收取不低于**1.5%**的赎回费

### 6.3.3 申购与赎回的确认

- 基金管理人应在**T+1日**内对申购、赎回申请进行确认
- 投资者应在**T+2日**后查询申购、赎回的确认情况

### 6.3.4 巨额赎回

**定义：** 单个开放日基金净赎回申请超过基金总份额的**10%**时，为巨额赎回。

**处理方式：**
- **接受全额赎回**：基金管理人认为有能力全额支付时
- **部分延期赎回**：基金管理人认为无法全额支付时
  - 基金管理人当日办理不低于上一日基金总份额**10%**的赎回
  - 对其余部分延期办理
  - 延期办理的赎回申请转到下一个开放日处理，享有赎回优先权

## 6.4 基金份额的登记

### 6.4.1 登记机构

- 基金份额登记机构由基金管理人或其委托的机构担任
- 主要职责：基金份额的登记、存管、过户

### 6.4.2 登记流程

```
投资者提交申请 → 销售机构确认 → 登记机构办理登记 → 完成登记
```

---

## 6.5 基金的转换、非交易过户与转托管

### 6.5.1 基金转换

- 将持有的基金份额转换为同一基金管理人管理的另一基金的基金份额
- 通常在同一销售机构办理
- 转换费用包括转出基金的赎回费和申购补差费

### 6.5.2 非交易过户

- 不通过申购、赎回而进行的基金份额过户
- 包括：继承、捐赠、司法强制执行等

### 6.5.3 转托管

- 将基金份额从一个销售机构转到另一个销售机构
- 场外转托管、跨系统转托管

## 考点速记

::: tip 高频考点
1. 基金合同生效条件（募集规模、持有人人数）
2. 封闭式基金上市交易条件
3. 开放式基金申购赎回原则（未知价、金额申购份额赎回）
4. 巨额赎回的定义和处理方式
5. 基金转换、非交易过户、转托管
:::

::: warning 数字考点
- 基金合同生效：募集规模≥**2亿份/2亿元**，持有人≥**200人**
- 封闭式基金上市：募集金额≥**2亿元**，持有人≥**1000人**
- 基金合同期限：≥**5年**
- 巨额赎回：净赎回申请>基金总份额的**10%**
- 巨额赎回处理：办理不低于上一日总份额**10%**的赎回
- 持有少于7日的赎回费：≥**1.5%**
- 申购费上限：不超过申购金额的**5%**
- 赎回费上限：不超过赎回金额的**5%**
- 中国证监会审批时限：**6个月**
- 基金募集期限：≤**3个月**
- 基金募集失败退款：**30日**内返还
:::
