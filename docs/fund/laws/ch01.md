# 第一章：金融、资产管理与投资基金

> 分值占比：5%-8% | 重要程度：★★★

## 考情分析

本章是基金法律法规的基础章节，主要介绍金融市场的基本概念、资产管理行业的特点以及投资基金的定义与分类。考试以概念理解为主，需掌握基本定义和分类。

---

## 1.1 金融与资产管理概述

### 1.1.1 金融市场的分类

**按交易标的物分类：**
- **货币市场**：短期（1年以内）金融工具交易市场
  - 包括：同业拆借市场、回购协议市场、票据市场、大额可转让存单市场
- **资本市场**：中长期（1年以上）金融工具交易市场
  - 包括：股票市场、债券市场、基金市场
- **外汇市场**：外汇买卖交易的市场
- **金融衍生品市场**：金融衍生工具交易的市场
  - 包括：期货市场、期权市场、远期市场、互换市场

**按发行流通分类：**
- **一级市场（发行市场）**：证券首次发行的市场
- **二级市场（流通市场）**：已发行证券买卖交易的市场

**按组织方式分类：**
- **场内市场（交易所市场）**：有固定交易场所、组织制度
- **场外市场（OTC市场）**：无固定交易场所，分散交易

### 1.1.2 金融市场的功能

1. **资金融通功能**：实现资金从盈余方到短缺方的转移
2. **价格发现功能**：通过交易形成资产的市场价格
3. **风险管理功能**：通过衍生工具转移和分散风险
4. **提供流动性功能**：为金融资产提供变现渠道
5. **降低交易成本功能**：减少信息不对称和搜寻成本

### 1.1.3 金融资产与金融机构

**金融资产分类：**
- 基础金融资产：股票、债券、货币
- 衍生金融资产：期货、期权、互换

**金融机构分类：**
- **存款性金融机构**：商业银行、储蓄机构
- **非存款性金融机构**：保险公司、证券公司、基金公司、信托公司

## 1.2 资产管理行业

### 1.2.1 资产管理的定义

资产管理是指投资者将资金委托给专业机构，由专业机构按照约定的方式和范围进行投资管理，以实现资产保值增值的行为。

**核心要素：**
- 委托人（投资者）
- 受托人（资产管理机构）
- 托管人（资产托管机构）
- 投资标的

### 1.2.2 资产管理行业的特点

1. **专业性**：需要专业的投资研究和风险管理能力
2. **受托责任**：管理人对投资者负有信义义务
3. **组合投资**：通过分散投资降低风险
4. **利益冲突管理**：需要严格防范利益冲突
5. **监管严格**：受到金融监管部门的严格监管

### 1.2.3 中国资产管理行业格局

**主要机构类型：**
| 机构类型 | 监管部门 | 主要产品 |
|---------|---------|---------|
| 公募基金 | 证监会 | 公募基金产品 |
| 私募基金 | 证监会/基金业协会 | 私募基金产品 |
| 银行理财 | 银保监会 | 银行理财产品 |
| 信托计划 | 银保监会 | 信托产品 |
| 券商资管 | 证监会 | 集合/定向资产管理计划 |
| 保险资管 | 银保监会 | 保险资管产品 |

## 1.3 投资基金概述

### 1.3.1 投资基金的定义

投资基金是通过发售基金份额，将众多投资者的资金集中起来，形成独立财产，由基金管理人管理、基金托管人托管，以投资组合的方式进行证券投资的一种利益共享、风险共担的集合投资方式。

### 1.3.2 投资基金的特征

1. **集合理资、专业管理**：汇集众多投资者资金，由专业基金管理人管理
2. **组合投资、分散风险**：通过投资组合分散非系统性风险
3. **利益共享、风险共担**：投资者按持有份额承担收益和风险
4. **严格监管、信息透明**：受到监管机构严格监管，信息披露要求高
5. **独立托管、保障安全**：基金财产由独立托管人托管

### 1.3.3 投资基金的分类

**按运作方式分类：**
- **封闭式基金**：基金份额总额在合同期限内固定不变
- **开放式基金**：基金份额总额不固定，投资者可申购赎回

**按投资对象分类：**
- **股票基金**：投资股票比例≥80%
- **债券基金**：投资债券比例≥80%
- **货币市场基金**：投资货币市场工具
- **混合基金**：投资于股票、债券和货币市场工具
- **基金中基金（FOF）**：80%以上投资于其他基金份额

**按募集方式分类：**
- **公募基金**：面向社会公众公开发售
- **私募基金**：面向特定投资者非公开发售

**按法律形式分类：**
- **契约型基金**：基于基金合同设立
- **公司型基金**：依据基金公司章程设立
- **合伙型基金**：依据合伙协议设立（主要为私募基金）
