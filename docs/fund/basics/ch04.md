# 第四章：另类投资

> 分值占比：5%-8% | 重要程度：★★★

## 考情分析

本章介绍另类投资的主要类型和特点。考试重点在于各类另类投资的定义、特点、风险收益特征等。

---

## 4.1 另类投资概述

### 4.1.1 另类投资的定义

另类投资是指传统股票、债券和现金之外的投资类型，包括私募股权、房地产、大宗商品、对冲基金等。

### 4.1.2 另类投资的特点

| 特点 | 说明 |
|------|------|
| 流动性较差 | 通常有较长的锁定期 |
| 信息透明度低 | 信息披露要求较低 |
| 杠杆率较高 | 通常使用较高杠杆 |
| 与传统资产相关性低 | 可分散投资组合风险 |
| 收益潜力较高 | 可能获得较高收益 |

---

## 4.2 私募股权投资

### 4.2.1 私募股权投资的定义

私募股权投资（PE）是指对非上市企业进行的权益性投资，通过上市、并购或管理层回购等方式退出并获利。

### 4.2.2 私募股权投资的类型

| 类型 | 说明 |
|------|------|
| 风险投资(VC) | 投资于初创期企业 |
| 成长资本 | 投资于成长期企业 |
| 并购基金 | 投资于成熟期企业，进行并购重组 |
| 夹层资本 | 介于股权和债权之间的投资 |

### 4.2.3 私募股权投资的退出方式

- **首次公开发行(IPO)**：收益最高，但时间最长
- **兼并收购(M&A)**：收益较高，时间较短
- **管理层回购(MBO)**：收益较低，时间较短
- **破产清算**：收益最低，可能亏损

---

## 4.3 房地产投资

### 4.3.1 房地产投资的类型

- **直接投资**：购买房地产实物
- **间接投资**：投资房地产股票、REITs等

### 4.3.2 房地产投资信托(REITs)

**特点：**
- 收益主要来自租金收入
- 强制分红（通常90%以上收益分红）
- 流动性较好
- 与股票、债券相关性低

---

## 4.4 大宗商品投资

### 4.4.1 大宗商品的类型

- **能源类**：原油、天然气、煤炭
- **金属类**：黄金、白银、铜
- **农产品类**：玉米、大豆、小麦

### 4.4.2 大宗商品投资的特点

- **抗通胀**：商品价格与通胀正相关
- **与传统资产低相关**：可分散投资组合风险
- **波动性较大**：受供需影响大

---

## 4.5 对冲基金

### 4.5.1 对冲基金的特点

- 使用多种投资策略
- 使用杠杆和衍生品
- 追求绝对收益
- 收取业绩报酬

### 4.5.2 对冲基金的策略

| 策略 | 说明 |
|------|------|
| 股票多空 | 做多看涨股票，做空看跌股票 |
| 事件驱动 | 投资于公司事件（并购、重组等） |
| 宏观策略 | 基于宏观经济判断投资 |
| 相对价值 | 利用相关资产价格差异套利 |

---

## 4.6 另类投资的风险

### 4.6.1 主要风险

- **流动性风险**：难以以合理价格变现
- **信息不对称风险**：信息披露不充分
- **杠杆风险**：放大损失
- **估值风险**：估值困难，可能存在偏差
- **监管风险**：监管政策变化
