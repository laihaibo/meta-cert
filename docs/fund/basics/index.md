## 核心计算公式

| 公式 | 名称 | 说明 |
|------|------|------|
| S = (Rp - Rf) / σp | 夏普比率 | 单位风险超额收益 |
| T = (Rp - Rf) / βp | 特雷诺比率 | 单位系统风险超额收益 |
| α = Rp - [Rf + β(Rm - Rf)] | 詹森α | 超额收益 |
| IR = α / σ(残差) | 信息比率 | 单位主动风险超额收益 |
| Dmod = D / (1+y) | 修正久期 | 利率敏感性 |
| P = D1 / (k - g) | Gordon模型 | 股票内在价值 |

## 备考建议

1. **重点章节**：第二章（权益投资）和第三章（固定收益投资）分值最高
2. **计算题**：本科目涉及大量计算，需熟练掌握各类公式
3. **理解为主**：不能死记硬背，需要理解投资原理和计算逻辑
4. **多做练习**：通过做题巩固知识点，提高解题速度
