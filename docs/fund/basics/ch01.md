本章介绍投资管理的基本概念和理论基础，包括财务报表分析、货币时间价值、投资组合理论等。考试以概念理解为主，需掌握基本定义和计算方法。

## 1.1 财务报表分析

### 1.1.1 资产负债表

**基本等式：** 资产 = 负债 + 所有者权益

**主要项目：**
- **流动资产**：货币资金、应收账款、存货等
- **非流动资产**：固定资产、无形资产、长期投资等
- **流动负债**：短期借款、应付账款、应付职工薪酬等
- **非流动负债**：长期借款、应付债券等
- **所有者权益**：实收资本、资本公积、盈余公积、未分配利润

### 1.1.2 利润表

**基本等式：** 收入 - 费用 = 利润

**主要项目：**
- 营业收入
- 营业成本
- 毛利润 = 营业收入 - 营业成本
- 营业利润 = 毛利润 - 税金及附加 - 期间费用
- 利润总额 = 营业利润 + 营业外收入 - 营业外支出
- 净利润 = 利润总额 - 所得税费用

### 1.1.3 现金流量表

**三类现金流：**
- **经营活动现金流**：与日常经营相关的现金流
- **投资活动现金流**：与长期资产相关的现金流
- **筹资活动现金流**：与资本结构相关的现金流

### 1.1.4 财务比率分析

| 类别 | 指标 | 公式 |
|------|------|------|
| 流动性 | 流动比率 | 流动资产 / 流动负债 |
| 流动性 | 速动比率 | (流动资产 - 存货) / 流动负债 |
| 偿债能力 | 资产负债率 | 负债总额 / 资产总额 |
| 偿债能力 | 利息保障倍数 | EBIT / 利息费用 |
| 盈利能力 | 净资产收益率(ROE) | 净利润 / 股东权益 |
| 盈利能力 | 总资产收益率(ROA) | 净利润 / 总资产 |
| 盈利能力 | 销售净利率 | 净利润 / 营业收入 |
| 营运能力 | 存货周转率 | 营业成本 / 平均存货 |
| 营运能力 | 应收账款周转率 | 营业收入 / 平均应收账款 |

## 1.2 货币时间价值

### 1.2.1 基本概念

- **现值(PV)**：未来某一时点的资金折算到现在的价值
- **终值(FV)**：现在的资金在未来某一时点的价值
- **利率(i)**：资金的时间价值率
- **期数(n)**：计息期数

### 1.2.2 单利与复利

**单利：** FV = PV × (1 + i × n)

**复利：** FV = PV × (1 + i)^n

### 1.2.3 年金

**普通年金（后付年金）：**
- 终值：FV = A × [(1+i)^n - 1] / i
- 现值：PV = A × [1 - (1+i)^(-n)] / i

**即付年金（先付年金）：**
- 终值：FV = A × [(1+i)^n - 1] / i × (1+i)
- 现值：PV = A × [1 - (1+i)^(-n)] / i × (1+i)

**永续年金：** PV = A / i

## 1.3 投资组合理论

### 1.3.1 现代投资组合理论(MPT)

**马科维茨投资组合理论：**
- 投资者是风险厌恶的
- 投资者追求效用最大化
- 用期望收益率和方差描述投资机会
- 投资者通过分散投资降低非系统性风险

### 1.3.2 有效前沿

- 在给定风险水平下，期望收益率最大的投资组合
- 在给定期望收益率下，风险最小的投资组合
- 有效前沿上的投资组合是理性投资者的选择

### 1.3.3 资本市场线(CML)

- 无风险资产与市场组合的组合
- CML的斜率 = (E(Rm) - Rf) / σm
- 所有投资者都持有市场组合与无风险资产的组合

### 1.3.4 资本资产定价模型(CAPM)

**公式：** E(Ri) = Rf + βi × [E(Rm) - Rf]

**其中：**
- E(Ri) = 资产i的期望收益率
- Rf = 无风险利率
- βi = 资产i的系统性风险
- E(Rm) = 市场组合的期望收益率
- E(Rm) - Rf = 市场风险溢价

**β系数：** 衡量资产对市场组合风险的贡献
- β = 1：资产风险与市场风险相同
- β > 1：资产风险大于市场风险
- β < 1：资产风险小于市场风险
