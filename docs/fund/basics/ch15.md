# 第十五章：基金份额持有人

> 分值占比：3%-5% | 重要程度：★★★

## 考情分析

本章介绍基金份额持有人的权利和义务。考试重点在于基金份额持有人大会的召集、议事和表决等。

---

## 15.1 基金份额持有人概述

### 15.1.1 基金份额持有人的定义

基金份额持有人是指依基金合同和招募说明书持有基金份额的自然人和法人。

### 15.1.2 基金份额持有人的地位

- 基金的出资人
- 基金资产的所有者
- 基金投资回报的受益人

---

## 15.2 基金份额持有人的权利

### 15.2.1 基本权利

| 权利 | 说明 |
|------|------|
| 分享基金财产收益 | 按持有份额比例享有收益 |
| 参与分配清算后的剩余基金财产 | 基金终止时的剩余财产分配权 |
| 依法转让或者申请赎回其持有的基金份额 | 份额转让权和赎回权 |
| 按照规定要求召开基金份额持有人大会 | 召集权 |
| 对基金份额持有人大会审议事项行使表决权 | 表决权 |
| 查阅或者复制公开披露的基金信息资料 | 知情权 |
| 对基金管理人、托管人、销售机构损害其合法权益的行为依法提起诉讼 | 诉讼权 |

### 15.2.2 基金份额持有人大会

**召集条件：**
- 基金管理人召集
- 基金托管人召集
- 代表基金份额**10%以上**的基金份额持有人召集

**审议事项：**
- 提前终止基金合同
- 基金扩募或者延长基金合同期限
- 转换基金运作方式
- 提高基金管理人、基金托管人的报酬标准
- 更换基金管理人、基金托管人
- 基金合同约定的其他事项

---

## 15.3 基金份额持有人的义务

### 15.3.1 基本义务

- 遵守基金合同
- 缴纳基金认购、申购款项
- 承担基金亏损
- 不从事任何有损基金及其他基金持有人合法权益的活动

### 15.3.2 基金份额持有人大会的议事规则

**召开条件：**
- 代表基金份额**50%以上**的基金份额持有人参加

**表决规则：**
- 一般事项：经参加大会的基金份额持有人所持表决权的**50%以上**通过
- 重大事项：经参加大会的基金份额持有人所持表决权的**2/3以上**通过

**重大事项包括：**
- 转换基金运作方式
- 更换基金管理人或者基金托管人
- 提前终止基金合同

---

## 15.4 基金份额持有人大会的召集

### 15.4.1 召集人

- **基金管理人**：正常情况下由基金管理人召集
- **基金托管人**：基金管理人未按规定召集时，由基金托管人召集
- **基金份额持有人**：基金管理人和托管人均不召集时，代表基金份额**10%以上**的基金份额持有人可自行召集

### 15.4.2 召集程序

- 提前**30日**公告大会召开时间、会议形式、审议事项等
- 召开方式：现场会议、通讯方式
- 表决方式：记名投票

---

## 15.5 基金份额持有人的保护

### 15.5.1 保护措施

- 基金财产独立于基金管理人和托管人的固有财产
- 基金管理人和托管人不得将基金财产归入其固有财产
- 基金份额持有人大会行使监督权
- 中国证监会对基金管理人和托管人进行监管

### 15.5.2 救济途径

- 向基金管理人、托管人提出异议
- 向中国证监会投诉
- 向人民法院提起诉讼
