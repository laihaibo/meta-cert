# 第三章：固定收益投资

> 分值占比：8%-12% | 重要程度：★★★★★

## 考情分析

本章是本科目的重点章节，主要介绍债券投资的基本理论和分析方法。考试重点在于债券估值、久期、凸性、收益率曲线等概念和计算。

---

## 3.1 债券的特征与类型

### 3.1.1 债券的特征

| 特征 | 说明 |
|------|------|
| 偿还性 | 有固定到期日，到期偿还本金 |
| 流动性 | 可在二级市场买卖 |
| 安全性 | 收益相对稳定，风险较低 |
| 收益性 | 利息收入和资本利得 |

### 3.1.2 债券的类型

**按发行主体分类：**
- **政府债券**：国债、地方政府债券
- **金融债券**：政策性金融债、商业银行债券
- **公司债券**：企业债券、公司债券

**按期限分类：**
- 短期债券：1年以内
- 中期债券：1-10年
- 长期债券：10年以上

**按利息支付方式分类：**
- 附息债券
- 贴现债券
- 零息债券

## 3.2 债券估值

### 3.2.1 债券定价公式

**附息债券：** P = Σ [C / (1+r)^t] + [M / (1+r)^n]

**其中：**
- P = 债券价格
- C = 每期利息
- M = 面值
- r = 折现率（到期收益率）
- n = 期数

### 3.2.2 债券收益率

**当期收益率：** 当期收益率 = 年利息 / 债券价格

**到期收益率(YTM)：** 使债券未来现金流现值等于当前价格的折现率

**持有期收益率：** 持有期收益率 = (卖出价格 - 买入价格 + 持有期间利息) / 买入价格

## 3.3 债券风险

### 3.3.1 利率风险

- 利率上升，债券价格下降
- 利率下降，债券价格上升
- 期限越长，利率风险越大

### 3.3.2 信用风险

- 发行人无法按时支付利息或偿还本金
- 信用评级越低，信用风险越大

### 3.3.3 流动性风险

- 债券无法以合理价格变现的风险

### 3.3.4 通胀风险

- 通货膨胀导致实际收益率下降

---

## 3.4 久期与凸性

### 3.4.1 久期

**麦考利久期：** D = Σ [t × PV(Ct) / P]

**修正久期：** Dmod = D / (1+y)

**其中：**
- D = 麦考利久期
- Dmod = 修正久期
- y = 到期收益率

**久期的含义：**
- 衡量债券价格对利率变动的敏感性
- 久期越长，利率风险越大

### 3.4.2 凸性

- 衡量久期对利率变动的敏感性
- 正凸性：利率下降时，债券价格上涨幅度大于久期预测
- 负凸性：利率上升时，债券价格下跌幅度大于久期预测

### 3.4.3 久期的应用

**免疫策略：** 通过调整债券组合的久期，使其等于投资期限，以规避利率风险

## 3.5 收益率曲线

### 3.5.1 收益率曲线的类型

| 类型 | 形态 | 市场预期 |
|------|------|---------|
| 正向收益率曲线 | 长期利率>短期利率 | 利率上升 |
| 反向收益率曲线 | 长期利率<短期利率 | 利率下降 |
| 水平收益率曲线 | 长期利率=短期利率 | 利率不变 |

### 3.5.2 收益率曲线的变动

- **平行移动**：所有期限利率变动相同
- **非平行移动**：不同期限利率变动不同
