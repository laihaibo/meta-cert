# 第十六章：基金监管

> 分值占比：3%-5% | 重要程度：★★★

## 考情分析

本章介绍基金监管的基本框架和监管措施。考试重点在于监管机构的职责、监管措施的类型等。

---

## 16.1 基金监管概述

### 16.1.1 基金监管的含义

基金监管是指监管部门运用法律、经济及必要的行政手段，对基金市场参与者的行为进行的监督与管理。

### 16.1.2 基金监管的目标

- **保护投资者利益**：首要目标
- **保证市场的公平、效率和透明**
- **降低系统风险**
- **促进基金行业健康发展**

### 16.1.3 基金监管的原则

| 原则 | 说明 |
|------|------|
| 依法监管原则 | 监管活动必须依法进行 |
| 公开、公平、公正原则 | 保护投资者合法权益 |
| 适度监管原则 | 不干预市场主体的正常经营 |
| 高效监管原则 | 提高监管效率 |

---

## 16.2 监管机构

### 16.2.1 中国证监会

**性质：** 国务院直属事业单位，基金行业的主管部门

**主要职责：**
- 制定基金监管的规章、规则
- 审批基金募集申请
- 核准基金管理人、基金托管人的资格
- 监督基金管理人、基金托管人的经营活动
- 查处违法行为
- 法律、行政法规规定的其他职责

### 16.2.2 中国证券投资基金业协会

**性质：** 基金行业的自律性组织，社会团体法人

**主要职责：**
- 教育和组织会员遵守法律法规
- 依法维护会员合法权益
- 制定和实施行业自律规则
- 制定和实施行业执业标准和业务规范
- 组织基金从业人员资格考试
- 组织基金从业人员的后续培训
- 对会员之间、会员与客户之间发生的基金业务纠纷进行调解
- 对违反法律法规或自律规则的会员进行纪律处分

---

## 16.3 监管措施

### 16.3.1 行政监管措施

| 措施 | 适用情形 |
|------|---------|
| 责令改正 | 一般违规行为 |
| 出示警示函 | 轻微违规行为 |
| 限制业务活动 | 较严重违规 |
| 责令更换董监高 | 严重违规 |
| 限制股东权利 | 股东违规 |
| 责令停业整顿 | 严重违规 |
| 吊销业务许可证 | 最严重违规 |

### 16.3.2 行政处罚

- **警告**
- **罚款**
- **没收违法所得**
- **暂停或者撤销基金从业资格**
- **暂停或者撤销相关业务许可**

---

## 16.4 基金监管的内容

### 16.4.1 对基金管理人的监管

- 市场准入监管
- 投资运作监管
- 信息披露监管
- 内部控制监管

### 16.4.2 对基金托管人的监管

- 市场准入监管
- 资产保管监管
- 投资监督监管

### 16.4.3 对基金销售机构的监管

- 销售资格监管
- 销售行为监管
- 销售费用监管

---

## 16.5 法律责任

### 16.5.1 民事责任

- 赔偿损失
- 返还财产
- 支付违约金

### 16.5.2 行政责任

- 行政处罚
- 行政处分

### 16.5.3 刑事责任

- 非法经营罪
- 内幕交易罪
- 操纵市场罪
- 挪用资金罪
