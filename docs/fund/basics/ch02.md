## 考情分析

本章是本科目的重点章节，主要介绍股票投资的基本理论和分析方法。考试重点在于股票估值方法、股票分析方法、行业分析和公司分析等。

## 2.1 股票的特征与类型

### 2.1.1 股票的特征

| 特征 | 说明 |
|------|------|
| 收益性 | 股息收入和资本利得 |
| 风险性 | 收益不确定，可能亏损 |
| 流动性 | 可在二级市场买卖 |
| 永久性 | 无到期日，不可偿还 |
| 参与性 | 参与公司重大决策 |

### 2.1.2 股票的类型

**按股东权利分类：**
- **普通股**：享有公司经营参与权、收益分配权、剩余资产分配权
- **优先股**：优先分配股息、优先分配剩余资产、无表决权

**按上市地域分类：**
- **A股**：人民币普通股票
- **B股**：人民币特种股票（外币认购和买卖）
- **H股**：注册地在内地、上市地在香港的外资股
- **N股**：注册地在内地、上市地在纽约的外资股

## 2.2 股票估值方法

### 2.2.1 内在价值法

**1. 股利贴现模型(DDM)**

**Gordon增长模型：** P = D1 / (k - g)

**其中：**
- P = 股票内在价值
- D1 = 下一期股利
- k = 必要收益率
- g = 股利增长率（g < k）

**两阶段DDM：** 适用于高增长后稳定增长的公司

**2. 自由现金流贴现模型(FCF)**

**企业自由现金流(FCFF)：** FCFF = EBIT × (1-t) + 折旧 - 资本支出 - 营运资本增加

**股权自由现金流(FCFE)：** FCFE = FCFF - 利息费用 × (1-t) + 净借款

### 2.2.2 相对价值法

| 指标 | 公式 | 适用场景 |
|------|------|---------|
| 市盈率(P/E) | 价格 / 每股收益 | 盈利稳定的公司 |
| 市净率(P/B) | 价格 / 每股净资产 | 资产密集型公司 |
| 市销率(P/S) | 价格 / 每股销售收入 | 亏损或微利公司 |
| 企业价值倍数(EV/EBITDA) | 企业价值 / EBITDA | 资本结构差异大的公司 |

## 2.3 股票分析方法

### 2.3.1 基本面分析

**宏观经济分析：**
- GDP增长率
- 通货膨胀率
- 利率水平
- 汇率变动
- 货币政策
- 财政政策

**行业分析：**
- 行业生命周期：初创期、成长期、成熟期、衰退期
- 行业竞争结构：波特五力模型
- 行业景气度

**公司分析：**
- 公司基本面
- 财务状况
- 竞争优势
- 管理层能力

### 2.3.2 技术分析

**三大假设：**
- 市场行为包含一切信息
- 价格沿趋势运动
- 历史会重演

**主要工具：**
- K线图
- 移动平均线
- 成交量
- 技术指标（RSI、MACD等）

## 2.4 股票投资策略

### 2.4.1 价值型投资

- 关注低估值股票
- 低市盈率、低市净率
- 注重安全边际
- 代表人物：巴菲特

### 2.4.2 成长型投资

- 关注高成长公司
- 高市盈率、高市净率
- 注重未来增长潜力
- 代表人物：彼得·林奇

### 2.4.3 指数型投资

- 被动跟踪指数
- 低管理费用
- 分散风险
- 代表产品：ETF
