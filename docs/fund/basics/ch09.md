# 第九章：基金业绩评价

> 分值占比：5%-8% | 重要程度：★★★★

## 考情分析

本章介绍基金业绩评价的方法和指标。考试重点在于各类业绩评价指标的计算和应用、业绩归因分析、基准选择等。

---

## 9.1 基金业绩评价概述

### 9.1.1 业绩评价的目的

- 评估基金管理人的投资能力
- 为投资者提供决策依据
- 促进基金行业的优胜劣汰

### 9.1.2 业绩评价的原则

| 原则 | 说明 |
|------|------|
| 公平性 | 同类基金进行比较 |
| 全面性 | 考虑收益和风险 |
| 可比性 | 使用相同的基准和方法 |
| 及时性 | 定期进行评价 |

---

## 9.2 业绩评价指标

### 9.2.1 收益率指标

**时间加权收益率：**
- 消除了资金流入流出的影响
- 衡量基金管理人的投资能力
- R = [(1+R1) × (1+R2) × ... × (1+Rn)] - 1

**金额加权收益率：**
- 考虑了资金流入流出的影响
- 衡量投资者的实际收益
- 使现金流现值等于投资现值的折现率

**年化收益率：**
- 将不同期限的收益率转换为年度收益率
- 便于比较不同期限的投资

### 9.2.2 风险调整后收益指标

| 指标 | 公式 | 含义 |
|------|------|------|
| 夏普比率 | (Rp-Rf)/σp | 单位总风险的超额收益 |
| 特雷诺比率 | (Rp-Rf)/βp | 单位系统风险的超额收益 |
| 詹森α | Rp-[Rf+β(Rm-Rf)] | 超额收益 |
| 信息比率 | α/σ(残差) | 单位主动风险的超额收益 |
| M²测度 | Rp-Rm* | 风险调整后的超额收益 |

### 9.2.3 相对收益指标

**信息比率(IR)：**
- 衡量主动管理的能力
- IR = α / σ(残差)
- IR越大，主动管理能力越强

**M²测度：**
- 由Modigliani和Modigliani提出
- 将投资组合的风险调整到与市场组合相同
- 比较调整后的收益与市场收益

## 9.3 业绩归因分析

### 9.3.1 资产配置归因

- 分析资产配置对业绩的贡献
- 比较实际配置比例与基准配置比例
- 计算资产配置的超额收益

### 9.3.2 证券选择归因

- 分析证券选择对业绩的贡献
- 比较实际证券收益与基准证券收益
- 计算证券选择的超额收益

### 9.3.3 交互作用归因

- 分析资产配置和证券选择的交互作用
- 交互作用 = 总超额收益 - 资产配置贡献 - 证券选择贡献

---

## 9.4 基准选择

### 9.4.1 基准的类型

| 类型 | 说明 |
|------|------|
| 市场指数 | 如沪深300、中证500 |
| 同类基金平均 | 同类型基金的平均收益 |
| 绝对收益 | 如存款利率、通胀率 |

### 9.4.2 基准选择的原则

- **相关性**：基准应与基金的投资策略相关
- **可投资性**：基准应该是可投资的
- **透明性**：基准的编制方法应该透明
- **可衡量性**：基准的收益应该可以衡量

---

## 9.5 业绩评价的注意事项

### 9.5.1 样本选择偏差

- 存活者偏差：只考虑存续的基金
- 样本期选择偏差：选择特定时期进行评价

### 9.5.2 风险调整的局限性

- 不同风险调整指标可能得出不同结论
- 风险度量方法可能不准确
- 历史业绩不代表未来表现

### 9.5.3 费用的影响

- 业绩评价应考虑费用的影响
- 不同基金的费用结构可能不同
- 费用对长期业绩影响较大
