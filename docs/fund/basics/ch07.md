# 第七章：投资交易管理

> 分值占比：5%-8% | 重要程度：★★★

## 考情分析

本章介绍投资交易管理的基本流程和规范。考试重点在于交易执行、交易成本、交易限制等。

---

## 7.1 交易执行

### 7.1.1 交易执行的方式

**指令驱动制度（竞价交易）：**
- 价格优先、时间优先
- 买卖双方直接竞价
- 适用：证券交易所

**报价驱动制度（做市商制度）：**
- 做市商提供买卖报价
- 投资者与做市商交易
- 适用：场外市场

### 7.1.2 交易指令的类型

| 指令类型 | 说明 |
|---------|------|
| 市价指令 | 以当前市场价格立即成交 |
| 限价指令 | 限定价格，达到价格才成交 |
| 止损指令 | 达到止损价格时触发市价指令 |
| 止损限价指令 | 达到止损价格时触发限价指令 |

### 7.1.3 交易执行的原则

- **最佳执行原则**：以最优价格执行交易
- **公平交易原则**：公平对待不同基金和客户
- **合规交易原则**：遵守法律法规和内部制度

---

## 7.2 交易成本

### 7.2.1 显性成本

| 成本类型 | 说明 |
|---------|------|
| 佣金 | 支付给券商的费用 |
| 印花税 | 卖出时支付的税费 |
| 过户费 | 证券过户的费用 |
| 交易手续费 | 交易所收取的费用 |

### 7.2.2 隐性成本

| 成本类型 | 说明 |
|---------|------|
| 买卖价差 | 买入价与卖出价的差额 |
| 市场冲击成本 | 大额交易对市场价格的影响 |
| 时机成本 | 延迟交易导致的价格变化 |
| 机会成本 | 未执行交易的潜在收益损失 |

### 7.2.3 执行差额

- 实际成交价格与决策价格的差异
- 衡量交易执行质量的指标

---

## 7.3 交易限制

### 7.3.1 投资限制

**比例限制：**
- 单只基金持有一家公司发行的证券，不得超过基金资产净值的**10%**
- 同一基金管理人管理的全部基金持有一家公司发行的证券，不得超过该证券的**10%**

**投资范围限制：**
- 基金合同约定的投资范围
- 法律法规规定的投资限制

### 7.3.2 交易限制

- 不得进行内幕交易
- 不得操纵市场价格
- 不得进行利益输送
- 不得挪用基金财产

---

## 7.4 交易记录与报告

### 7.4.1 交易记录

- 记录每笔交易的详细信息
- 包括：交易时间、交易价格、交易数量等
- 保存期限不少于**20年**

### 7.4.2 交易报告

- 定期编制交易报告
- 报告内容包括：交易汇总、交易分析等
- 提交给投资决策委员会
