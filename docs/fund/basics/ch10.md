# 第十章：基金估值、费用与会计

> 分值占比：5%-8% | 重要程度：★★★

## 考情分析

本章介绍基金估值、费用和会计的基本概念和方法。考试重点在于基金资产估值的方法、各类费用的计算、基金会计核算等。

---

## 10.1 基金资产估值

### 10.1.1 估值的目的

- 确定基金份额净值
- 为申购赎回提供价格依据
- 反映基金的真实价值

### 10.1.2 估值的基本原则

| 原则 | 说明 |
|------|------|
| 公允性 | 估值应反映资产的公允价值 |
| 及时性 | 估值应及时更新 |
| 准确性 | 估值应准确反映资产价值 |
| 一致性 | 估值方法应保持一致 |

### 10.1.3 估值方法

**股票估值：**
- 上市流通股票：按交易所收盘价估值
- 首次发行未上市股票：按成本估值
- 配股和增发新股：按交易所收盘价估值

**债券估值：**
- 上市债券：按交易所收盘价估值
- 银行间债券：按第三方估值机构估值
- 未上市债券：按成本估值

**其他资产估值：**
- 银行存款：按本金加应计利息估值
- 权证：按交易所收盘价估值
- 资产支持证券：按第三方估值机构估值

### 10.1.4 估值错误的处理

- 发现估值错误应及时纠正
- 基金管理人应承担估值错误造成的损失
- 基金托管人应复核估值结果

## 10.2 基金费用

### 10.2.1 基金管理费

**计算方法：** 管理费 = 基金资产净值 × 管理费率 × 计提天数 / 365

**费率标准：**
- 股票基金：通常1.5%
- 债券基金：通常0.6%-0.8%
- 货币市场基金：通常0.15%-0.33%

### 10.2.2 基金托管费

**计算方法：** 托管费 = 基金资产净值 × 托管费率 × 计提天数 / 365

**费率标准：**
- 股票基金：通常0.25%
- 债券基金：通常0.1%-0.2%
- 货币市场基金：通常0.05%-0.1%

### 10.2.3 基金销售服务费

- 从基金财产中扣除
- 不另向投资者收取
- C类基金份额通常收取销售服务费

### 10.2.4 基金交易费用

| 费用类型 | 说明 |
|---------|------|
| 交易佣金 | 支付给券商的费用 |
| 印花税 | 卖出股票时支付的税费 |
| 过户费 | 证券过户的费用 |
| 经手费 | 交易所收取的费用 |

## 10.3 基金会计

### 10.3.1 基金会计的特点

- 独立于基金管理人的会计
- 以基金为会计主体
- 采用公允价值计量
- 按日核算

### 10.3.2 基金会计核算的内容

- 资产的核算
- 负债的核算
- 收入的核算
- 费用的核算
- 基金份额净值的计算

### 10.3.3 基金财务报表

| 报表 | 内容 |
|------|------|
| 资产负债表 | 基金的资产、负债和净资产 |
| 利润表 | 基金的收入和费用 |
| 净资产变动表 | 基金净资产的变动情况 |
| 附注 | 对报表内容的说明 |

---

## 10.4 基金利润

### 10.4.1 基金利润的构成

- 利息收入
- 投资收益
- 公允价值变动收益
- 其他收入
- 减：费用

### 10.4.2 基金利润的计算

**基金利润 = 收入 - 费用**

**收入包括：**
- 利息收入
- 投资收益
- 公允价值变动收益
- 其他收入

**费用包括：**
- 管理费
- 托管费
- 销售服务费
- 交易费用
- 其他费用
