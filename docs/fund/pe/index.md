## 高频重点专题

### 1. 基金组织形式对比（合伙型 vs 公司型 vs 契约型）

| 对比项 | 合伙型 | 公司型 | 契约型 |
|-------|--------|--------|--------|
| 法律依据 | 《合伙企业法》 | 《公司法》 | 《基金法》 |
| 法律地位 | 非法人 | 独立法人 | 非法人 |
| 投资者身份 | 合伙人 | 股东 | 基金份额持有人 |
| 决策机制 | 合伙人会议 | 股东大会/董事会 | 基金份额持有人大会 |
| 税收 | 穿透征税 | 双重征税 | 穿透征税 |
| 有限责任 | LP承担有限责任 | 股东承担有限责任 | 投资者承担有限责任 |

### 2. 合格投资者标准

- 机构投资者：净资产不低于**1000万元**
- 个人投资者：金融资产不低于**300万元**或最近3年个人年均收入不低于**50万元**

### 3. 估值方法

- 市场法
- 收益法
- 成本法
- PE倍数法

### 4. 投资条款

- 对赌协议
- 反稀释条款
- 优先权条款

### 5. 退出方式及收益分配

- IPO退出
- 股权转让退出
- 回购退出
- 清算退出
- 瀑布式分配

## 备考建议

1. **重点章节**：第三章（募集与设立）、第四章（投资）、第六章（退出）分值最高
2. **对比记忆**：基金组织形式对比是常考内容
3. **数字考点**：合格投资者标准、基金募集人数限制等
4. **理解为主**：私募股权投资的流程和实务操作需要理解
