# 第八章：行业自律与监管

> 分值占比：10%-15% | 重要程度：★★★★

## 考情分析

本章介绍股权投资基金的行业自律和监管。考试重点在于监管机构的职责、自律管理的内容、监管措施等。

---

## 8.1 监管概述

### 8.1.1 监管的意义

- 保护投资者利益
- 维护市场秩序
- 防范系统性风险
- 促进行业健康发展

### 8.1.2 监管的原则

| 原则 | 说明 |
|------|------|
| 依法监管 | 监管活动必须依法进行 |
| 适度监管 | 不干预市场主体正常经营 |
| 分类监管 | 根据不同类型采取不同措施 |
| 协同监管 | 多部门协同监管 |

---

## 8.2 监管机构

### 8.2.1 中国证监会

**性质：** 国务院直属事业单位，基金行业的主管部门

**主要职责：**
- 制定监管规则
- 监督基金管理人
- 查处违法行为
- 保护投资者利益

### 8.2.2 中国证券投资基金业协会

**性质：** 基金行业的自律性组织

**主要职责：**
- 制定自律规则
- 组织从业资格考试
- 开展行业培训
- 进行自律检查
- 处理投诉举报

---

## 8.3 基金管理人登记

### 8.3.1 登记条件

- 符合法定条件
- 具备专业团队
- 具备经营场所
- 具备管理制度

### 8.3.2 登记材料

- 基金管理人登记申请表
- 公司章程或合伙协议
- 主要股东或合伙人基本情况
- 高级管理人员基本情况
- 法律意见书

### 8.3.3 登记流程

```
提交材料 → 协会审核 → 公示 → 完成登记
```

---

## 8.4 基金备案

### 8.4.1 备案要求

- 基金管理人应当在基金募集完毕后**20个工作日内**办理备案
- 备案材料包括基金合同、招募说明书等

### 8.4.2 备案流程

```
提交备案材料 → 协会审核 → 完成备案
```

---

## 8.5 信息披露

### 8.5.1 信息披露的要求

- 真实、准确、完整
- 及时、公平
- 不得有虚假记载、误导性陈述或重大遗漏

### 8.5.2 信息披露的内容

**定期报告：**
- 基金季度报告
- 基金年度报告

**临时报告：**
- 重大事项报告
- 投资项目重大变化报告

### 8.5.3 信息披露的时限

| 报告类型 | 披露时限 |
|---------|---------|
| 季度报告 | 每季度结束后**15个工作日内** |
| 年度报告 | 每年结束后**4个月内** |
| 重大事项 | 发生之日起**10个工作日内** |

---

## 8.6 自律管理

### 8.6.1 自律规则

- 行业自律规则
- 执业标准
- 业务规范
- 职业道德规范

### 8.6.2 自律检查

- 定期检查
- 专项检查
- 随机抽查

### 8.6.3 自律处分

| 处分类型 | 适用情形 |
|---------|---------|
| 谈话提醒 | 轻微违规 |
| 书面警示 | 一般违规 |
| 公开谴责 | 较严重违规 |
| 暂停备案 | 严重违规 |
| 取消登记 | 最严重违规 |

---

## 8.7 行政监管

### 8.7.1 行政监管措施

| 措施 | 适用情形 |
|------|---------|
| 责令改正 | 一般违规 |
| 出示警示函 | 轻微违规 |
| 限制业务活动 | 较严重违规 |
| 责令停业整顿 | 严重违规 |
| 吊销业务许可证 | 最严重违规 |

### 8.7.2 行政处罚

- 警告
- 罚款
- 没收违法所得
- 市场禁入

---

## 8.8 法律责任

### 8.8.1 民事责任

- 赔偿损失
- 返还财产
- 支付违约金

### 8.8.2 行政责任

- 行政处罚
- 行政处分

### 8.8.3 刑事责任

- 非法集资罪
- 非法经营罪
- 内幕交易罪
- 操纵市场罪
