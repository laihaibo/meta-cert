# 第三章：募集与设立

> 分值占比：15%-20% | 重要程度：★★★★★

## 考情分析

本章是本科目的重点章节，主要介绍股权投资基金的募集和设立流程。考试重点在于合格投资者标准、基金组织形式的选择、基金合同的主要条款等。

---

## 3.1 基金募集

### 3.1.1 募集方式

**自行募集：**
- 基金管理人自行募集资金
- 需要取得基金销售资格

**委托募集：**
- 委托具有基金销售资格的机构募集资金
- 需要签订委托募集协议

### 3.1.2 募集对象

**合格投资者标准：**

| 投资者类型 | 标准 |
|-----------|------|
| 机构投资者 | 净资产不低于**1000万元** |
| 个人投资者 | 金融资产不低于**300万元**或最近3年个人年均收入不低于**50万元** |

**视为合格投资者：**
- 社会保障基金、企业年金等养老基金
- 慈善基金等社会公益基金
- 依法设立并在基金业协会备案的投资计划
- 投资于所管理私募基金的私募基金管理人及其从业人员

### 3.1.3 募集人数限制

- **合伙型基金**：合伙人不超过**50人**
- **公司型基金**：股东不超过**200人**
- **契约型基金**：投资者不超过**200人**

### 3.1.4 募集程序

```
特定对象确认 → 投资者适当性匹配 → 基金推介 → 签署风险揭示书 → 合格投资者确认 → 签署基金合同 → 投资冷静期 → 回访确认
```

## 3.2 基金组织形式

### 3.2.1 合伙型基金

**法律依据：** 《合伙企业法》

**特点：**
- 非法人组织
- 穿透征税
- 决策灵活
- LP承担有限责任，GP承担无限责任

**合伙协议主要内容：**
- 合伙企业的名称和主要经营场所的地点
- 合伙目的和合伙经营范围
- 合伙人的姓名或者名称、住所
- 合伙人的出资方式、数额和缴付期限
- 利润分配、亏损分担方式
- 合伙事务的执行
- 入伙与退伙
- 争议解决办法
- 合伙企业的解散与清算
- 违约责任

### 3.2.2 公司型基金

**法律依据：** 《公司法》

**特点：**
- 独立法人
- 双重征税
- 治理结构完善
- 股东承担有限责任

**公司章程主要内容：**
- 公司名称和住所
- 公司经营范围
- 公司注册资本
- 股东的姓名或者名称
- 股东的出资方式、出资额和出资时间
- 公司的机构及其产生办法、职权、议事规则
- 公司法定代表人
- 股东会会议认为需要规定的其他事项

### 3.2.3 契约型基金

**法律依据：** 《基金法》

**特点：**
- 非法人组织
- 穿透征税
- 结构简单
- 投资者承担有限责任

**基金合同主要内容：**
- 基金的基本情况
- 基金的募集和设立
- 基金的投资
- 基金的收益分配
- 基金的费用和税收
- 基金的信息披露
- 基金的终止和清算

### 3.2.4 组织形式对比

| 对比项 | 合伙型 | 公司型 | 契约型 |
|-------|--------|--------|--------|
| 法律依据 | 《合伙企业法》 | 《公司法》 | 《基金法》 |
| 法律地位 | 非法人 | 独立法人 | 非法人 |
| 投资者身份 | 合伙人 | 股东 | 基金份额持有人 |
| 决策机制 | 合伙人会议 | 股东大会/董事会 | 基金份额持有人大会 |
| 税收 | 穿透征税 | 双重征税 | 穿透征税 |
| 有限责任 | LP承担有限责任 | 股东承担有限责任 | 投资者承担有限责任 |

## 3.3 基金备案

### 3.3.1 备案要求

- 基金管理人应当在基金募集完毕后**20个工作日内**，向中国证券投资基金业协会办理基金备案手续

### 3.3.2 备案材料

- 基金合同
- 基金公司章程或者合伙协议
- 基金招募说明书
- 基金托管协议（如有）
- 基金管理人、基金托管人的基本情况
- 中国证券投资基金业协会规定的其他材料

---

## 3.4 基金投资冷静期和回访确认

### 3.4.1 投资冷静期

- 基金合同应当约定给予投资者**24小时**的投资冷静期
- 投资者在冷静期内可以解除基金合同

### 3.4.2 回访确认

- 基金管理人应当在投资冷静期满后，通过录音电话、电邮、信函等方式进行回访确认
- 回访确认的内容包括：确认投资者身份、确认投资者知悉投资冷静期、确认投资者知悉基金风险等
