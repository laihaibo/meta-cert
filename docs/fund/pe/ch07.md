# 第七章：管理与内控

> 分值占比：8%-12% | 重要程度：★★★

## 考情分析

本章介绍股权投资基金的管理和内部控制。考试重点在于基金管理人的内部控制、风险管理等。

---

## 7.1 基金管理人的内部控制

### 7.1.1 内部控制的目标

- 保证基金管理人合法合规经营
- 防范和化解经营风险
- 保护投资者利益
- 提高经营管理效率

### 7.1.2 内部控制的原则

| 原则 | 说明 |
|------|------|
| 健全性原则 | 覆盖各项业务和各级人员 |
| 有效性原则 | 科学、合理、有效 |
| 独立性原则 | 内部控制部门独立 |
| 相互制约原则 | 相互制约、相互监督 |
| 成本效益原则 | 权衡成本与效益 |

### 7.1.3 内部控制的内容

**投资管理控制：**
- 投资决策流程
- 投资授权制度
- 投资风险控制

**运营管理控制：**
- 资金管理
- 财务管理
- 档案管理

**信息披露控制：**
- 信息披露制度
- 信息披露流程
- 信息披露责任

---

## 7.2 风险管理

### 7.2.1 风险类型

| 风险类型 | 说明 |
|---------|------|
| 投资风险 | 投资项目失败的风险 |
| 市场风险 | 市场环境变化的风险 |
| 流动性风险 | 无法及时退出的风险 |
| 操作风险 | 内部流程不完善的风险 |
| 合规风险 | 违反法律法规的风险 |

### 7.2.2 风险管理措施

**风险识别：**
- 建立风险识别机制
- 定期进行风险评估

**风险控制：**
- 设定风险限额
- 建立风险预警机制
- 制定应急预案

**风险转移：**
- 通过保险转移风险
- 通过担保转移风险

---

## 7.3 合规管理

### 7.3.1 合规管理的目标

- 确保基金管理人合法合规经营
- 防范合规风险
- 保护投资者利益

### 7.3.2 合规管理的内容

- 法律法规合规
- 行业规范合规
- 内部制度合规
- 职业道德合规

### 7.3.3 合规管理的措施

- 建立合规管理制度
- 设立合规管理部门
- 开展合规培训
- 进行合规检查

---

## 7.4 人力资源管理

### 7.4.1 人员要求

- 取得基金从业资格
- 具备专业知识和技能
- 遵守职业道德

### 7.4.2 激励约束机制

**激励机制：**
- 薪酬激励
- 股权激励
- 业绩激励

**约束机制：**
- 绩效考核
- 合规约束
- 竞业限制

---

## 7.5 信息技术管理

### 7.5.1 信息系统建设

- 投资管理系统
- 风险管理系统
- 运营管理系统
- 信息披露系统

### 7.5.2 信息安全

- 数据备份
- 访问控制
- 安全审计

---

## 7.6 外包服务管理

### 7.6.1 外包服务的类型

- 基金行政管理
- 基金估值核算
- 基金份额登记
- 信息技术服务

### 7.6.2 外包服务管理

- 选择合格的服务机构
- 签订外包服务协议
- 监督服务质量
- 保护投资者信息
