# 第一章：股权投资基金概述

> 分值占比：5%-8% | 重要程度：★★★

## 考情分析

本章介绍股权投资基金的基本概念和特点。考试重点在于股权投资基金的定义、特点、分类等。

---

## 1.1 股权投资基金的定义

### 1.1.1 基本概念

股权投资基金（Private Equity Fund，简称PE基金）是指主要投资于非公开发行的企业股权的投资基金。

### 1.1.2 核心特征

| 特征 | 说明 |
|------|------|
| 投资对象 | 非公开发行的企业股权 |
| 投资期限 | 较长（通常3-7年） |
| 流动性 | 较差 |
| 收益来源 | 资本增值和分红 |
| 风险水平 | 较高 |

---

## 1.2 股权投资基金的分类

### 1.2.1 按投资阶段分类

| 类型 | 投资阶段 | 特点 |
|------|---------|------|
| 风险投资(VC) | 初创期 | 高风险、高收益 |
| 成长期投资 | 成长期 | 中等风险、中等收益 |
| 并购基金 | 成熟期 | 较低风险、稳定收益 |
| 夹层资本 | 各阶段 | 介于股权和债权之间 |

### 1.2.2 按组织形式分类

| 类型 | 法律依据 | 特点 |
|------|---------|------|
| 合伙型 | 《合伙企业法》 | 穿透征税、灵活 |
| 公司型 | 《公司法》 | 独立法人、双重征税 |
| 契约型 | 《基金法》 | 非法人、结构简单 |

### 1.2.3 按资金来源分类

- **人民币基金**：以人民币募集
- **外币基金**：以外币募集

---

## 1.3 股权投资基金的运作流程

```
募集 → 设立 → 投资 → 投后管理 → 退出
```

### 1.3.1 募集阶段

- 向合格投资者募集资金
- 签署基金合同
- 完成基金备案

### 1.3.2 投资阶段

- 项目筛选
- 尽职调查
- 投资决策
- 签署投资协议

### 1.3.3 投后管理阶段

- 参与公司治理
- 提供增值服务
- 监控经营状况

### 1.3.4 退出阶段

- IPO退出
- 股权转让
- 回购
- 清算

---

## 1.4 股权投资基金的市场参与者

### 1.4.1 基金投资者

- 机构投资者：社保基金、保险资金、企业年金等
- 高净值个人：合格投资者

### 1.4.2 基金管理人

- 负责基金的投资管理
- 收取管理费和业绩报酬

### 1.4.3 基金托管人

- 保管基金资产
- 监督基金管理人

### 1.4.4 中介服务机构

- 律师事务所
- 会计师事务所
- 评估机构
