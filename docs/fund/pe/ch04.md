## 4.2 尽职调查

### 4.2.1 尽职调查的定义

尽职调查是指投资人在与目标企业达成初步合作意向后，经协商一致，投资人对目标企业一切与本次投资有关的事项进行现场调查、资料分析的一系列活动。

### 4.2.2 尽职调查的类型

| 类型 | 内容 |
|------|------|
| 业务尽职调查 | 企业业务模式、市场地位、竞争优势等 |
| 财务尽职调查 | 企业财务状况、盈利能力、现金流等 |
| 法律尽职调查 | 企业法律合规性、诉讼风险等 |

### 4.2.3 尽职调查的内容

**业务尽职调查：**
- 企业基本情况
- 行业分析
- 产品和服务
- 业务模式
- 市场地位
- 竞争优势
- 发展战略

**财务尽职调查：**
- 财务报表分析
- 盈利能力分析
- 偿债能力分析
- 营运能力分析
- 现金流分析
- 税务情况

**法律尽职调查：**
- 公司设立和变更
- 股权结构
- 重大合同
- 知识产权
- 诉讼仲裁
- 合规情况

### 4.2.4 尽职调查的方法

- 查阅文件资料
- 现场走访
- 人员访谈
- 第三方调查
- 专家咨询

## 4.3 投资估值

### 4.3.1 估值方法概述

| 方法 | 原理 | 适用场景 |
|------|------|---------|
| 市场法 | 参照可比公司估值 | 有可比公司 |
| 收益法 | 未来现金流折现 | 盈利稳定 |
| 成本法 | 资产重置成本 | 资产密集型 |
| PE倍数法 | 市盈率倍数 | 盈利企业 |

### 4.3.2 市场法

**可比公司法：**
- 选择可比上市公司
- 计算估值倍数（P/E、P/B、EV/EBITDA等）
- 应用到目标公司

**可比交易法：**
- 选择可比交易案例
- 计算交易估值倍数
- 应用到目标公司

### 4.3.3 收益法

**现金流折现法(DCF)：**
- 预测未来自由现金流
- 确定折现率
- 计算现值

**公式：** V = Σ [FCFt / (1+r)^t] + [TV / (1+r)^n]

### 4.3.4 成本法

- 评估企业各项资产的重置成本
- 扣除负债
- 得出企业价值

### 4.3.5 PE倍数法

**公式：** 企业价值 = 净利润 × PE倍数

**PE倍数的确定：**
- 参照同行业上市公司市盈率
- 考虑流动性折价
- 考虑控制权溢价

## 4.4 投资条款清单

### 4.4.1 投资条款清单的定义

投资条款清单（Term Sheet）是投资方与融资方就投资事项达成的初步意向书，是正式投资协议的基础。

### 4.4.2 主要条款

**估值条款：**
- 投前估值
- 投后估值
- 投资金额
- 持股比例

**对赌条款（估值调整条款）：**
- **业绩对赌**：根据企业未来业绩调整估值
- **上市对赌**：根据企业是否上市调整估值
- **回购对赌**：未达到约定条件时，企业或股东回购股权

**反稀释条款：**
- **完全棘轮法**：以最低价格调整
- **加权平均法**：以加权平均价格调整

**优先权条款：**
- **优先认购权**：新股发行时优先认购
- **优先购买权**：股权转让时优先购买
- **优先清算权**：清算时优先获得分配
- **优先分红权**：优先获得分红

**其他条款：**
- **拖带权**：强制其他股东一起出售
- **跟随权**：跟随其他股东出售
- **回购权**：要求企业或股东回购
- **知情权**：了解企业经营信息
- **董事会席位**：派驻董事

## 4.5 投资决策

### 4.5.1 投资决策流程

```
项目筛选 → 立项 → 尽职调查 → 估值 → 投资条款 → 投资决策委员会审批 → 签署投资协议
```

### 4.5.2 投资决策委员会

- 由基金管理人的高级管理人员组成
- 负责投资项目的最终决策
- 采用投票表决方式
